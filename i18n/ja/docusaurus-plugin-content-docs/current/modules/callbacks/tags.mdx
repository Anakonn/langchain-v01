# タグ

コールバックにタグを追加するには、`call()`/`run()`/`apply()` メソッドに `tags` 引数を渡します。これは、ログをフィルタリングする際に便利です。例えば、特定の `LLMChain` に対するすべてのリクエストをログに記録したい場合は、タグを追加し、そのタグでログをフィルタリングできます。コンストラクタとリクエストのコールバックの両方にタグを渡すことができます。詳細については、上記の例を参照してください。これらのタグは、`on_llm_start`、`on_chat_model_start`、`on_chain_start`、`on_tool_start` の "start" コールバック メソッドの `tags` 引数に渡されます。
