# プロンプト

言語モデルのプロンプトとは、ユーザーが提供する一連の指示や入力で、モデルの応答を導くものです。これにより、モデルが文脈を理解し、関連性のある一貫性のある言語出力を生成することができます。例えば、質問に答えたり、文章を補完したり、会話に参加したりすることができます。
