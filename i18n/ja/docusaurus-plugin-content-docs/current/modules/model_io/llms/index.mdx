# LLMs

大規模言語モデル (LLMs) は LangChain の中核的な構成要素です。
LangChain は独自の LLMs を提供するのではなく、さまざまな LLMs と標準的なインターフェースで対話できるようにしています。具体的には、入力文字列を受け取り、出力文字列を返すインターフェースです。

LLM プロバイダー (OpenAI、Cohere、Hugging Face など) は多数あります。`LLM` クラスは、これらすべてに対する標準的なインターフェースを提供するように設計されています。
