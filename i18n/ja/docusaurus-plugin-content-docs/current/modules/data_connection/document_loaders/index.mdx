# ドキュメントローダー

:::info
組み込みのドキュメントローダーインテグレーションについては、[Integrations](/docs/integrations/document_loaders/)のドキュメントをご覧ください。
:::

ドキュメントローダーを使用して、ソースからデータを `Document` として読み込みます。`Document` は、テキストとメタデータの組み合わせです。例えば、単純な `.txt` ファイルを読み込むためのドキュメントローダー、Webページのテキストコンテンツを読み込むためのドキュメントローダー、YouTube動画の文字起こしを読み込むためのドキュメントローダーなどがあります。

ドキュメントローダーには、設定されたソースからドキュメントを読み込むための "load" メソッドがあります。また、メモリに遅延読み込みする "lazy load" も実装されています。

## 始めましょう

最も単純なローダーは、ファイルをテキストとして読み込み、すべてを1つのドキュメントに配置します。

```python
<!--IMPORTS:[{"imported": "TextLoader", "source": "langchain_community.document_loaders", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.text.TextLoader.html", "title": "Document loaders"}]-->
from langchain_community.document_loaders import TextLoader

loader = TextLoader("./index.md")
loader.load()
```
