# 構成

このセクションには、他の任意のシステム(外部APIやサービスなど)や/またはLangChainプリミティブを組み合わせた高レベルのコンポーネントが含まれています。

このセクションを理解するためのよい入門書は、[LangChain Expression Language](/docs/expression_language/get_started)のセクションを読み、パイピングやさまざまなプリミティブを使ってシーケンスを構築することに慣れることです。

このセクションでカバーされるコンポーネントは以下のとおりです:

## [ツール](/docs/modules/tools/)

ツールは、LLMやその他のコンポーネントが他のシステムと対話するためのインターフェイスを提供します。Wikipedia、電卓、Pythonのリアルタイム評価などの例があります。

## [エージェント](/docs/modules/agents)

エージェントは、言語モデルを使ってツールで定義された行動を決定します。エージェントには「実行者」が必要で、これはエージェントの実行時環境です。
実行者は、エージェントを呼び出し、エージェントが選択したツールを実行し、前回の結果の出力をエージェントに返し、この過程を繰り返します。
エージェントは、前回の結果の出力を解析し、次のステップを選択する責任があります。

## [チェーン](/docs/modules/chains)

他のプリミティブやコンポーネントを組み合わせたブロック式の構成。
