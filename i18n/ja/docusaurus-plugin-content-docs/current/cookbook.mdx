# Introduction to MDX

MDX is a superset of Markdown that allows you to use JSX (React components) inside Markdown documents. This enables you to create rich, interactive content that goes beyond the static pages that traditional Markdown provides.

## What is MDX?

MDX is a file format that lets you seamlessly write JSX in your Markdown documents. You can import components, use component logic, and output dynamic content all from within your Markdown files.

This makes MDX a powerful tool for creating interactive documentation, blogs with live code examples, and other rich content experiences.
