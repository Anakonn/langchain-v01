# LangChainの拡張

LangChainの基本抽象クラスを拡張することは、オープンソースリポジトリへの貢献を計画しているか、カスタムの内部統合を構築しているかに関わらず、推奨されています。

以下のモジュールのカスタムクラスを構築するためのガイドを確認してください:

- [チャットモデル](/docs/modules/model_io/chat/custom_chat_model)でチューニングされた言語モデルとのインターフェースを構築する。
- [LLM](/docs/modules/model_io/llms/custom_llm)でテキスト言語モデルとのインターフェースを構築する。
- [出力パーサー](/docs/modules/model_io/output_parsers/custom)で言語モデルの出力を処理する。
