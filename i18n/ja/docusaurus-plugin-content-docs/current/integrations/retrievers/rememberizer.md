# Rememberizer

>[Rememberizer](https://rememberizer.ai/)は、SkyDeck AI Inc.が開発したAIアプリケーション向けの知識強化サービスです。

このノートブックでは、`Rememberizer`から文書を取得し、下流で使用される文書形式に変換する方法を示します。

# 準備

APIキーが必要です。[https://rememberizer.ai](https://rememberizer.ai/)で共通知識を作成すると、APIキーを取得できます。APIキーは環境変数`REMEMBERIZER_API_KEY`に設定するか、`RememberizerRetriever`の初期化時に`rememberizer_api_key`として渡す必要があります。

`RememberizerRetriever`には以下の引数があります:
- オプションの`top_k_results`: デフォルトは10。返される文書数を制限するのに使用します。
- オプションの`rememberizer_api_key`: 環境変数`REMEMBERIZER_API_KEY`を設定していない場合に必要です。

`get_relevant_documents()`には1つの引数`query`があります。これは、`Rememberizer.ai`の共通知識内の文書を検索するためのフリーテキストです。

# 例

## 基本的な使用方法

```python
# Setup API key
from getpass import getpass

REMEMBERIZER_API_KEY = getpass()
```

```python
import os

from langchain_community.retrievers import RememberizerRetriever

os.environ["REMEMBERIZER_API_KEY"] = REMEMBERIZER_API_KEY
retriever = RememberizerRetriever(top_k_results=5)
```

```python
docs = retriever.get_relevant_documents(query="How does Large Language Models works?")
```

# チェーンでの使用

```python
OPENAI_API_KEY = getpass()
```

```python
os.environ["OPENAI_API_KEY"] = OPENAI_API_KEY
```

```python
from langchain.chains import ConversationalRetrievalChain
from langchain_openai import ChatOpenAI

model = ChatOpenAI(model_name="gpt-3.5-turbo")
qa = ConversationalRetrievalChain.from_llm(model, retriever=retriever)
```

```python
questions = [
    "What is RAG?",
    "How does Large Language Models works?",
]
chat_history = []

for question in questions:
    result = qa.invoke({"question": question, "chat_history": chat_history})
    chat_history.append((question, result["answer"]))
    print(f"-> **Question**: {question} \n")
    print(f"**Answer**: {result['answer']} \n")
```

