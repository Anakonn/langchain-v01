# プロバイダ

:::info

独自のインテグレーションを書きたい場合は、[LangChainの拡張](/docs/guides/development/extending_langchain/)を参照してください。
インテグレーションに貢献したい場合は、[インテグレーションへの貢献](/docs/contributing/integrations/)を参照してください。
