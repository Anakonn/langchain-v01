# OpenAI

OpenAIに関連するすべての機能

>[OpenAI](https://en.wikipedia.org/wiki/OpenAI)は、非営利の`OpenAI Incorporated`と営利子会社の`OpenAI Limited Partnership`から成る、アメリカのAI研究所です。
>`OpenAI`は、友好的なAIの促進と開発を目的として、AI研究を行っています。
>`OpenAI`のシステムは、`Microsoft`の`Azure`ベースのスーパーコンピューティングプラットフォームで動作しています。

>[OpenAI API](https://platform.openai.com/docs/models)は、さまざまな機能と価格帯のモデルによって動作しています。

>[ChatGPT](https://chat.openai.com)は、`OpenAI`が開発した人工知能(AI)チャットボットです。

## インストールとセットアップ

統合パッケージをインストールします。

```bash
pip install langchain-openai
```

OpenAIのAPIキーを取得し、環境変数(`OPENAI_API_KEY`)に設定します。

## トークナイザー

`tiktoken`トークナイザーは、さまざまな場所で使用できます。デフォルトでは、OpenAI LLMのトークン数をカウントするために使用されます。

ドキュメントを分割する際にも、トークン数をカウントするために使用できます。

```python
<!--IMPORTS:[{"imported": "CharacterTextSplitter", "source": "langchain.text_splitter", "docs": "https://api.python.langchain.com/en/latest/character/langchain_text_splitters.character.CharacterTextSplitter.html", "title": "OpenAI"}]-->
```

この詳細については、[このノートブック](/docs/modules/data_connection/document_transformers/split_by_token#tiktoken)を参照してください。
