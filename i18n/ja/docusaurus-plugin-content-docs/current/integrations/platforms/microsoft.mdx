### Azure Blob Storage

>[Azure Blob Storage](https://learn.microsoft.com/en-us/azure/storage/blobs/storage-blobs-introduction)は、Microsoftのクラウド向けオブジェクトストレージソリューションです。Blob Storageは、構造化されていないデータ(テキストやバイナリデータなど)を大量に保存するのに最適化されています。

>[Azure Files](https://learn.microsoft.com/en-us/azure/storage/files/storage-files-introduction)は、業界標準のServer Message Block (SMB)プロトコル、Network File System (NFS)プロトコル、Azure Files REST APIを使ってアクセスできる、完全に管理されたクラウドファイル共有サービスです。`Azure Files`は`Azure Blob Storage`に基づいています。

`Azure Blob Storage`は以下のような用途に適しています:
- ブラウザから直接イメージやドキュメントを提供する
- 分散アクセスのためのファイル保存
- ビデオやオーディオのストリーミング
- ログファイルの書き込み
- バックアップ、災害復旧、アーカイブのためのデータ保存
- オンプレミスまたはAzureホストのサービスによる分析のためのデータ保存

```bash
pip install azure-storage-blob
```

## ベクトルストア

### Azure Cosmos DB

>[Azure Cosmos DB for MongoDB vCore](https://learn.microsoft.com/en-us/azure/cosmos-db/mongodb/vcore/) は、完全なネイティブ MongoDB サポートを備えたデータベースを簡単に作成できます。
> MongoDB のエクスペリエンスを活かし、お気に入りの MongoDB ドライバー、SDK、ツールを使い続けることができます。API for MongoDB vCore アカウントの接続文字列を指定するだけで、アプリケーションを接続できます。
> Azure Cosmos DB for MongoDB vCore のベクトル検索を使用して、Azure Cosmos DB に格納されているデータと、AI ベースのアプリケーションを簡単に統合できます。

#### インストールと設定

[詳細な設定手順](/docs/integrations/vectorstores/azure_cosmos_db)をご覧ください。

`pymongo` Pythonパッケージをインストールする必要があります。

```bash
pip install pymongo
```

#### Microsoft Azureでの Azure Cosmos DBのデプロイ

Azure Cosmos DB for MongoDB vCore は、開発者に完全に管理されたMongoDB互換のデータベースサービスを提供し、既存のアプリケーションの移行や新しいアプリケーションの構築に役立ちます。

Cosmos DB for MongoDB vCore を使うと、ネイティブなAzureインテグレーション、低い総所有コスト(TCO)、そして馴染みのあるvCoreアーキテクチャの恩恵を受けることができます。

[無料で始める](https://azure.microsoft.com/en-us/free/)ことができます。

[使用例](/docs/integrations/vectorstores/azure_cosmos_db)をご覧ください。

## リトリーバー

### Azure AI Search

>[Azure AI Search](https://learn.microsoft.com/en-us/azure/search/search-what-is-azure-search) (旧 `Azure Search` または `Azure Cognitive Search`) は、Web、モバイル、エンタープライズアプリケーションで、プライベートで異種のコンテンツに対して豊富な検索体験を構築するためのクラウド検索サービスです。

>検索は、ユーザーにテキストを表示するアプリケーションの基盤となるものです。一般的なシナリオには、カタログや文書の検索、オンラインの小売アプリ、独自のコンテンツに対するデータ探索などがあります。検索サービスを作成する際は、以下の機能を活用できます:
>- ユーザー所有のコンテンツに対するフルテキスト検索を行う検索エンジン
>- 語彙解析やオプションのAI強化によるリッチなインデックス作成
>- テキスト検索、ファジー検索、自動補完、地理検索などのリッチなクエリ構文
>- REST APIとAzure SDKのクライアントライブラリによるプログラミング性
>- データ層、機械学習層、AI(AIサービス)でのAzureインテグレーション

[セットアップ手順](https://learn.microsoft.com/en-us/azure/search/search-create-service-portal)をご覧ください。
