# グーグル

[Google Cloud Platform](https://cloud.google.com/) およびその他の `Google` 製品に関連するすべての機能。

## LLMs

個々の開発者には、まず Gemini API (`langchain-google-genai`) を使用し、商業サポートと高いレート制限が必要になった時に Vertex AI (`langchain-google-vertexai`) に移行することをお勧めします。すでにクラウドフレンドリーまたはクラウドネイティブな場合は、すぐに Vertex AI を開始できます。
詳細情報は[こちら](https://ai.google.dev/gemini-api/docs/migrate-to-cloud)をご覧ください。

### Google Generative AI

`GoogleGenerativeAI` クラスを通じて GoogleAI `Gemini` モデル（`gemini-pro` および `gemini-pro-vision`）にアクセスします。

Python パッケージをインストールします。

```bash
pip install langchain-google-genai
```

[使用例](/docs/integrations/llms/google_ai)をご覧ください。

```python
from langchain_google_genai import GoogleGenerativeAI
```

### Vertex AI Model Garden

`Vertex AI Model Garden` サービスを介して `PaLM` や数百の OSS モデルにアクセスします。

`langchain-google-vertexai` Python パッケージをインストールする必要があります。

```bash
pip install langchain-google-vertexai
```

[使用例](/docs/integrations/llms/google_vertex_ai_palm#vertex-model-garden)をご覧ください。

```python
from langchain_google_vertexai import VertexAIModelGarden
```

## チャットモデル

### Google Generative AI

`ChatGoogleGenerativeAI` クラスを通じて GoogleAI `Gemini` モデル（`gemini-pro` および `gemini-pro-vision`）にアクセスします。

```bash
pip install -U langchain-google-genai
```

API キーを設定します。

```bash
export GOOGLE_API_KEY=your-api-key
```

```python
from langchain_google_genai import ChatGoogleGenerativeAI

llm = ChatGoogleGenerativeAI(model="gemini-pro")
llm.invoke("Sing a ballad of LangChain.")
```

Gemini ビジョンモデルは、単一のチャットメッセージを提供する際に画像入力をサポートします。

```python
<!--IMPORTS:[{"imported": "HumanMessage", "source": "langchain_core.messages", "docs": "https://api.python.langchain.com/en/latest/messages/langchain_core.messages.human.HumanMessage.html", "title": "Google"}]-->
from langchain_core.messages import HumanMessage
from langchain_google_genai import ChatGoogleGenerativeAI

llm = ChatGoogleGenerativeAI(model="gemini-pro-vision")

message = HumanMessage(
    content=[
        {
            "type": "text",
            "text": "What's in this image?",
        },  # You can optionally provide text parts
        {"type": "image_url", "image_url": "https://picsum.photos/seed/picsum/200/300"},
    ]

llm.invoke([message])
```

image_url の値は以下のいずれかにすることができます:

- 公開画像 URL
- gcs ファイル (例: "gcs://path/to/file.png")
- ローカルファイルパス
- base64 エンコードされた画像 (例: data:image/png;base64,abcd124)
- PIL 画像
