# Vsdx

> A [visio file](https://fr.wikipedia.org/wiki/Microsoft_Visio) (with extension .vsdx) は Microsoft Visio、図作成ソフトウェアに関連付けられています。図の構造、レイアウト、グラフィック要素に関する情報を格納しています。このフォーマットは、ビジネス、エンジニアリング、コンピューター科学などの分野における視覚化の作成と共有を容易にします。

Visio ファイルには複数のページが含まれる場合があります。それらのいくつかは他のページの背景として機能する可能性があり、これは複数のレイヤーにわたって発生する可能性があります。この **loader** は各ページとその関連ページからテキストコンテンツを抽出し、OCRアルゴリズムと同様に各ページから表示されるすべてのテキストを抽出できるようにします。

**警告**: この loader と互換性があるのは **.vsdx** 拡張子を持つVisioファイルのみです。.vsd などの拡張子のファイルは圧縮 XML に変換できないため、互換性がありません。

```python
from langchain_community.document_loaders import VsdxLoader
```

```python
loader = VsdxLoader(file_path="./example_data/fake.vsdx")
documents = loader.load()
```

**ロードされたドキュメントの表示**

```python
for i, doc in enumerate(documents):
    print(f"\n------ Page {doc.metadata['page']} ------")
    print(f"Title page : {doc.metadata['page_name']}")
    print(f"Source : {doc.metadata['source']}")
    print("\n==> CONTENT <== ")
    print(doc.page_content)
```
