# 現代の財務

>[現代の財務](https://www.moderntreasury.com/)は複雑な支払い業務を簡素化します。これは、お金を動かすための製品やプロセスを駆動するための統一されたプラットフォームです。
>- 銀行や支払いシステムに接続する
>- 取引と残高をリアルタイムで追跡する
>- 規模に合わせて支払い業務を自動化する
