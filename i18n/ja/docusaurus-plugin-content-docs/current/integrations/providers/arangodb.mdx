# ArangoDB

>[ArangoDB](https://github.com/arangodb/arangodb) は、接続されたデータから価値を迅速に引き出すことができる、スケーラブルなグラフデータベースシステムです。ネイティブグラフ、統合された検索エンジン、単一のクエリ言語によるJSON サポートを備えています。ArangoDB は、オンプレミスでもクラウドでも、どこでも実行できます。

## インストールとセットアップ

[ArangoDB Python Driver](https://github.com/ArangoDB-Community/python-arango) パッケージをインストールします。

```bash
pip install python-arango
```
