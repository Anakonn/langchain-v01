# KDB.AI

>[KDB.AI](https://kdb.ai)は、リアルタイムデータを使用して、高度な検索、推奨、パーソナライゼーションを提供することで、スケーラブルで信頼性の高いAIアプリケーションを構築できる強力な知識ベースのベクトルデータベースおよび検索エンジンです。

## インストールとセットアップ

PythonのSDKをインストールします:

```bash
pip install kdbai-client
```

## ベクトルストア

KDB.AIインデックスを囲むラッパーが存在し、セマンティック検索や例の選択のためにベクトルストアとして使用できます。
