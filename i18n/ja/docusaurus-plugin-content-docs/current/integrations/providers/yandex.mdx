# Yandex

Yandexクラウドに関連するすべての機能

>[Yandexクラウド](https://cloud.yandex.com/en/)は、パブリッククラウドプラットフォームです。

## インストールとセットアップ

YandexクラウドSDKは、PyPIからpipを介してインストールできます:

```bash
pip install yandexcloud
```
