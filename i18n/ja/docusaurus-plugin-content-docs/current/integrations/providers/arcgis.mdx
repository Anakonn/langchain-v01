# ArcGIS

>[ArcGIS](https://www.esri.com/en-us/arcgis/about-arcgis/overview)は、[Esri](https://www.esri.com/)が開発・管理しているクライアント、サーバー、オンラインの地理情報システムソフトウェアのファミリーです。

>`ArcGISLoader`は`arcgis`パッケージを使用します。
>`arcgis`は、ベクトルおよびラスター解析、ジオコーディング、地図作成、ルーティングおよび方向性のためのPythonライブラリです。
>ユーザー、グループ、情報アイテムを管理、整理、管理します。
>Esriや他の信頼できるソースからの使い勝手の良い地図やキュレーションされた地理データにアクセスでき、独自のデータとも連携できます。

## インストールとセットアップ

`arcgis`パッケージをインストールする必要があります。

```bash
pip install -U arcgis
```
