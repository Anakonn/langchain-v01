# Git

>[Git](https://en.wikipedia.org/wiki/Git) は、ソフトウェア開発中にプログラマーが共同で作業を調整するために使用される、コンピューターファイルの変更を追跡する分散型バージョン管理システムです。

## インストールとセットアップ

まず、`GitPython` Pythonパッケージをインストールする必要があります。

```bash
pip install GitPython
```
