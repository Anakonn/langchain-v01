# YouTube

>[YouTube](https://www.youtube.com/) は Googleによるオンラインビデオ共有およびソーシャルメディアプラットフォームです。
> `YouTube`の文字起こしとビデオ情報をダウンロードします。

## インストールとセットアップ

```bash
pip install youtube-transcript-api
pip install pytube
```

[使用例](/docs/integrations/document_loaders/youtube_transcript)を参照してください。
