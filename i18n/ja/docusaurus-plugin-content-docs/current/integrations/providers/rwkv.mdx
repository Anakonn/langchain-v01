# RWKV-4

このページでは、LangChainの中で `RWKV-4` ラッパーを使用する方法について説明します。
インストールとセットアップ、そして使用例の2つの部分に分かれています。

## インストールとセットアップ

- `pip install rwkv`でPythonパッケージをインストールします
- `pip install tokenizer`でトークナイザーのPythonパッケージをインストールします
- [RWKV モデル](https://huggingface.co/BlinkDL/rwkv-4-raven/tree/main)をダウンロードし、希望のディレクトリに配置します
- [トークンファイル](https://raw.githubusercontent.com/BlinkDL/ChatRWKV/main/20B_tokenizer.json)をダウンロードします

## 使用

### RWKV

RWKVラッパーを使用するには、事前学習済みのモデルファイルへのパスとトークナイザーの設定を提供する必要があります。

```python
<!--IMPORTS:[{"imported": "RWKV", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.rwkv.RWKV.html", "title": "RWKV-4"}]-->
from langchain_community.llms import RWKV
```

# モデルをテストする

```python

def generate_prompt(instruction, input=None):
    if input:
        return f"""Below is an instruction that describes a task, paired with an input that provides further context. Write a response that appropriately completes the request.

# Instruction:
{instruction}

# Input:
{input}

# Response:
"""
    else:
        return f"""Below is an instruction that describes a task. Write a response that appropriately completes the request.

# Instruction:
{instruction}

# Response:
"""


model = RWKV(model="./models/RWKV-4-Raven-3B-v7-Eng-20230404-ctx4096.pth", strategy="cpu fp32", tokens_path="./rwkv/20B_tokenizer.json")
response = model.invoke(generate_prompt("Once upon a time, "))
```

## モデルファイル

[RWKV-4-Raven](https://huggingface.co/BlinkDL/rwkv-4-raven/tree/main)リポジトリでモデルファイルのダウンロードリンクを見つけることができます。

### Rwkv-4 models -> 推奨 VRAM

```text
RWKV VRAM
Model | 8bit | bf16/fp16 | fp32
14B   | 16GB | 28GB      | >50GB
7B    | 8GB  | 14GB      | 28GB
3B    | 2.8GB| 6GB       | 12GB
1b5   | 1.3GB| 3GB       | 6GB
```

ストリーミングやCUDAサポートなどの戦略については、[rwkv pip](https://pypi.org/project/rwkv/)ページを参照してください。
