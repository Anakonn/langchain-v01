# Docugami

>[Docugami](https://docugami.com) は、ビジネス文書をドキュメントXML Knowledge Graphに変換し、文書全体を表す複数のXMLセマンティックツリーを生成します。これは、文書の各部分のセマンティックおよび構造的特性をXMLツリーとして表現する豊かな表現です。

## インストールとセットアップ

```bash
pip install dgml-utils
pip install docugami-langchain
```

## ドキュメントローダー

[使用例](/docs/integrations/document_loaders/docugami)を参照してください。

```python
from docugami_langchain.document_loaders import DocugamiLoader
```
