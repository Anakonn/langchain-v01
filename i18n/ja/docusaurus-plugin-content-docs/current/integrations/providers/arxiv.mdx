# Arxiv

>[arXiv](https://arxiv.org/) は、物理学、数学、コンピューターサイエンス、定量生物学、定量金融、統計学、電気工学、システム科学、経済学の分野で、200万件を超える学術論文を収録したオープンアクセスのアーカイブです。

## インストールとセットアップ

まず、`arxiv` Pythonパッケージをインストールする必要があります。

```bash
pip install arxiv
```

次に、`arxiv.org`サイトからダウンロードしたPDFファイルをテキスト形式に変換する`PyMuPDF` Pythonパッケージをインストールする必要があります。

```bash
pip install pymupdf
```
