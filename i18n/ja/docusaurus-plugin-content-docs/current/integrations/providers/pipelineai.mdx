# PipelineAI

このページでは、LangChainでPipelineAIエコシステムを使用する方法について説明します。
2つの部分に分かれています: インストールとセットアップ、そして特定のPipelineAIラッパーへの参照。

## インストールとセットアップ

- `pip install pipeline-ai`でインストールする
- Pipeline Cloudのapi keyを取得し、環境変数(`PIPELINE_API_KEY`)に設定する
