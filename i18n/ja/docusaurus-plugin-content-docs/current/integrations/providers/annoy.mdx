# 煩わせる

> [Annoy](https://github.com/spotify/annoy) (`Approximate Nearest Neighbors Oh Yeah`)
> は、与えられたクエリポイントに近い点を検索するためのC++ライブラリで、Pythonバインディングが付属しています。また、多くのプロセスが同じデータを共有できるように、メモリにマッピングされた大規模な読み取り専用ファイルベースのデータ構造を作成します。

## インストールとセットアップ

```bash
pip install annoy
```
