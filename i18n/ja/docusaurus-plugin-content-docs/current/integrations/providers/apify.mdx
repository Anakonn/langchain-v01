# Apify

>[Apify](https://apify.com)は、Web スクレイピングとデータ抽出のためのクラウドプラットフォームで、
>さまざまなスクレイピング、クロール、抽出のユースケースに対応した
>1,000 以上の事前構築されたアプリ「*Actors*」のエコシステム[](https://apify.com/store)を提供しています。

[![Apify Actors](/img/ApifyActors.png)](https://apify.com/store)

このインテグレーションにより、`Apify`プラットフォーム上でActorsを実行し、その結果をLangChainにロードして、
ウェブサイトのドキュメント、ブログ、ナレッジベースなどからドキュメントとデータをベクトルインデックスに供給し、
回答を生成することができます。

## インストールとセットアップ

- Python用のApify APIクライアントを`pip install apify-client`でインストールします
- [Apify APIトークン](https://console.apify.com/account/integrations)を取得し、環境変数(`APIFY_API_TOKEN`)に設定するか、
  `ApifyWrapper`のコンストラクタに`apify_api_token`として渡します。
