# DataForSEO

>[DataForSeo](https://dataforseo.com/)は、APIを通じて包括的なSEOおよびデジタルマーケティングデータソリューションを提供しています。

このページでは、LangChainでDataForSEO検索APIを使用する方法について説明します。

## インストールと設定

[DataForSEO APIアクセスログインとパスワード](https://app.dataforseo.com/register)を取得し、環境変数(`DATAFORSEO_LOGIN`と`DATAFORSEO_PASSWORD`)に設定してください。

```python
import os

os.environ["DATAFORSEO_LOGIN"] = "your_login"
os.environ["DATAFORSEO_PASSWORD"] = "your_password"
```

## ユーティリティ

DataForSEOユーティリティはAPIをラップしています。このユーティリティをインポートするには、以下を使用します:

```python
<!--IMPORTS:[{"imported": "DataForSeoAPIWrapper", "source": "langchain_community.utilities.dataforseo_api_search", "docs": "https://api.python.langchain.com/en/latest/utilities/langchain_community.utilities.dataforseo_api_search.DataForSeoAPIWrapper.html", "title": "DataForSEO"}]-->
from langchain_community.utilities.dataforseo_api_search import DataForSeoAPIWrapper
```

このラッパーの詳細なウォークスルーについては、[このノートブック](/docs/integrations/tools/dataforseo)を参照してください。

## ツール

このラッパーをツールとしてロードして、Agentで使用することもできます:

```python
<!--IMPORTS:[{"imported": "load_tools", "source": "langchain.agents", "docs": "https://api.python.langchain.com/en/latest/agent_toolkits/langchain_community.agent_toolkits.load_tools.load_tools.html", "title": "DataForSEO"}]-->
from langchain.agents import load_tools
tools = load_tools(["dataforseo-api-search"])
```

## 使用例

```python
dataforseo = DataForSeoAPIWrapper(api_login="your_login", api_password="your_password")
result = dataforseo.run("Bill Gates")
print(result)
```
