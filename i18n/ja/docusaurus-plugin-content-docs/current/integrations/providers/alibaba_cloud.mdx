# Alibaba Cloud

>[Alibaba Group Holding Limited (Wikipedia)](https://en.wikipedia.org/wiki/Alibaba_Group)、または `Alibaba`
> (中国語: 阿里巴巴)は、eコマース、小売、
> インターネット、テクノロジーを専門とする中国の多国籍テクノロジー企業です。
>
> [Alibaba Cloud (Wikipedia)](https://en.wikipedia.org/wiki/Alibaba_Cloud)、また `Aliyun`
> (中国語: 阿里云; ピンイン: Ālǐyún; 直訳すると 'Ali Cloud')は、クラウドコンピューティング企業で、
> `Alibaba Group`の子会社です。 `Alibaba Cloud`はオンラインビジネスとAlibaba自身のeコマースエコシステムにクラウドコンピューティングサービスを提供しています。
