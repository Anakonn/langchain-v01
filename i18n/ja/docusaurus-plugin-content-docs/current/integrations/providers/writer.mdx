# ライター

このページでは、LangChainのライターエコシステムの使用方法について説明します。
2つの部分に分かれています: インストールとセットアップ、そして特定のライターラッパーへの参照。

## インストールとセットアップ

- ライターのAPIキーを取得し、環境変数(`WRITER_API_KEY`)に設定します。
