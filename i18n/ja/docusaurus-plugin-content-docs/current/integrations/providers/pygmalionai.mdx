# PygmalionAI

>[PygmalionAI](https://pygmalion.chat/)は[Aphrodite Engine](https://github.com/PygmalionAI/aphrodite-engine)のインファレンスエンドポイントを提供することで、オープンソースモデルをサポートする企業です。

## インストールとセットアップ

```bash
pip install aphrodite-engine
```
