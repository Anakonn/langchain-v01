# GooseAI

このページでは、LangChainでGooseAIエコシステムを使用する方法について説明します。
インストールとセットアップ、そしてGooseAIラッパーの参照に分かれています。

## インストールとセットアップ

- `pip install openai`でPythonSDKをインストールします。
- このリンク[here](https://goose.ai/)からGooseAIのAPIキーを取得します。
- 環境変数(`GOOSEAI_API_KEY`)を設定します。

```python
import os
os.environ["GOOSEAI_API_KEY"] = "YOUR_API_KEY"
```

## ラッパー
