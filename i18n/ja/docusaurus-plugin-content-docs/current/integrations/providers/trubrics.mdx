# Trubrics

>[Trubrics](https://trubrics.com)は、ユーザープロンプトとAIモデルへのフィードバックを収集、分析、管理できるLLMユーザーアナリティクスプラットフォームです。
>
>`Trubrics`の詳細については、[Trubrics repo](https://github.com/trubrics/trubrics-sdk)をご覧ください。

## インストールとセットアップ

`trubrics`Pythonパッケージをインストールする必要があります:

```bash
pip install trubrics
```
