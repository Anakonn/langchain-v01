# Salute Devices

Salute Devicesは、GigaChatのLLMモデルを提供しています。

GigaChatにアクセスする方法については、[こちら](https://developers.sber.ru/docs/ru/gigachat/api/integration)をご覧ください。

## インストールとセットアップ

GigaChatパッケージはPyPIからpipでインストールできます:

```bash
pip install gigachat
```

## LLMs

[使用例](/docs/integrations/llms/gigachat)をご覧ください。
