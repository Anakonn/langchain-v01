# Doctran

>[Doctran](https://github.com/psychic-api/doctran)は、Pythonパッケージです。LLMsとオープンソースのNLPライブラリを使用して、生のテキストを、ベクトル空間検索に最適化された、きれいで構造化された情報密度の高い文書に変換します。`Doctran`は、雑多な文字列が入力され、きれいで、ラベル付きの文字列が出力される、ブラックボックスのようなものだと考えることができます。

## インストールとセットアップ

```bash
pip install doctran
```
