# LLモニター

>[LLモニター](https://llmonitor.com?utm_source=langchain&utm_medium=py&utm_campaign=docs)は、コストと使用状況の分析、ユーザー追跡、トレーシング、評価ツールを提供するオープンソースの可観測性プラットフォームです。

## インストールとセットアップ

[llmonitor.com](https://llmonitor.com?utm_source=langchain&utm_medium=py&utm_campaign=docs)でアカウントを作成し、新しいアプリのトラッキングIDをコピーします。

それが分かったら、次のように環境変数に設定します:

```bash
export LLMONITOR_APP_ID="..."
```
