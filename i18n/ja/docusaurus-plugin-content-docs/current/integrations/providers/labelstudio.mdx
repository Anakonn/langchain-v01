# ラベルスタジオ

>[ラベルスタジオ](https://labelstud.io/guide/get_started)は、大規模言語モデル(LLM)のファインチューニングのためのデータラベリングに柔軟性を提供するオープンソースのデータラベリングプラットフォームです。また、カスタムトレーニングデータの準備や、人間のフィードバックを通じた応答の収集と評価も可能にします。

## インストールとセットアップ

インストールオプションについては、[ラベルスタジオのインストールガイド](https://labelstud.io/guide/install)を参照してください。

`label-studio`と`label-studio-sdk-python`のPythonパッケージをインストールする必要があります:

```bash
pip install label-studio label-studio-sdk
```
