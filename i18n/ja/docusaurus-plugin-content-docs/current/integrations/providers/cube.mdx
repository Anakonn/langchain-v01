# キューブ

>[キューブ](https://cube.dev/)は、データアプリを構築するためのセマンティックレイヤーです。データエンジニアやアプリケーション開発者が、モダンなデータストアからデータにアクセスし、一貫した定義に整理し、すべてのアプリケーションに提供するのに役立ちます。

## インストールとセットアップ

API キーとキューブインスタンスのURLを取得する必要があります。[これらの手順](https://cube.dev/docs/product/apis-integrations/rest-api#configuration-base-path)を参照してください。
