# アウトライン

> [アウトライン](https://www.getoutline.com/)は、チームの情報共有のために設計されたオープンソースのコラボレーティブナレッジベースプラットフォームです。

## セットアップ

まず、Outlineインスタンスの[APIキーを作成](https://www.getoutline.com/developers#section/Authentication)する必要があります。次に、以下の環境変数を設定する必要があります:

```python
import os

os.environ["OUTLINE_API_KEY"] = "xxx"
os.environ["OUTLINE_INSTANCE_URL"] = "https://app.getoutline.com"
```
