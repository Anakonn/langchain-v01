# DeepInfra

>[DeepInfra](https://deepinfra.com/docs)を使うと、
>[最新のマシンラーニングモデル](https://deepinfra.com/models)を簡単に実行できます。
>DeepInfraは、モデルの実行、スケーリング、監視に関するすべての重要な作業を行います。
>ユーザーはアプリケーションに集中し、シンプルなREST APIコールでモデルを統合できます。

>DeepInfraは、LangChainとの統合の[例](https://deepinfra.com/docs/advanced/langchain)を提供しています。

このページでは、`LangChain`内の`DeepInfra`エコシステムの使用方法について説明します。
インストールとセットアップ、そして特定のDeepInfraラッパーへの参照の2つの部分に分かれています。

## インストールとセットアップ

- このリンク[here](https://deepinfra.com/)からDeepInfraのAPIキーを取得してください。
- DeepInfraのAPIキーを取得し、環境変数(`DEEPINFRA_API_TOKEN`)に設定してください。

## 利用可能なモデル

DeepInfraは、展開準備ができた幅広いオープンソースのLLMを提供しています。

[text-generation](https://deepinfra.com/models?type=text-generation)と[embeddings](https://deepinfra.com/models?type=embeddings)でサポートされているモデルを確認できます。

[リクエストとレスポンスのパラメーター](https://deepinfra.com/meta-llama/Llama-2-70b-chat-hf/api)の一覧を表示できます。

チャットモデルは[openai apiに準拠](https://deepinfra.com/meta-llama/Llama-2-70b-chat-hf/api?example=openai-http)しています。
