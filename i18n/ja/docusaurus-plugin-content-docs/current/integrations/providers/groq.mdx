# Groq

Groqへようこそ! 🚀 Groqでは、世界初のLanguage Processing Unit™、または LPUを開発しました。Groq LPUには、決定論的な単一コアストリーミングアーキテクチャがあり、任意の作業負荷に対して予測可能で繰り返し可能なパフォーマンスを持つGenAIインファレンススピードの基準を設定しています。

アーキテクチャ以外にも、私たちのソフトウェアは、革新的で強力なAIアプリケーションを作成するために、開発者の皆さんに必要なツールを提供するように設計されています。Groqをエンジンとして使えば、以下のことができます:

* リアルタイムのAIおよびHPCインファレンスに対して、妥協のないlow latencyとパフォーマンスを達成 🔥
* 任意の作業負荷に対して正確なパフォーマンスと計算時間を知ることができる 🔮
* 最先端の技術を活用して競争に先駆けることができる 💪

Groqについてもっと知りたい場合は、[website](https://groq.com)のリソースをチェックしたり、[Discord community](https://discord.gg/JvNsBDKeCG)に参加して開発者と交流したりしてください。

## インストールとセットアップ

統合パッケージをインストールします:

```bash
pip install langchain-groq
```

[API key](https://wow.groq.com)をリクエストし、環境変数として設定します:

```bash
export GROQ_API_KEY=gsk_...
```
