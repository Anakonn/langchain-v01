# ブラウザレス

>[ブラウザレス](https://www.browserless.io/docs/start)は、クラウド上でヘッドレスChrome インスタンスを実行できるサービスです。自分のインフラストラクチャを管理する必要がなく、スケールでブラウザベースの自動化を実行できる優れた方法です。

## インストールとセットアップ

[ここ](https://www.browserless.io/pricing/)からAPIキーを取得する必要があります。
