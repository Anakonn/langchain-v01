# GitHub

>[GitHub](https://github.com/) は開発者がコードを作成、保存、管理、共有できるデベロッパープラットフォームです。`Git`ソフトウェアを使用し、Gitの分散バージョン管理に加えて、アクセス制御、バグ追跡、ソフトウェア機能リクエスト、タスク管理、継続的インテグレーション、およびプロジェクトごとのWikiを提供します。

## インストールとセットアップ

GitHub APIにアクセスするには、[個人アクセストークン](https://github.com/settings/tokens)が必要です。

## ドキュメントローダー
