# Ontotext GraphDB

>[Ontotext GraphDB](https://graphdb.ontotext.com/)は、RDFおよびSPARQLに準拠したグラフデータベースおよび知識発見ツールです。

## 依存関係

[rdflib](https://github.com/RDFLib/rdflib)パッケージをインストールします。

```bash
pip install rdflib==7.0.0
```

## Graph QA Chain
