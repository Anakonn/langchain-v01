# ModelScope

>[ModelScope](https://www.modelscope.cn/home)は、モデルとデータセットの大きなリポジトリです。

このページでは、LangChainでModelScopeエコシステムを使う方法について説明します。
インストールとセットアップ、そして特定のModelScopeラッパーについての参照に分かれています。

## インストールとセットアップ

`modelscope`パッケージをインストールします。

```bash
pip install modelscope
```
