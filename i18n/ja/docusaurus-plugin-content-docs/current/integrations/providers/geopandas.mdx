# Geopandas

>[GeoPandas](https://geopandas.org/)は、Pythonでの地理空間データの操作を容易にするためのオープンソースプロジェクトです。`GeoPandas`は、`pandas`で使用されるデータ型を拡張し、幾何学的な型に対する空間演算を可能にします。
幾何学演算は`shapely`によって実行されます。

## インストールとセットアップ

いくつかのPythonパッケージをインストールする必要があります。

```bash
pip install -U sodapy pandas geopandas
```
