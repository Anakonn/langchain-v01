# Minimax

>[Minimax](https://api.minimax.chat)は、企業や個人向けの自然言語処理モデルを提供する中国のスタートアップです。

## インストールとセットアップ

[Minimax APIキー](https://api.minimax.chat/user-center/basic-information/interface-key)を取得し、環境変数(`MINIMAX_API_KEY`)に設定してください。
[Minimax グループID](https://api.minimax.chat/user-center/basic-information)を取得し、環境変数(`MINIMAX_GROUP_ID`)に設定してください。
