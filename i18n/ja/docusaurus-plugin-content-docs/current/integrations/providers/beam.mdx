# ビーム

>[ビーム](https://www.beam.cloud/)は、GPUを搭載したリモートサーバー上でコードを実行できるクラウドコンピューティングプラットフォームです。

## インストールとセットアップ

- [アカウントを作成する](https://www.beam.cloud/)
- `curl https://raw.githubusercontent.com/slai-labs/get-beam/main/get-beam.sh -sSfL | sh`でBeam CLIをインストールする
- `beam configure`でAPIキーを登録する
- 環境変数(`BEAM_CLIENT_ID`)と(`BEAM_CLIENT_SECRET`)を設定する
- Beam SDKをインストールする:

```bash
pip install beam-sdk
```
