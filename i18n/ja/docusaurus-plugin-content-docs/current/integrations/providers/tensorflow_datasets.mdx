# TensorFlow Datasets

>[TensorFlow Datasets](https://www.tensorflow.org/datasets) は、TensorFlow や Jax などの他のPython ML フレームワークと共に使用可能なデータセットのコレクションです。すべてのデータセットは [tf.data.Datasets](https://www.tensorflow.org/api_docs/python/tf/data/Dataset) として公開されており、使いやすく高性能な入力パイプラインを実現します。始めるには [ガイド](https://www.tensorflow.org/datasets/overview) および [データセットのリスト](https://www.tensorflow.org/datasets/catalog/overview#all_datasets) を参照してください。

## インストールとセットアップ

`tensorflow` と `tensorflow-datasets` の Python パッケージをインストールする必要があります。

```bash
pip install tensorflow
```

```bash
pip install tensorflow-dataset
```
