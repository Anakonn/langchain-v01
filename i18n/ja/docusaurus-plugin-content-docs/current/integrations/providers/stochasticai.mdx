# StochasticAI

このページでは、LangChain内のStochasticAIエコシステムの使用方法について説明します。
インストールとセットアップ、そして特定のStochasticAIラッパーへの参照の2つの部分に分かれています。

## インストールとセットアップ

- `pip install stochasticx`でインストールします
- StochasticAIのAPIキーを取得し、環境変数(`STOCHASTICAI_API_KEY`)に設定します

## ラッパー

### LLM

StochasticAI LLMラッパーが存在し、以下のようにアクセスできます。
