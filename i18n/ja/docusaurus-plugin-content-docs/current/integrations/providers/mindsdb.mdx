# MindsDB

MindsDBは、企業データからAIをカスタマイズするためのプラットフォームです。MindsDBとその約200の[データソース](https://docs.mindsdb.com/integrations/data-overview)および[AI/MLフレームワーク](https://docs.mindsdb.com/integrations/ai-overview)への統合により、開発者は企業データを使ってAIをより迅速かつ安全にカスタマイズできます。

MindsDBを使えば、任意のデータソースと任意のAI/MLモデルを接続して、AIパワーアプリケーションを実装および自動化できます。データベース、ベクトアストア、アプリケーションからのデータを使って、リアルタイムでモデルをデプロイ、提供、微調整できます。開発者が既に知っているユニバーサルツールを使ってすべてを行えます。

MindsDBはLangChainと統合しており、ユーザーは以下のことができます:

- LangChainで利用可能なモデルをMindsDBにデプロイし、多数のデータソースから利用可能にする。
- LangChainで利用可能なモデルをMindsDBで微調整し、リアルタイムかつダイナミックなデータを使う。
- LangChainとMindsDBでAIワークフローを自動化する。

[ドキュメント](https://docs.mindsdb.com/integrations/ai-engines/langchain)を参照して、MindsDBとLangChainの統合について詳しく学び、サンプルを確認してください。
