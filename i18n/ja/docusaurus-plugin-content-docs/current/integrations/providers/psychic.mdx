# サイキック

:::warning
このプロバイダーはもはや管理されておらず、動作しない可能性があります。注意して使用してください。
:::

>[サイキック](https://www.psychic.dev/)は、`Notion`、`Zendesk`、`Confluence`、`Google Drive`などのSaaSツールとOAuthを介して統合し、これらのアプリケーションからのドキュメントをSQLまたはベクトルデータベースに同期するプラットフォームです。これは、構造化されていないデータのPlaidのようなものと考えることができます。

## インストールとセットアップ

```bash
pip install psychicapi
```

サイキックの設定は簡単です - `react`ライブラリをインポートし、[サイキックダッシュボード](https://dashboard.psychic.dev/)から取得した`Sidekick API`キーで構成します。アプリケーションを接続すると、ダッシュボードからこれらの接続を表示し、サーバーサイドのライブラリを使用してデータを取得できます。

1. [ダッシュボード](https://dashboard.psychic.dev/)でアカウントを作成します。
2. [reactライブラリ](https://docs.psychic.dev/sidekick-link)を使用して、サイキックのリンクモーダルをフロントエンドのreactアプリに追加します。これを使用してSaaSアプリを接続します。
3. 接続を作成したら、[サンプルノートブック](/docs/integrations/document_loaders/psychic)に従って`PsychicLoader`を使用できます。

## 他のドキュメントローダーとの比較

1.	**汎用API:** 各SaaSアプリのOAuthフローを構築し、APIを学習する代わりに、サイキックを1回統合するだけで、汎用APIを活用してデータを取得できます。
2.	**データ同期:** お客様のSaaSアプリのデータは非常に早く古くなる可能性があります。サイキックを使用すると、毎日または実時間でドキュメントを最新の状態に保つようにWebフックを構成できます。
3.	**簡略化されたOAuth:** サイキックはエンドツーエンドのOAuthを処理するため、各統合のためのOAuthクライアントの作成、アクセストークンの更新、OAuthリダイレクトロジックの処理を行う必要がありません。
