# Datadog ログ

>[Datadog](https://www.datadoghq.com/)はクラウドスケールアプリケーションの監視およびアナリティクスプラットフォームです。

## インストールとセットアップ

```bash
pip install datadog_api_client
```

Datadogの API キーとAPPキーでローダーを初期化し、目的のログを抽出するクエリを設定する必要があります。
