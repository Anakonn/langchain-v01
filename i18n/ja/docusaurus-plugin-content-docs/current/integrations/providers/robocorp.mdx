# Robocorp

>[Robocorp](https://robocorp.com/) は、どこでも、どのスケールでも、シームレスに実行される Python ワーカーを構築およびオペレーションするのに役立ちます

## インストールとセットアップ

`langchain-robocorp` Python パッケージをインストールする必要があります:

```bash
pip install langchain-robocorp
```

エージェントアプリケーションから通信するには、Action Server のインスタンスが必要です。 [Robocorp クイックスタート](https://github.com/robocorp/robocorp#quickstart) を参照して、Action Server のセットアップと Actions の作成を行ってください。

Action Server の `new` コマンドを使用して、新しいプロジェクトをブートストラップできます。

```bash
action-server new
cd ./your-project-name
action-server start
```
