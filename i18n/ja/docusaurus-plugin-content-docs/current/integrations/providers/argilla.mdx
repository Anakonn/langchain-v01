# Argilla

>[Argilla](https://argilla.io/)は、LLMのためのオープンソースのデータキュレーションプラットフォームです。
>`Argilla`を使えば、人間とマシンのフィードバックを使って、より迅速なデータキュレーションを通じて、堅牢な言語モデルを誰もが構築できます。`Argilla`は、データラベリングからモデル監視まで、MLOpsサイクルのすべての段階をサポートしています。

## インストールとセットアップ

[APIキー](https://platform.openai.com/account/api-keys)を取得してください。

Pythonパッケージをインストールします:

```bash
pip install argilla
```
