# Tair

>[Alibaba Cloud Tair](https://www.alibabacloud.com/help/en/tair/latest/what-is-tair)は、`Alibaba Cloud`が開発したクラウドネイティブのインメモリデータベースサービスです。豊富なデータモデルと企業レベルの機能を提供し、オープンソースの`Redis`との完全な互換性を維持しながら、リアルタイムのオンラインシナリオをサポートします。`Tair`はまた、新しい不揮発性メモリ(NVM)ストレージメディアに基づく、パーシステントメモリ最適化インスタンスを導入しています。

## インストールとセットアップ

Tair Python SDKをインストールします:

```bash
pip install tair
```
