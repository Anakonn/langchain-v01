# HTMLからテキストへ

>[html2text](https://github.com/Alir3z4/html2text/)は、ページの`HTML`を読みやすい平文の`ASCII text`に変換するPythonパッケージです。

ASCIIはまた有効な`Markdown`(テキストからHTMLフォーマット)でもあります。

## インストールとセットアップ

```bash
pip install html2text
```

## ドキュメントトランスフォーマー

[使用例](/docs/integrations/document_transformers/html2text)を参照してください。

```python
from langchain_community.document_loaders import Html2TextTransformer
```
