# Baidu

>[Baidu Cloud](https://cloud.baidu.com/)は、北京に本社を置く`Baidu, Inc.`が提供するクラウドサービスです。
>クラウドストレージサービス、クライアントソフトウェア、ファイル管理、リソース共有、サードパーティ統合を提供しています。

## インストールとセットアップ

[ここ](https://cloud.baidu.com/product/wenxinworkshop)で`Qianfan` `AK`と`SK`キーを登録して取得してください。
