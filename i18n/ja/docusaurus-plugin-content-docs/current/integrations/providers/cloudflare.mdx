# Cloudflare

>[Cloudflare, Inc. (Wikipedia)](https://en.wikipedia.org/wiki/Cloudflare) は、コンテンツ配信ネットワークサービス、クラウドサイバーセキュリティ、DDoS緩和、およびICANN認定ドメイン登録サービスを提供する米国企業です。
