# BibTeX

>[BibTeX](https://www.ctan.org/pkg/bibtex)は、`LaTeX`組版と併用される一般的なファイル形式およびリファレンス管理システムです。学術および研究文書の書誌情報を整理および保存する方法として機能します。

## インストールとセットアップ

`bibtexparser`および`pymupdf`パッケージをインストールする必要があります。

```bash
pip install bibtexparser pymupdf
```
