# AINetwork

>[AI Network](https://www.ainetwork.ai/build-on-ain) は、大規模なAIモデルに対応するよう設計されたレイヤー1ブロックチェーンであり、分散型GPUネットワークを利用して
> [$AINトークン](https://www.ainetwork.ai/token) によって強化されたAI駆動の`NFTs`（`AINFTs`）を提供します。

## インストールとセットアップ

`ain-py` Pythonパッケージをインストールする必要があります。

```bash
pip install ain-py
```

`AIN_BLOCKCHAIN_ACCOUNT_PRIVATE_KEY` 環境変数をAINブロックチェーンアカウントの秘密鍵に設定する必要があります。
