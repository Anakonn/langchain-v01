# Eden AI

>[Eden AI](https://docs.edenai.co/docs/getting-started-with-eden-ai) ユーザーインターフェイス (UI)
> は、AIプロジェクトを処理するために設計されています。 `Eden AI Portal`を使用すると、
> 市場最高のエンジンを使ってノーコードAIを実行できます。

## インストールとセットアップ

Eden AI APIにアクセスするには、APIキーが必要です。[アカウントを作成](https://app.edenai.run/user/register)し、
[こちら](https://app.edenai.run/admin/account/settings)からAPIキーを取得できます。

[使用例](/docs/integrations/tools/edenai_tools)を参照してください。
