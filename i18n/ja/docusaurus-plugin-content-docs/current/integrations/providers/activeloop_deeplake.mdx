# Activeloop Deep Lake

>[Activeloop Deep Lake](https://docs.activeloop.ai/)は、ベクトルストアとしての使用を可能にするDeep Learningアプリケーション用のデータレイクです。

## なぜDeep Lakeなのか?

- (マルチモーダルな)ベクトルストアだけではありません。独自のLLMモデルのファインチューニングにデータセットを使用できます。
- エンベディングだけでなく、自動バージョン管理付きの元のデータも保存できます。
- 本当にサーバーレス。別のサービスを必要とせず、主要なクラウドプロバイダー(`AWS S3`、`GCS`など)で使用できます。

`Activeloop Deep Lake`は`SelfQuery Retrieval`をサポートしています:
[Activeloop Deep Lake Self Query Retrieval](/docs/integrations/retrievers/self_query/activeloop_deeplake_self_query)
