# Vespa

>[Vespa](https://vespa.ai/)は、完全な機能を備えた検索エンジンとベクトルデータベースです。
> ベクトル検索(ANN)、語彙検索、構造化データの検索を、同一のクエリで実行できます。

## インストールとセットアップ

```bash
pip install pyvespa
```
