# リメンブロール

>[リメンブロール](https://remembrall.dev/)は、言語モデルに長期記憶、検索強化生成、完全な可観測性を提供するプラットフォームです。

## インストールとセットアップ

始めるには、[リメンブロールプラットフォームにGithubでサインインし](https://remembrall.dev/login)、[設定ページからAPIキーをコピーします](https://remembrall.dev/dashboard/settings)。
