# Anyscale

>[Anyscale](https://www.anyscale.com) は、本番環境対応のAPIを介してLLMを実行、微調整、およびスケーリングするためのプラットフォームです。
> [Anyscale Endpoints](https://docs.anyscale.com/endpoints/overview) は、多くのオープンソースモデルを費用対効果の高い方法で提供します。

`Anyscale` は、`LangChain` を使った高度なチャットエージェントのセットアップ方法の[例](https://docs.anyscale.com/endpoints/model-serving/examples/langchain-integration)も提供しています。

## インストールとセットアップ

- AnyscaleサービスのURL、ルート、およびAPIキーを取得し、それらを環境変数 (`ANYSCALE_SERVICE_URL`,`ANYSCALE_SERVICE_ROUTE`, `ANYSCALE_SERVICE_TOKEN`) として設定します。
- 詳細については、[Anyscaleのドキュメント](https://www.anyscale.com/get-started) を参照してください。

`openai` パッケージをインストールする必要があります:

```bash
pip install openai
```
