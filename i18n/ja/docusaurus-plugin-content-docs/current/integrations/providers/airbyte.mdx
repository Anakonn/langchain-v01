# Airbyte

>[Airbyte](https://github.com/airbytehq/airbyte)は、APIやデータベース、ファイルからデータウェアハウスやレイクへのELTパイプラインのためのデータ統合プラットフォームです。データウェアハウスやデータベースへのELTコネクタのカタログが最も大きいです。

## インストールとセットアップ

```bash
pip install -U langchain-airbyte
```

:::note

現在、`langchain-airbyte`ライブラリはPydantic v2をサポートしていません。
このパッケージを使用するには、Pydantic v1にダウングレードする必要があります。

このパッケージは現在、Python 3.10以上を要求しています。

:::

このインテグレーションパッケージには、設定する必要のある環境変数はありませんが、一部のインテグレーション(例: `source-github`)では資格情報を渡す必要があります。
