# Tigris

> [Tigris](https://tigrisdata.com)は、高性能なベクトル検索アプリケーションの構築を簡素化するためのオープンソースのサーバーレスNoSQLデータベースおよび検索プラットフォームです。
> `Tigris`は、複数のツールの管理、運用、同期に関するインフラストラクチャの複雑さを排除し、優れたアプリケーションの構築に集中できるようにします。

## インストールとセットアップ

```bash
pip install tigrisdb openapi-schema-pydantic
```
