# Arcee

>[Arcee](https://www.arcee.ai/about/about-us) は、私たちが「SLM」と呼ぶ小規模で専門的で安全性の高い拡張可能な言語モデルの開発と進化を可能にします。
> SLM アダプテーション システムと シームレスで安全な統合を提供することで、
> `Arcee` は企業が分野適応型言語モデルの可能性を最大限に活用し、
> 業務における革新的な変革を推進することを後押しします。

## インストールとセットアップ
