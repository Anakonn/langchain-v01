# 花びら

このページでは、LangChainの中でPetalsエコシステムの使い方について説明します。
2つの部分に分かれています: インストールとセットアップ、そして特定のPetalsラッパーへの参照。

## インストールとセットアップ

- `pip install petals`でインストールする
- Hugging FaceのAPIキーを取得し、環境変数(`HUGGINGFACE_API_KEY`)に設定する
