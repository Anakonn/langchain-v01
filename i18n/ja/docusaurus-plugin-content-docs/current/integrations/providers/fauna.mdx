# Fauna

>[Fauna](https://fauna.com/)は、文書の柔軟性と、リージョン、クラウド、またはグローバルにわたって拡張可能な、リレーショナルでACID準拠のデータベースの強力さを組み合わせた分散型のドキュメント-リレーショナルデータベースです。

## インストールとセットアップ

シークレットキーを取得する必要があります。
詳細な[ガイド](https://docs.fauna.com/fauna/current/learn/security_model/)を参照してください。

`fauna`パッケージをインストールする必要があります。

```bash
pip install -U fauna
```
