# Facebook - Meta

>[Meta Platforms, Inc.](https://www.facebook.com/), 事業を行う `Meta`、以前は `Facebook, Inc.` および `TheFacebook, Inc.` と呼ばれていた、アメリカの多国籍テクノロジー企業です。同社は `Facebook`、`Instagram`、`Threads`、`WhatsApp` などの製品やサービスを所有・運営しています。

## ベクトルストア

### Facebook Faiss

>[Facebook AI Similarity Search (Faiss)](https://engineering.fb.com/2017/03/29/data-infrastructure/faiss-a-library-for-efficient-similarity-search/)
> は密な ベクトルの効率的な類似検索とクラスタリングのためのライブラリです。RAMに収まらない大規模なベクトルセットでも検索できるアルゴリズムが含まれており、評価やパラメータチューニングのためのサポートコードも含まれています。

[Faiss ドキュメント](https://faiss.ai/)

`faiss` Pythonパッケージをインストールする必要があります。

```bash
pip install faiss-gpu # For CUDA 7.5+ supported GPU's.
```

または

```bash
pip install faiss-cpu # For CPU Installation
```

[使用例](/docs/integrations/vectorstores/faiss)をご覧ください。

```python
<!--IMPORTS:[{"imported": "FAISS", "source": "langchain_community.vectorstores", "docs": "https://api.python.langchain.com/en/latest/vectorstores/langchain_community.vectorstores.faiss.FAISS.html", "title": "Facebook - Meta"}]-->
```
