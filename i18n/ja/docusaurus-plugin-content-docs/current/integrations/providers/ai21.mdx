# AI21 研究所

>[AI21 研究所](https://www.ai21.com/about)は自然言語処理(NLP)を専門とする会社で、自然言語を理解し生成するAIシステムを開発しています。

このページでは、`LangChain`内で`AI21`エコシステムを使用する方法について説明します。

## インストールと設定

- AI21 APIキーを取得し、環境変数(`AI21_API_KEY`)に設定します
- Pythonパッケージをインストールします:

```bash
pip install langchain-ai21
```
