# MediaWikiダンプ

>[MediaWikiのXMLダンプ](https://www.mediawiki.org/wiki/Manual:Importing_XML_dumps)には、ウィキのコンテンツ(すべての改訂版のウィキページ)が含まれていますが、サイト関連のデータは含まれていません。XMLダンプはウィキデータベースの完全なバックアップを作成するものではなく、ユーザーアカウント、画像、編集履歴などは含まれていません。

## インストールとセットアップ

いくつかのPythonパッケージをインストールする必要があります。

`mediawiki-utilities`はXMLスキーマ0.11をサポートしていますが、マージされていないブランチにあります。

```bash
pip install -qU git+https://github.com/mediawiki-utilities/python-mwtypes@updates_schema_0.11
```

`mediawiki-utilities mwxml`にはバグがあり、修正のPRが保留中です。

```bash
pip install -qU git+https://github.com/gdedrouas/python-mwxml@xml_format_0.11
pip install -qU mwparserfromhell
```
