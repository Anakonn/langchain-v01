# チェーンデスク

>[チェーンデスク](https://chaindesk.ai) は、個人データを大規模言語モデルに接続するのに役立つ [オープンソース](https://github.com/gmpetrov/databerry) のドキュメント検索プラットフォームです。

## インストールとセットアップ

チェーンデスクにサインアップし、データストアを作成し、データを追加してデータストアのAPIエンドポイントURLを取得する必要があります。
[APIキー](https://docs.chaindesk.ai/api-reference/authentication) が必要です。
