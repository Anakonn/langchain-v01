# VoyageAI

VoyageAIに関連するすべての機能

>[VoyageAI](https://www.voyageai.com/) Voyage AIは、ドメインや企業に合わせてカスタマイズされた埋め込みモデルを構築し、より良い検索品質を実現します。

## インストールとセットアップ

統合パッケージをインストールします

```bash
pip install langchain-voyageai
```

VoyageAI APIキーを取得し、環境変数(`VOYAGE_API_KEY`)に設定します
