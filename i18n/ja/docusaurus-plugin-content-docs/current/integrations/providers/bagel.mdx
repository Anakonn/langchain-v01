# ベーグル

> [ベーグル](https://www.bagel.net/) (`Open Vector Database for AI`), は AI データの GitHub のようなものです。
ユーザーがベクトルデータセットを作成、共有、管理できる共同プラットフォームです。独立開発者のプライベートプロジェクト、企業の内部コラボレーション、データ DAOs の公開貢献をサポートできます。

## インストールとセットアップ

```bash
pip install bagelML
```
