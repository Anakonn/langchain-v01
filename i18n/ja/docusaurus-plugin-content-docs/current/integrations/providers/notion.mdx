# Notion DB

>[Notion](https://www.notion.so/)は、カンバンボード、タスク、Wiki、データベースを統合した共同作業プラットフォームで、Markdownをサポートしています。ノートテイキング、知識とデータ管理、プロジェクトとタスク管理のための総合的なワークスペースです。

## インストールとセットアップ

すべての手順は以下の例に記載されています。
