# OracleAI ベクトル検索

Oracle AI ベクトル検索は、セマンティクスに基づいてデータを検索できるAI(人工知能)ワークロード向けに設計されています。Oracle AI ベクトル検索の最大の利点の1つは、構造化データに対する関係検索と非構造化データに対するセマンティック検索を1つのシステムで組み合わせられることです。

これは強力であり、特殊なベクトルデータベースを追加する必要がないため、複数のシステム間でデータが断片化するという問題を解消できます。

さらに、Oracleはデータベース技術の構築に長年取り組んできたため、ベクトルはOracle Databaseの最強の機能を活用できます。以下がその一例です:

 * パーティショニングサポート
 * Real Application Clustersによるスケーラビリティ
 * Exadata スマートスキャン
 * 地理的に分散したデータベースにおけるシャード処理
 * トランザクション
 * パラレルSQL
 * ディザスターリカバリ
 * セキュリティ
 * Oracle Machine Learning
 * Oracle Graph Database
 * Oracle Spatial and Graph
 * Oracle Blockchain
 * JSON
