# Infinispan VS

> [Infinispan](https://infinispan.org) Infinispanは、Javaオブジェクトから単なるテキストまで、あらゆるタイプのデータを保持できるキー/値データストアを提供するオープンソースのインメモリデータグリッドです。バージョン15以降、Infinispanはキャッシュ上でベクトル検索をサポートしています。

## インストールとセットアップ

Infinispanサーバーを実行するには[Get Started](https://infinispan.org/get-started/)を参照してください。認証は現在サポートされていないため、無効にすることをお勧めします。
