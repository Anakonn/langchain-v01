# NLPCloud

>[NLP Cloud](https://docs.nlpcloud.com/#introduction)は、最先端のAIエンジンを使用したり、独自のデータでエンジンを訓練したりできる人工知能プラットフォームです。

## インストールとセットアップ

- `nlpcloud`パッケージをインストールします。

```bash
pip install nlpcloud
```

- NLPCloudのAPIキーを取得し、環境変数(`NLPCLOUD_API_KEY`)に設定します。
