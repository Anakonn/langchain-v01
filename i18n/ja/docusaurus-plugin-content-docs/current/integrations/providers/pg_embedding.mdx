# Postgres Embedding

> [pg_embedding](https://github.com/neondatabase/pg_embedding) は、`Postgres`と`Hierarchical Navigable Small Worlds`アルゴリズムを使用した近似最近傍探索を使用したベクトル類似検索のためのオープンソースパッケージです。

## インストールとセットアップ

いくつかのPythonパッケージをインストールする必要があります。

```bash
pip install psycopg2-binary
```
