# PromptLayer

>[PromptLayer](https://docs.promptlayer.com/introduction)は、プロンプトエンジニアリングのためのプラットフォームです。
>また、リクエストの可視化、プロンプトのバージョン管理、使用状況の追跡など、LLMの可視性を高めるのにも役立ちます。
>
>`PromptLayer`には、LangChainと直接統合するLLMもありますが(例えば、
>[`PromptLayerOpenAI`](https://docs.promptlayer.com/languages/langchain))など)、
>コールバックを使用して`PromptLayer`をLangChainと統合することをお勧めします。

## インストールと設定

`PromptLayer`を使用するには、以下の手順を行う必要があります:
- `PromptLayer`アカウントを作成する
- APIトークンを作成し、環境変数(`PROMPTLAYER_API_KEY`)に設定する

Pythonパッケージをインストールします:

```bash
pip install promptlayer
```
