# Spreedly

>[Spreedly](https://docs.spreedly.com/)は、クレジットカードを安全に保管し、さまざまな決済ゲートウェイやサードパーティのAPIを使って取引を行うことができるサービスです。これは、カードのトークン化/保管サービスとゲートウェイおよびレシーバー統合サービスを同時に提供することで実現しています。Spreedlyによってトークン化された支払方法は`Spreedly`に保管されるため、カードを個別に保管し、ビジネスの要件に応じて異なる終点に渡すことができます。
