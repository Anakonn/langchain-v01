# Streamlit

>[Streamlit](https://streamlit.io/)は、データアプリを構築して共有するための高速な方法です。
>`Streamlit`は、純粋なPythonを使って、わずかな時間でデータスクリプトをシェアできるWebアプリに変換します。フロントエンドの経験は不要です。
>[streamlit.io/generative-ai](https://streamlit.io/generative-ai)でさらに多くの例を確認できます。

## インストールとセットアップ

`streamlit`Pythonパッケージをインストールする必要があります:

```bash
pip install streamlit
```
