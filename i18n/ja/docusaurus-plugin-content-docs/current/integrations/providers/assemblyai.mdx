# AssemblyAI

>[AssemblyAI](https://www.assemblyai.com/) は、音声認識、話者分離、音声要約など、さまざまなタスクのための `Speech AI` モデルを構築しています。
>`AssemblyAI's` Speech AIモデルには、通話、バーチャルミーティング、ポッドキャストなどの音声データに対する正確な音声認識、話者検出、感情分析、章検出、個人情報保護などの機能が含まれています。

## インストールとセットアップ

[API キー](https://www.assemblyai.com/dashboard/signup)を取得してください。

`assemblyai` パッケージをインストールします。

```bash
pip install -U assemblyai
```
