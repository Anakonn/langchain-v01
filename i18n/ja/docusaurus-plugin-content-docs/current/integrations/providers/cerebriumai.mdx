# CerebriumAI

>[Cerebrium](https://docs.cerebrium.ai/cerebrium/getting-started/introduction)は、サーバーレスのGPUインフラストラクチャプロバイダーです。
> 複数のLLMモデルへのAPIアクセスを提供しています。

[CerebriumAI ドキュメント](https://docs.cerebrium.ai/examples/langchain)の例をご覧ください。

## インストールとセットアップ

- Pythonパッケージをインストールします:

```bash
pip install cerebrium
```

- [CerebriumAIのAPIキーを取得](https://docs.cerebrium.ai/cerebrium/getting-started/installation)し、環境変数(`CEREBRIUMAI_API_KEY`)に設定します。
