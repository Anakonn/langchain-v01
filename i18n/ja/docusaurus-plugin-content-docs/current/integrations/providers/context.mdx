# コンテキスト

>[コンテキスト](https://context.ai/)は、LLMベースの製品やフィーチャーのユーザー分析を提供します。

## インストールとセットアップ

`context-python` Pythonパッケージをインストールする必要があります:

```bash
pip install context-python
```
