# Dropbox

>[Dropbox](https://en.wikipedia.org/wiki/Dropbox)は、従来のファイル、クラウドコンテンツ、Webショートカットをすべて1つの場所に集めるファイルホスティングサービスです。

## インストールと設定

詳細な[インストールガイド](/docs/integrations/document_loaders/dropbox#prerequisites)を参照してください。

```bash
pip install -U dropbox
```
