# アトラス

>[Nomic Atlas](https://docs.nomic.ai/index.html)は、小規模および大規模なデータセットを操作するためのプラットフォームです。

## インストールとセットアップ

- `pip install nomic`でPythonパッケージをインストールします
- `Nomic`はlangchainsのpoetry extrasにも含まれています `poetry install -E all`
