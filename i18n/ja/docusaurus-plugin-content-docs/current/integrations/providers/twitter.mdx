# Twitter

>[Twitter](https://twitter.com/)は、オンラインのソーシャルメディアおよびソーシャルネットワーキングサービスです。

## インストールとセットアップ

```bash
pip install tweepy
```

Twitterの`API`トークンでローダーを初期化し、Twitterの`ユーザー名`を設定する必要があります。
