# 美しいスープ

>[美しいスープ](https://www.crummy.com/software/BeautifulSoup/)は、HTMLおよびXML文書(不完全なマークアップ、つまり閉じられていないタグを含む)を解析するためのPythonパッケージです。
>これは、HTMLからデータを抽出するのに役立つ解析ツリーを作成します。[3]

## インストールと設定

```bash
pip install beautifulsoup4
```

## ドキュメントトランスフォーマー

[使用例](/docs/integrations/document_transformers/beautiful_soup)を参照してください。

```python
from langchain_community.document_loaders import BeautifulSoupTransformer
```
