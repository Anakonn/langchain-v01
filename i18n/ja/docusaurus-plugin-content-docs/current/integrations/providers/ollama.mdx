# Ollama

>[Ollama](https://ollama.ai/)は、Pythonライブラリです。LLaMA2などのオープンソースの大規模言語モデルをローカルで実行することができます。

>`Ollama`は、Modelfileで定義された単一のパッケージにモデルの重み、設定、およびデータをバンドルします。
>セットアップと構成の詳細、GPU使用を最適化します。
>サポートされているモデルとモデルバリアントの完全なリストについては、[Ollamaモデルライブラリ](https://ollama.ai/library)を参照してください。

`Ollama`をLangChainで使用する方法の詳細については、[このガイド](/docs/guides/development/local_llms#quickstart)を参照してください。

## インストールとセットアップ

ローカルのOllamaインスタンスを設定して実行するには、[これらの手順](https://github.com/jmorganca/ollama?tab=readme-ov-file#ollama)に従ってください。
使用するには、`ANYSCALE_API_BASE`と`ANYSCALE_API_KEY`の環境変数を設定する必要があります。
