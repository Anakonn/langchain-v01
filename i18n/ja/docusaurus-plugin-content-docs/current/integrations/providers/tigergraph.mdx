# TigerGraph

>[TigerGraph](https://www.tigergraph.com/tigergraph-db/)は、ネイティブに分散された高性能なグラフデータベースです。
> 頂点とエッジのグラフ形式でデータを保存することで、
> LLMレスポンスを深化させるための理想的な関係性が生まれます。

## インストールとセットアップ

[TigerGraphデータベースに接続する方法](https://docs.tigergraph.com/pytigergraph/current/getting-started/connection)の手順に従ってください。

PythonのSDKをインストールします:

```bash
pip install pyTigerGraph
```
