# Chargeurs de documents

:::info
Rendez-vous sur [Intégrations](/docs/integrations/document_loaders/) pour la documentation sur les intégrations de chargeurs de documents intégrés avec des outils tiers.
:::

Utilisez les chargeurs de documents pour charger des données à partir d'une source en tant que `Document`s. Un `Document` est un morceau de texte et des métadonnées associées. Par exemple, il existe des chargeurs de documents pour charger un simple fichier `.txt`, pour charger le contenu textuel de n'importe quelle page Web, ou même pour charger une transcription d'une vidéo YouTube.

Les chargeurs de documents fournissent une méthode "load" pour charger des données en tant que documents à partir d'une source configurée. Ils implémentent également une "lazy load" pour charger paresseusement les données en mémoire.

## Démarrer

Le chargeur le plus simple lit un fichier en tant que texte et le place tout entier dans un document.

```python
<!--IMPORTS:[{"imported": "TextLoader", "source": "langchain_community.document_loaders", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.text.TextLoader.html", "title": "Document loaders"}]-->
from langchain_community.document_loaders import TextLoader

loader = TextLoader("./index.md")
loader.load()
```
