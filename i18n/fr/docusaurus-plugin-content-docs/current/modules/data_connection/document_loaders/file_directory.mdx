# Répertoire de fichiers

Cela couvre comment charger tous les documents dans un répertoire.

En coulisse, par défaut, cela utilise le [UnstructuredLoader](/docs/integrations/document_loaders/unstructured_file).

```python
<!--IMPORTS:[{"imported": "DirectoryLoader", "source": "langchain_community.document_loaders", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.directory.DirectoryLoader.html", "title": "File Directory"}]-->
from langchain_community.document_loaders import DirectoryLoader
```

Nous pouvons utiliser le paramètre `glob` pour contrôler quels fichiers charger. Notez qu'ici, il ne charge pas le fichier `.rst` ni les fichiers `.html`.

```python
loader = DirectoryLoader('../', glob="**/*.md")
```

```python
docs = loader.load()
```

```python
len(docs)
```

```output
    1
```

## Afficher une barre de progression

Par défaut, une barre de progression ne sera pas affichée. Pour afficher une barre de progression, installez la bibliothèque `tqdm` (par exemple `pip install tqdm`) et définissez le paramètre `show_progress` sur `True`.

```python
loader = DirectoryLoader('../', glob="**/*.md", show_progress=True)
docs = loader.load()
```

```output
    Requirement already satisfied: tqdm in /Users/jon/.pyenv/versions/3.9.16/envs/microbiome-app/lib/python3.9/site-packages (4.65.0)


    0it [00:00, ?it/s]
```

## Utiliser le multithreading

Par défaut, le chargement se fait dans un seul thread. Pour utiliser plusieurs threads, définissez le drapeau `use_multithreading` sur true.

```python
loader = DirectoryLoader('../', glob="**/*.md", use_multithreading=True)
docs = loader.load()
```

## Changer la classe de chargeur

Par défaut, cela utilise la classe `UnstructuredLoader`. Cependant, vous pouvez facilement changer le type de chargeur.

```python
<!--IMPORTS:[{"imported": "TextLoader", "source": "langchain_community.document_loaders", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.text.TextLoader.html", "title": "File Directory"}]-->
from langchain_community.document_loaders import TextLoader
```

```python
loader = DirectoryLoader('../', glob="**/*.md", loader_cls=TextLoader)
```

```python
docs = loader.load()
```

```python
len(docs)
```

```output
    1
```

Si vous devez charger des fichiers de code source Python, utilisez le `PythonLoader`.

```python
<!--IMPORTS:[{"imported": "PythonLoader", "source": "langchain_community.document_loaders", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.python.PythonLoader.html", "title": "File Directory"}]-->
from langchain_community.document_loaders import PythonLoader
```

```python
loader = DirectoryLoader('../../../../../', glob="**/*.py", loader_cls=PythonLoader)
```

```python
docs = loader.load()
```

```python
len(docs)
```

```output
    691
```

Le fichier `example-non-utf8.txt` utilise un encodage différent, donc la fonction `load()` échoue avec un message d'erreur indiquant quel fichier n'a pas pu être décodé.

Avec le comportement par défaut de `TextLoader`, tout échec de chargement d'un des documents entraînera l'échec de tout le processus de chargement et aucun document ne sera chargé.

### B. Échec silencieux

Nous pouvons passer le paramètre `silent_errors` au `DirectoryLoader` pour ignorer les fichiers qui n'ont pas pu être chargés et continuer le processus de chargement.

```python
loader = DirectoryLoader(path, glob="**/*.txt", loader_cls=TextLoader, silent_errors=True)
docs = loader.load()
```

```output
    Error loading ../../../../../tests/integration_tests/examples/example-non-utf8.txt
```

```python
doc_sources = [doc.metadata['source']  for doc in docs]
doc_sources
```

```output
    ['../../../../../tests/integration_tests/examples/whatsapp_chat.txt',
     '../../../../../tests/integration_tests/examples/example-utf8.txt']
```

### C. Détecter automatiquement les encodages

Nous pouvons également demander à `TextLoader` de détecter automatiquement l'encodage du fichier avant d'échouer, en passant le paramètre `autodetect_encoding` à la classe de chargeur.

```python
text_loader_kwargs={'autodetect_encoding': True}
loader = DirectoryLoader(path, glob="**/*.txt", loader_cls=TextLoader, loader_kwargs=text_loader_kwargs)
docs = loader.load()
```

```python
doc_sources = [doc.metadata['source']  for doc in docs]
doc_sources
```

```output
    ['../../../../../tests/integration_tests/examples/example-non-utf8.txt',
     '../../../../../tests/integration_tests/examples/whatsapp_chat.txt',
     '../../../../../tests/integration_tests/examples/example-utf8.txt']
```
