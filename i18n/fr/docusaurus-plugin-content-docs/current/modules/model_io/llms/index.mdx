# LLMs

Les modèles de langage à grande échelle (LLMs) sont un composant essentiel de LangChain.
LangChain ne fournit pas ses propres LLMs, mais offre plutôt une interface standard pour interagir avec de nombreux LLMs différents. Plus précisément, cette interface prend une chaîne de caractères en entrée et renvoie une chaîne de caractères.

Il existe de nombreux fournisseurs de LLM (OpenAI, Cohere, Hugging Face, etc.) - la classe `LLM` est conçue pour fournir une interface standard pour tous.
