# LangChain cookbook

Exemples de code pour construire des applications avec LangChain, avec un accent mis sur des exemples plus appliqués et plus complets que ceux contenus dans la [documentation principale](https://python.langchain.com).

[oracleai_demo.ipynb](https://github.com/langchain-ai/langchain/tree/master/cookbook/oracleai_demo.ipynb) | Ce guide décrit comment utiliser Oracle AI Vector Search aux côtés de Langchain pour une pipeline RAG de bout en bout, en fournissant des exemples étape par étape. Le processus comprend le chargement de documents à partir de diverses sources à l'aide d'OracleDocLoader, leur résumé soit à l'intérieur soit à l'extérieur de la base de données avec OracleSummary, et la génération d'embeddings de manière similaire via OracleEmbeddings. Il couvre également le découpage des documents selon des exigences spécifiques à l'aide des capacités avancées d'Oracle à partir d'OracleTextSplitter, et enfin, le stockage et l'indexation de ces documents dans un Vector Store pour les interroger avec OracleVS.
