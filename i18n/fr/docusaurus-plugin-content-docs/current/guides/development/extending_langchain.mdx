# Extension de LangChain

L'extension des abstractions de base de LangChain, que vous prévoyiez de contribuer au dépôt open-source ou de construire une intégration interne sur mesure, est encouragée.

Consultez ces guides pour créer vos propres classes personnalisées pour les modules suivants :

- [Modèles de discussion](/docs/modules/model_io/chat/custom_chat_model) pour interagir avec les modèles de langage adaptés à la discussion.
- [LLM](/docs/modules/model_io/llms/custom_llm) pour interagir avec les modèles de langage de texte.
- [Analyseurs de sortie](/docs/modules/model_io/output_parsers/custom) pour gérer les sorties des modèles de langage.
