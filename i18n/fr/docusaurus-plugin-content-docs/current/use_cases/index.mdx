# Cas d'utilisation

Cette section contient des parcours et des techniques pour les tâches d'utilisation finales courantes.

Si vous cherchez à construire quelque chose de spécifique ou que vous êtes plus un apprenant pratique, essayez-en un !
Bien qu'ils fassent référence à des blocs de construction qui sont expliqués plus en détail dans d'autres sections, nous encourageons absolument
les gens à commencer par les parcourir et à décomposer le code dans un contexte du monde réel.

Ou, si vous préférez d'abord examiner les fondamentaux, vous pouvez consulter les sections sur [Expression Language](/docs/expression_language/get_started)
et les différents [composants](/docs/modules) que LangChain fournit pour acquérir davantage de connaissances de base.
