# CnosDB

> [CnosDB](https://github.com/cnosdb/cnosdb) est une base de données de séries chronologiques distribuée open-source avec des performances élevées, un taux de compression élevé et une grande facilité d'utilisation.

## Installation et configuration

```python
pip install cnos-connector
```

## Connexion à CnosDB

Vous pouvez vous connecter à CnosDB en utilisant la méthode `SQLDatabase.from_cnosdb()`.

### Syntaxe

```python
def SQLDatabase.from_cnosdb(url: str = "127.0.0.1:8902",
                              user: str = "root",
                              password: str = "",
                              tenant: str = "cnosdb",
                              database: str = "public")
```

Args :
1. url (str) : Le nom d'hôte et le numéro de port de la connexion HTTP du service CnosDB, sans "http://" ou "https://", avec une valeur par défaut de "127.0.0.1:8902".
2. user (str) : Le nom d'utilisateur utilisé pour se connecter au service CnosDB, avec une valeur par défaut de "root".
3. password (str) : Le mot de passe de l'utilisateur se connectant au service CnosDB, avec une valeur par défaut de "".
4. tenant (str) : Le nom du tenant utilisé pour se connecter au service CnosDB, avec une valeur par défaut de "cnosdb".
5. database (str) : Le nom de la base de données dans le tenant CnosDB.
