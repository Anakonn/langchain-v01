# Bagel

> [Bagel](https://www.bagel.net/) (`Open Vector Database for AI`), est comme GitHub pour les données IA.
Il s'agit d'une plateforme collaborative où les utilisateurs peuvent créer,
partager et gérer des jeux de données vectoriels. Il peut prendre en charge des projets privés pour les développeurs indépendants,
des collaborations internes pour les entreprises et des contributions publiques pour les DAO de données.

## Installation et configuration

```bash
pip install bagelML
```
