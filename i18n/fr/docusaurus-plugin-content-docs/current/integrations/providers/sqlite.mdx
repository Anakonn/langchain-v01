# SQLite

>[SQLite](https://en.wikipedia.org/wiki/SQLite) est un moteur de base de données écrit dans le
> langage de programmation C. Ce n'est pas une application autonome ; c'est plutôt une bibliothèque que les
> développeurs de logiciels intègrent dans leurs applications. En tant que tel, il appartient à la famille des
> bases de données intégrées. C'est le moteur de base de données le plus largement déployé, car il est
> utilisé par plusieurs des principaux navigateurs web, systèmes d'exploitation, téléphones mobiles et autres systèmes intégrés.

## Installation et configuration

Nous devons installer le package python `SQLAlchemy`.

```bash
pip install SQLAlchemy
```
