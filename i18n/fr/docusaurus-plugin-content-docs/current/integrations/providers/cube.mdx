# Cube

>[Cube](https://cube.dev/) est la couche sémantique pour construire des applications de données. Elle aide les ingénieurs de données et les développeurs d'applications à accéder aux données des magasins de données modernes, à les organiser en définitions cohérentes et à les fournir à chaque application.

## Installation et configuration

Nous devons obtenir la clé API et l'URL de l'instance Cube. Voir
[ces instructions](https://cube.dev/docs/product/apis-integrations/rest-api#configuration-base-path).
