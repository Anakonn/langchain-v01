# YouTube

>[YouTube](https://www.youtube.com/) est une plateforme de partage de vidéos en ligne et de médias sociaux de Google.
> Nous téléchargeons les transcriptions et les informations vidéo de `YouTube`.

## Installation et configuration

```bash
pip install youtube-transcript-api
pip install pytube
```

Voir un [exemple d'utilisation](/docs/integrations/document_loaders/youtube_transcript).
