# Streamlit

>[Streamlit](https://streamlit.io/) est un moyen plus rapide de construire et de partager des applications de données.
>`Streamlit` transforme les scripts de données en applications Web partageables en quelques minutes. Le tout en Python pur. Aucune expérience en développement front-end requise.
>Voir plus d'exemples sur [streamlit.io/generative-ai](https://streamlit.io/generative-ai).

## Installation et configuration

Nous devons installer le package Python `streamlit` :

```bash
pip install streamlit
```
