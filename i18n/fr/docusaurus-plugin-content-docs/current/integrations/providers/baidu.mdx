# Baidu

>[Baidu Cloud](https://cloud.baidu.com/) est un service cloud fourni par `Baidu, Inc.`,
> dont le siège est à Pékin. Il offre un service de stockage cloud, un logiciel client,
> la gestion de fichiers, le partage de ressources et l'intégration tierce.

## Installation et configuration

Inscrivez-vous et obtenez les clés `Qianfan` `AK` et `SK` [ici](https://cloud.baidu.com/product/wenxinworkshop).
