# Meilisearch

> [Meilisearch](https://meilisearch.com) est un moteur de recherche open-source, ultra-rapide et hyper
> pertinent.
> Il est livré avec de superbes paramètres par défaut pour aider les développeurs à construire des expériences de recherche rapides.
>
> Vous pouvez [héberger vous-même Meilisearch](https://www.meilisearch.com/docs/learn/getting_started/installation#local-installation)
> ou l'exécuter sur [Meilisearch Cloud](https://www.meilisearch.com/pricing).
>
>`Meilisearch v1.3` prend en charge la recherche vectorielle.

## Installation et configuration

Voir un [exemple d'utilisation](/docs/integrations/vectorstores/meilisearch) pour obtenir des instructions de configuration détaillées.

Nous devons installer le package python `meilisearch`.

```bash
pip install meilisearch
```
