# ForefrontAI

Cette page couvre comment utiliser l'écosystème ForefrontAI dans LangChain.
Elle est divisée en deux parties : l'installation et la configuration, puis les références aux wrappers ForefrontAI spécifiques.

## Installation et configuration

- Obtenez une clé d'API ForefrontAI et définissez-la comme variable d'environnement (`FOREFRONTAI_API_KEY`)

## Wrappers

### LLM

Il existe un wrapper LLM ForefrontAI, que vous pouvez accéder avec

```python
<!--IMPORTS:[{"imported": "ForefrontAI", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.forefrontai.ForefrontAI.html", "title": "ForefrontAI"}]-->
from langchain_community.llms import ForefrontAI
```
