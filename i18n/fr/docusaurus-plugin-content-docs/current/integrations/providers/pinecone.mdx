# Pinecone

>[Pinecone](https://docs.pinecone.io/docs/overview) est une base de données vectorielle avec de larges fonctionnalités.

## Installation et configuration

Installez le SDK Python :

```bash
pip install langchain-pinecone
```

## Magasin vectoriel

Il existe un wrapper autour des index Pinecone, vous permettant de l'utiliser comme un vectorstore,
que ce soit pour la recherche sémantique ou la sélection d'exemples.

```python
<!--IMPORTS:[{"imported": "PineconeVectorStore", "source": "langchain_pinecone", "docs": "https://api.python.langchain.com/en/latest/vectorstores/langchain_pinecone.vectorstores.PineconeVectorStore.html", "title": "Pinecone"}]-->
```
