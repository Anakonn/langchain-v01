# Moniteur LL

>[Moniteur LL](https://llmonitor.com?utm_source=langchain&utm_medium=py&utm_campaign=docs) est une plateforme d'observabilité open source qui fournit des analyses de coûts et d'utilisation, le suivi des utilisateurs, le traçage et des outils d'évaluation.

## Installation et configuration

Créez un compte sur [llmonitor.com](https://llmonitor.com?utm_source=langchain&utm_medium=py&utm_campaign=docs), puis copiez l'`ID de suivi` de votre nouvelle application.

Une fois que vous l'avez, définissez-le en tant que variable d'environnement en exécutant :

```bash
export LLMONITOR_APP_ID="..."
```
