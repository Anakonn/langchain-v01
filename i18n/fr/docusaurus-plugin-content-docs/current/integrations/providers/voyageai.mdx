# VoyageAI

Toute la fonctionnalité liée à VoyageAI

>[VoyageAI](https://www.voyageai.com/) Voyage AI construit des modèles d'intégration, personnalisés pour votre domaine et votre entreprise, pour une meilleure qualité de récupération.

## Installation et configuration

Installez le package d'intégration avec

```bash
pip install langchain-voyageai
```

Obtenez une clé d'API VoyageAI et définissez-la comme une variable d'environnement (`VOYAGE_API_KEY`)
