# DuckDB

>[DuckDB](https://duckdb.org/) est un système de gestion de base de données OLAP SQL en cours d'exécution.

## Installation et configuration

Tout d'abord, vous devez installer le package python `duckdb`.

```bash
pip install duckdb
```
