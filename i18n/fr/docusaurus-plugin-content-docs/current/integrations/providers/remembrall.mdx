# Remembrall

>[Remembrall](https://remembrall.dev/) est une plateforme qui donne à un modèle de langue une mémoire à long terme, une génération augmentée par la récupération et une observabilité complète.

## Installation et configuration

Pour commencer, [connectez-vous avec Github sur la plateforme Remembrall](https://remembrall.dev/login) et copiez votre [clé API à partir de la page des paramètres](https://remembrall.dev/dashboard/settings).
