# StochasticAI

Cette page couvre comment utiliser l'écosystème StochasticAI dans LangChain.
Elle est divisée en deux parties : l'installation et la configuration, puis les références aux wrappers StochasticAI spécifiques.

## Installation et configuration

- Installer avec `pip install stochasticx`
- Obtenir une clé d'API StochasticAI et la définir en tant que variable d'environnement (`STOCHASTICAI_API_KEY`)

## Wrappers

### LLM

Il existe un wrapper LLM StochasticAI, que vous pouvez accéder avec

```python
<!--IMPORTS:[{"imported": "StochasticAI", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.stochasticai.StochasticAI.html", "title": "StochasticAI"}]-->
from langchain_community.llms import StochasticAI
```
