# DeepInfra

>[DeepInfra](https://deepinfra.com/docs) nous permet d'exécuter les
> [derniers modèles d'apprentissage automatique](https://deepinfra.com/models) avec facilité.
> DeepInfra s'occupe de tout le travail lourd lié à l'exécution, à la mise à l'échelle et à la surveillance
> des modèles. Les utilisateurs peuvent se concentrer sur leur application et intégrer les modèles avec de simples appels d'API REST.

>DeepInfra fournit des [exemples](https://deepinfra.com/docs/advanced/langchain) d'intégration avec LangChain.

Cette page couvre comment utiliser l'écosystème `DeepInfra` dans `LangChain`.
Elle est divisée en deux parties : l'installation et la configuration, puis les références aux wrappers DeepInfra spécifiques.

## Installation et configuration

- Obtenez votre clé d'API DeepInfra à partir de ce lien [ici](https://deepinfra.com/).
- Obtenez une clé d'API DeepInfra et définissez-la comme une variable d'environnement (`DEEPINFRA_API_TOKEN`)

## Modèles disponibles

DeepInfra fournit une gamme de LLM open source prêts pour le déploiement.

Vous pouvez voir les modèles pris en charge pour
[la génération de texte](https://deepinfra.com/models?type=text-generation) et
[les embeddings](https://deepinfra.com/models?type=embeddings).

Vous pouvez consulter une [liste des paramètres de requête et de réponse](https://deepinfra.com/meta-llama/Llama-2-70b-chat-hf/api).

Les modèles de chat [suivent l'API openai](https://deepinfra.com/meta-llama/Llama-2-70b-chat-hf/api?example=openai-http)
