# Iugu

>[Iugu](https://www.iugu.com/) est une entreprise brésilienne de services et de logiciels en tant que service (SaaS). Elle offre des logiciels de traitement des paiements et des interfaces de programmation d'applications pour les sites web de commerce électronique et les applications mobiles.
