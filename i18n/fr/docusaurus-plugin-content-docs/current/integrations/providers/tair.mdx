# Tair

>[Alibaba Cloud Tair](https://www.alibabacloud.com/help/en/tair/latest/what-is-tair) est un service de base de données en mémoire native du cloud
> développé par `Alibaba Cloud`. Il fournit des modèles de données riches et des capacités de niveau entreprise pour
> prendre en charge vos scénarios en ligne en temps réel tout en maintenant une compatibilité complète avec `Redis` open-source.
> `Tair` introduit également des instances optimisées pour la mémoire persistante qui sont basées sur
> un nouveau support de stockage de mémoire non volatile (NVM).

## Installation et configuration

Installez le SDK Python Tair :

```bash
pip install tair
```
