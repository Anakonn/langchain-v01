# HTML vers texte

>[html2text](https://github.com/Alir3z4/html2text/) est un package Python qui convertit une page `HTML` en un texte `ASCII` clair et facile à lire.

L'ASCII est également un `Markdown` valide (un format texte-vers-HTML).

## Installation et configuration

```bash
pip install html2text
```

## Transformateur de document

Voir un [exemple d'utilisation](/docs/integrations/document_transformers/html2text).

```python
from langchain_community.document_loaders import Html2TextTransformer
```
