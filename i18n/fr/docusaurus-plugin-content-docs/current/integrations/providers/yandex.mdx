# Yandex

Toutes les fonctionnalités liées à Yandex Cloud

>[Yandex Cloud](https://cloud.yandex.com/en/) est une plateforme de cloud public.

## Installation et configuration

Le SDK Yandex Cloud peut être installé via pip depuis PyPI :

```bash
pip install yandexcloud
```
