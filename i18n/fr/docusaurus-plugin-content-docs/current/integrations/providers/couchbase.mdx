# Couchbase

>[Couchbase](http://couchbase.com/) est une base de données cloud NoSQL distribuée primée
> qui offre une polyvalence, des performances, une évolutivité et une valeur financière inégalées
> pour toutes vos applications cloud, mobiles, d'IA et de calcul en périphérie.

## Installation et configuration

Nous devons installer le package `couchbase`.

```bash
pip install couchbase
```
