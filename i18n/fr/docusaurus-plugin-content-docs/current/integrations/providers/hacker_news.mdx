# Hacker News

>[Hacker News](https://en.wikipedia.org/wiki/Hacker_News) (parfois abrégé en `HN`) est un site d'actualités sociales
> se concentrant sur l'informatique et l'entrepreneuriat. Il est géré par le fonds d'investissement et l'incubateur de startups
> `Y Combinator`. En général, le contenu qui peut être soumis est défini comme "tout ce qui satisfait
> la curiosité intellectuelle de quelqu'un."

## Installation et configuration

Il n'y a pas de configuration spéciale pour cela.
