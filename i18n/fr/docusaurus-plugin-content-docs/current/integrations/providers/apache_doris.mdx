# Apache Doris

>[Apache Doris](https://doris.apache.org/) est un entrepôt de données moderne pour l'analyse en temps réel.
Il offre une analyse ultra-rapide sur des données en temps réel à grande échelle.

>Généralement, `Apache Doris` est classé dans la catégorie OLAP, et il a montré des performances excellentes
> dans [ClickBench — un benchmark pour les SGBD analytiques](https://benchmark.clickhouse.com/).
> Étant donné qu'il dispose d'un moteur d'exécution vectorisé ultra-rapide, il peut également être utilisé comme un vectordb rapide.

## Installation et configuration

```bash
pip install pymysql
```
