# AssemblyAI

>[AssemblyAI](https://www.assemblyai.com/) construit des modèles `Speech AI` pour des tâches comme
la transcription de la parole, la diarisation des locuteurs, le résumé de la parole et plus encore.
> Les modèles `Speech AI` d'`AssemblyAI` incluent une transcription précise de la parole pour les données vocales
> (comme les appels, les réunions virtuelles et les podcasts), la détection des locuteurs, l'analyse des sentiments,
> la détection des chapitres, la suppression des informations d'identification personnelle.

## Installation et configuration

Obtenez votre [clé d'API](https://www.assemblyai.com/dashboard/signup).

Installez le package `assemblyai`.

```bash
pip install -U assemblyai
```
