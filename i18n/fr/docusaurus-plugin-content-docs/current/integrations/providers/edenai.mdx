# Eden AI

>[Interface utilisateur (UI) d'Eden AI](https://docs.edenai.co/docs/getting-started-with-eden-ai)
> est conçue pour gérer les projets d'IA. Avec `Eden AI Portal`,
> vous pouvez effectuer de l'IA sans code en utilisant les meilleurs moteurs du marché.

## Installation et configuration

L'accès à l'API Eden AI nécessite une clé API, que vous pouvez obtenir en
[créant un compte](https://app.edenai.run/user/register) et
en vous rendant [ici](https://app.edenai.run/admin/account/settings).

## Outils

Eden AI fournit une liste d'outils qui donnent à votre agent la capacité d'effectuer de multiples tâches, telles que :
* reconnaissance vocale
* synthèse vocale
* détection de contenu explicite dans le texte
* détection de contenu explicite dans les images
* détection d'objets
* extraction de données de factures par OCR
* extraction de données de pièces d'identité par OCR

Voir un [exemple d'utilisation](/docs/integrations/tools/edenai_tools).
