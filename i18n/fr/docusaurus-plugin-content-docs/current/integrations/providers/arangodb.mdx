# ArangoDB

>[ArangoDB](https://github.com/arangodb/arangodb) est un système de base de données de graphe évolutif pour
> tirer parti des données connectées, plus rapidement. Graphes natifs, un moteur de recherche intégré et une prise en charge de JSON, via un seul langage de requête. ArangoDB s'exécute sur site, dans le cloud - partout.

## Installation et configuration

Installez le package [ArangoDB Python Driver](https://github.com/ArangoDB-Community/python-arango) avec

```bash
pip install python-arango
```
