# Docugami

>[Docugami](https://docugami.com) convertit les documents d'entreprise en un graphique de connaissances XML de document, générant des forêts
> d'arbres sémantiques XML représentant des documents entiers. Il s'agit d'une représentation riche qui inclut les caractéristiques sémantiques et
> structurelles de diverses parties du document sous forme d'arbre XML.

## Installation et configuration

```bash
pip install dgml-utils
pip install docugami-langchain
```

## Chargeur de documents

Voir un [exemple d'utilisation](/docs/integrations/document_loaders/docugami).

```python
from docugami_langchain.document_loaders import DocugamiLoader
```
