# Trubrics

>[Trubrics](https://trubrics.com) est une plateforme d'analyse des utilisateurs d'un LLM qui vous permet de collecter, d'analyser et de gérer les invites et les commentaires des utilisateurs sur les modèles d'IA.
>
>Consultez le [dépôt Trubrics](https://github.com/trubrics/trubrics-sdk) pour plus d'informations sur `Trubrics`.

## Installation et configuration

Nous devons installer le package Python `trubrics` :

```bash
pip install trubrics
```
