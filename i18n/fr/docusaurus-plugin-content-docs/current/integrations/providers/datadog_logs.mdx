# Journaux Datadog

>[Datadog](https://www.datadoghq.com/) est une plateforme de surveillance et d'analyse pour les applications à l'échelle du cloud.

## Installation et configuration

```bash
pip install datadog_api_client
```

Nous devons initialiser le chargeur avec la clé API Datadog et la clé APP, et nous devons configurer la requête pour extraire les journaux souhaités.
