# Apify

>[Apify](https://apify.com) est une plateforme cloud pour le web scraping et l'extraction de données,
>qui fournit un [écosystème](https://apify.com/store) de plus d'un millier
>d'applications prêtes à l'emploi appelées *Actors* pour divers cas d'utilisation de scraping, de crawling et d'extraction.

[![Acteurs Apify](/img/ApifyActors.png)](https://apify.com/store)

Cette intégration vous permet d'exécuter des Actors sur la plateforme `Apify` et de charger leurs résultats dans LangChain pour alimenter vos index vectoriels avec des documents et des données du web, par exemple pour générer des réponses à partir de sites Web avec de la documentation,
des blogs ou des bases de connaissances.

## Installation et configuration

- Installez le client API Apify pour Python avec `pip install apify-client`
- Obtenez votre [jeton API Apify](https://console.apify.com/account/integrations) et définissez-le soit comme
  une variable d'environnement (`APIFY_API_TOKEN`) soit transmettez-le à `ApifyWrapper` comme `apify_api_token` dans le constructeur.
