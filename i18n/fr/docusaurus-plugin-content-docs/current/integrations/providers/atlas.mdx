# Atlas

>[Nomic Atlas](https://docs.nomic.ai/index.html) est une plateforme pour interagir avec des jeux de données non structurés, à petite et à grande échelle.

## Installation et configuration

- Installez le package Python avec `pip install nomic`
- `Nomic` est également inclus dans les extras poetry de langchains `poetry install -E all`
