# Doctran

>[Doctran](https://github.com/psychic-api/doctran) est un package python. Il utilise des LLM et des bibliothèques NLP open-source pour transformer du texte brut en documents propres, structurés et riches en informations, optimisés pour la récupération dans un espace vectoriel. Vous pouvez considérer `Doctran` comme une boîte noire où des chaînes de caractères désordonnées entrent et des chaînes de caractères propres et étiquetées en sortent.

## Installation et configuration

```bash
pip install doctran
```
