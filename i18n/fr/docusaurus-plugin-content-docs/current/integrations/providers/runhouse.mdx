# Runhouse

Cette page couvre comment utiliser l'écosystème [Runhouse](https://github.com/run-house/runhouse) dans LangChain.
Elle est divisée en trois parties : installation et configuration, LLMs et Embeddings.

## Installation et configuration

- Installez le SDK Python avec `pip install runhouse`
- Si vous souhaitez utiliser un cluster à la demande, vérifiez vos identifiants cloud avec `sky check`

## LLMs auto-hébergés

Pour un LLM de base auto-hébergé, vous pouvez utiliser la classe `SelfHostedHuggingFaceLLM`. Pour des LLMs plus personnalisés, vous pouvez utiliser la classe parente `SelfHostedPipeline`.
