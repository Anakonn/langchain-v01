# lakeFS

>[lakeFS](https://docs.lakefs.io/) fournit un contrôle de version évolutif sur
> le lac de données et utilise une sémantique similaire à Git pour créer et accéder à ces versions.

## Installation et configuration

Obtenez l'`ENDPOINT`, la `LAKEFS_ACCESS_KEY` et la `LAKEFS_SECRET_KEY`.
Vous pouvez trouver les instructions d'installation [ici](https://docs.lakefs.io/quickstart/launch.html).
