# Réseau IA

>[Réseau IA](https://www.ainetwork.ai/build-on-ain) est une blockchain de couche 1 conçue pour accueillir
> des modèles IA à grande échelle, utilisant un réseau GPU décentralisé alimenté par le
> jeton [$AIN](https://www.ainetwork.ai/token), enrichissant les `NFT` (`AINFTs`) pilotés par l'IA.

## Installation et configuration

Vous devez installer le package python `ain-py`.

```bash
pip install ain-py
```

Vous devez définir la variable d'environnement `AIN_BLOCKCHAIN_ACCOUNT_PRIVATE_KEY` sur votre clé privée de compte blockchain AIN.
