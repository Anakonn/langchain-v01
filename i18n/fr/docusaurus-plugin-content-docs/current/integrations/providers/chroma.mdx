# Chroma

>[Chroma](https://docs.trychroma.com/getting-started) est une base de données pour construire des applications d'IA avec des embeddings.

## Installation et configuration

```bash
pip install langchain-chroma
```

## VectorStore

Il existe un wrapper autour des bases de données vectorielles Chroma, vous permettant de l'utiliser comme un vectorstore,
que ce soit pour la recherche sémantique ou la sélection d'exemples.
