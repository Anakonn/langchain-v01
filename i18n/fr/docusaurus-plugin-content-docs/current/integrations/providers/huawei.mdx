# Huawei

>[Huawei Technologies Co., Ltd.](https://www.huawei.com/) est une entreprise technologique de communications numériques multinationale chinoise.
>
>[Huawei Cloud](https://www.huaweicloud.com/intl/en-us/product/) fournit une suite complète de services de cloud computing mondiaux.

## Installation et configuration

Pour accéder à `Huawei Cloud`, vous avez besoin d'un jeton d'accès.

Vous devez également installer une bibliothèque Python :

```bash
pip install -U esdk-obs-python
```
