# Neo4j

>Qu'est-ce que `Neo4j` ?

>- Neo4j est un `système de gestion de base de données open-source` qui se spécialise dans la technologie de base de données de graphes.
>- Neo4j vous permet de représenter et de stocker des données dans des nœuds et des arêtes, ce qui en fait un outil idéal pour gérer les données connectées et les relations.
>- Neo4j fournit un `langage de requête Cypher`, ce qui facilite l'interaction et l'interrogation de vos données de graphe.
>- Avec Neo4j, vous pouvez réaliser des `traversées et des requêtes de graphe à haute performance`, adaptées aux systèmes de production.

>Commencez à utiliser Neo4j en visitant [leur site web](https://neo4j.com/).

## Installation et configuration

- Installez le SDK Python avec `pip install neo4j`

## Construire un graphe de connaissances à partir de texte

Les données textuelles contiennent souvent des relations et des informations riches qui peuvent être utiles pour diverses analyses, moteurs de recommandation ou applications de gestion des connaissances.
L'API NLP de Diffbot permet d'extraire des entités, des relations et du sens sémantique à partir de données textuelles non structurées.
En couplant l'API NLP de Diffbot avec Neo4j, une base de données de graphes, vous pouvez créer des structures de graphe puissantes et dynamiques en fonction des informations extraites du texte.
Ces structures de graphe sont entièrement interrogeables et peuvent être intégrées dans diverses applications.
