# RWKV-4

Cette page couvre comment utiliser le wrapper `RWKV-4` dans LangChain.
Elle est divisée en deux parties : l'installation et la configuration, puis l'utilisation avec un exemple.

## Installation et configuration

- Installez le package Python avec `pip install rwkv`
- Installez le package Python du tokenizer avec `pip install tokenizer`
- Téléchargez un [modèle RWKV](https://huggingface.co/BlinkDL/rwkv-4-raven/tree/main) et placez-le dans le répertoire de votre choix
- Téléchargez le [fichier de jetons](https://raw.githubusercontent.com/BlinkDL/ChatRWKV/main/20B_tokenizer.json)

## Utilisation

### RWKV

Pour utiliser le wrapper RWKV, vous devez fournir le chemin du fichier du modèle pré-entraîné et la configuration du tokenizer.

```python
<!--IMPORTS:[{"imported": "RWKV", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.rwkv.RWKV.html", "title": "RWKV-4"}]-->
from langchain_community.llms import RWKV
```

# Testez le modèle

```python

def generate_prompt(instruction, input=None):
    if input:
        return f"""Below is an instruction that describes a task, paired with an input that provides further context. Write a response that appropriately completes the request.

# Instruction:
{instruction}

# Input:
{input}

# Response:
"""
    else:
        return f"""Below is an instruction that describes a task. Write a response that appropriately completes the request.

# Instruction:
{instruction}

# Response:
"""


model = RWKV(model="./models/RWKV-4-Raven-3B-v7-Eng-20230404-ctx4096.pth", strategy="cpu fp32", tokens_path="./rwkv/20B_tokenizer.json")
response = model.invoke(generate_prompt("Once upon a time, "))
```
