# Cloudflare

>[Cloudflare, Inc. (Wikipédia)](https://en.wikipedia.org/wiki/Cloudflare) est une entreprise américaine qui fournit
> des services de réseau de diffusion de contenu, de cybersécurité cloud, d'atténuation DDoS et
> de services d'enregistrement de domaines accrédités par l'ICANN.

>[Cloudflare Workers AI](https://developers.cloudflare.com/workers-ai/) vous permet d'exécuter des modèles d'apprentissage
> automatique, sur le réseau `Cloudflare`, à partir de votre code via l'API REST.
