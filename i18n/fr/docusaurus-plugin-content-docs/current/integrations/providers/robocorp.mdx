# Robocorp

>[Robocorp](https://robocorp.com/) aide à construire et à exploiter des travailleurs Python qui s'exécutent sans problème n'importe où à n'importe quelle échelle

## Installation et configuration

Vous devez installer le package python `langchain-robocorp` :

```bash
pip install langchain-robocorp
```

Vous aurez besoin d'une instance en cours d'exécution du serveur d'actions pour communiquer avec votre application agent. Consultez le [Démarrage rapide Robocorp](https://github.com/robocorp/robocorp#quickstart) pour savoir comment configurer le serveur d'actions et créer vos actions.

Vous pouvez amorcer un nouveau projet à l'aide de la commande `new` du serveur d'actions.

```bash
action-server new
cd ./your-project-name
action-server start
```
