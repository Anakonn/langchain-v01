# Zilliz

>[Zilliz Cloud](https://zilliz.com/doc/quick_start) est un service entièrement géré sur le cloud pour `LF AI Milvus®`,

## Installation et configuration

Installez le SDK Python :

```bash
pip install pymilvus
```
