# Elasticsearch

> [Elasticsearch](https://www.elastic.co/elasticsearch/) est un moteur de recherche et d'analyse distribué et RESTful.
> Il fournit un moteur de recherche en texte intégral distribué, multi-locataire avec une interface Web HTTP et des documents JSON sans schéma.

## Installation et configuration

Il existe deux façons de commencer avec Elasticsearch :

#### Installer Elasticsearch sur votre machine locale via docker

Exemple : Exécuter une instance Elasticsearch mono-nœud avec la sécurité désactivée. Ceci n'est pas recommandé pour une utilisation en production.

```bash
    docker run -p 9200:9200 -e "discovery.type=single-node" -e "xpack.security.enabled=false" -e "xpack.security.http.ssl.enabled=false" docker.elastic.co/elasticsearch/elasticsearch:8.9.0
```

#### Déployer Elasticsearch sur Elastic Cloud

Elastic Cloud est un service Elasticsearch géré. Inscrivez-vous pour un [essai gratuit](https://cloud.elastic.co/registration?utm_source=langchain&utm_content=documentation).

### Installer le client

```bash
pip install elasticsearch
pip install langchain-elasticsearch
```
