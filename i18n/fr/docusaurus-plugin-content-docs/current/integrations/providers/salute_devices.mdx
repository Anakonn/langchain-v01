# Saluer les appareils

Salute Devices fournit les modèles GigaChat LLM.

Pour plus d'informations sur la façon d'accéder à GigaChat [suivez ici](https://developers.sber.ru/docs/ru/gigachat/api/integration).

## Installation et configuration

Le package GigaChat peut être installé via pip à partir de PyPI :

```bash
pip install gigachat
```
