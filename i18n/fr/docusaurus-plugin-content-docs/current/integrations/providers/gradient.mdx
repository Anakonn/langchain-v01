# Gradient

>[Gradient](https://gradient.ai/) permet d'affiner et d'obtenir des compléments sur les LLM avec une simple API web.

## Installation et configuration

- Installez le SDK Python :

```bash
pip install gradientai
```

Obtenez un [jeton d'accès Gradient et un espace de travail](https://gradient.ai/) et définissez-les comme variables d'environnement (`Gradient_ACCESS_TOKEN`) et (`GRADIENT_WORKSPACE_ID`)
