# Postgres Embedding

> [pg_embedding](https://github.com/neondatabase/pg_embedding) est un package open-source pour
> la recherche de similarité vectorielle à l'aide de `Postgres` et de l'algorithme `Hierarchical Navigable Small Worlds`
> pour la recherche approximative des plus proches voisins.

## Installation et configuration

Nous devons installer plusieurs packages python.

```bash
pip install psycopg2-binary
```
