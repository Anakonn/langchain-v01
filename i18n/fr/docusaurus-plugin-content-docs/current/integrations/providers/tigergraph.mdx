# TigerGraph

>[TigerGraph](https://www.tigergraph.com/tigergraph-db/) est une base de données de graphes distribuée de manière native et à haute performance.
> Le stockage des données dans un format de graphe de sommets et d'arêtes conduit à des relations riches,
> idéal pour ancrer les réponses des LLM.

## Installation et configuration

Suivez les instructions [comment se connecter à la base de données `TigerGraph`](https://docs.tigergraph.com/pytigergraph/current/getting-started/connection).

Installez le SDK Python :

```bash
pip install pyTigerGraph
```
