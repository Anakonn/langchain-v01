# AnalyticDB

>[AnalyticDB pour PostgreSQL](https://www.alibabacloud.com/help/en/analyticdb-for-postgresql/latest/product-introduction-overview)
> est un service d'entreposage de données à traitement massivement parallèle (MPP)
> de [Alibaba Cloud](https://www.alibabacloud.com/)
> conçu pour analyser de grands volumes de données en ligne.

>`AnalyticDB pour PostgreSQL` est développé sur la base du projet open-source `Greenplum Database`
> et est enrichi d'extensions approfondies par `Alibaba Cloud`. AnalyticDB
> pour PostgreSQL est compatible avec la syntaxe ANSI SQL 2003 et les écosystèmes de bases de données PostgreSQL et
> Oracle. AnalyticDB pour PostgreSQL prend également en charge le stockage en ligne et
> le stockage en colonne. AnalyticDB pour PostgreSQL traite des pétaoctets de données hors ligne à un
> niveau de performance élevé et prend en charge une très grande concurrence.

Cette page couvre comment utiliser l'écosystème AnalyticDB dans LangChain.

## Installation et configuration

Vous devez installer le package python `sqlalchemy`.

```bash
pip install sqlalchemy
```
