# Faune

>[Faune](https://fauna.com/) est une base de données de documents relationnels distribuée
> qui combine la flexibilité des documents avec la puissance d'une base de données
> relationnelle, conforme à l'ACID, qui s'étend sur plusieurs régions, clouds ou le globe.

## Installation et configuration

Nous devons obtenir la clé secrète.
Voir le [guide](https://docs.fauna.com/fauna/current/learn/security_model/) détaillé.

Nous devons installer le package `fauna`.

```bash
pip install -U fauna
```
