# iFixit

>[iFixit](https://www.ifixit.com) est la plus grande communauté de réparation ouverte sur le web. Le site contient près de 100 000 manuels de réparation, 200 000 questions et réponses sur 42 000 appareils, et toutes les données sont sous licence `CC-BY-NC-SA 3.0`.

## Installation et configuration

Il n'y a pas de configuration spéciale pour cela.
