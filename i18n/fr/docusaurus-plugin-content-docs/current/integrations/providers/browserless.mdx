# Browserless

>[Browserless](https://www.browserless.io/docs/start) est un service qui vous permet d'exécuter des instances Chrome sans interface graphique dans le cloud. C'est un excellent moyen d'automatiser des tâches basées sur le navigateur à grande échelle sans avoir à vous soucier de la gestion de votre propre infrastructure.

## Installation et configuration

Nous devons obtenir la clé API [ici](https://www.browserless.io/pricing/).
