# Écrivain

Cette page couvre comment utiliser l'écosystème Writer dans LangChain.
Il est divisé en deux parties : l'installation et la configuration, puis les références aux wrappers Writer spécifiques.

## Installation et configuration

- Obtenez une clé d'API Writer et définissez-la comme une variable d'environnement (`WRITER_API_KEY`)

## Wrappers

### LLM

Il existe un wrapper LLM Writer, que vous pouvez accéder avec

```python
<!--IMPORTS:[{"imported": "Writer", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.writer.Writer.html", "title": "Writer"}]-->
```
