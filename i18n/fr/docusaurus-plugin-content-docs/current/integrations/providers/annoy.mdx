# Agacer

> [Agacer](https://github.com/spotify/annoy) (`Approximate Nearest Neighbors Oh Yeah`)
> est une bibliothèque C++ avec des liaisons Python pour rechercher des points dans l'espace qui sont
> proches d'un point de requête donné. Il crée également de grandes structures de données en lecture seule basées sur des fichiers
> qui sont mappées en mémoire afin que de nombreux processus puissent partager les mêmes données.

## Installation et configuration

```bash
pip install annoy
```
