# AI21 Labs

>[AI21 Labs](https://www.ai21.com/about) est une entreprise spécialisée dans le traitement du langage naturel (NLP), qui développe des systèmes d'IA capables de comprendre et de générer du langage naturel.

Cette page couvre comment utiliser l'écosystème `AI21` dans `LangChain`.

## Installation et configuration

- Obtenez une clé d'API AI21 et définissez-la comme une variable d'environnement (`AI21_API_KEY`)
- Installez le package Python :

```bash
pip install langchain-ai21
```
