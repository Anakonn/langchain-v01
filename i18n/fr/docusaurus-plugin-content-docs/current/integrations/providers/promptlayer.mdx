# PromptLayer

>[PromptLayer](https://docs.promptlayer.com/introduction) est une plateforme pour l'ingénierie des invites.
> Elle aide également à l'observabilité des LLM pour visualiser les requêtes, les versions d'invites et suivre l'utilisation.
>
>Bien que `PromptLayer` ait des LLM qui s'intègrent directement à LangChain (par exemple,
> [`PromptLayerOpenAI`](https://docs.promptlayer.com/languages/langchain)),
> l'utilisation d'un rappel est le moyen recommandé pour intégrer `PromptLayer` à LangChain.

## Installation et configuration

Pour travailler avec `PromptLayer`, nous devons :
- Créer un compte `PromptLayer`
- Créer un jeton d'API et le définir comme variable d'environnement (`PROMPTLAYER_API_KEY`)

Installer un package Python :

```bash
pip install promptlayer
```
