# Joplin

>[Joplin](https://joplinapp.org/) est une application de prise de notes open source. Elle capture vos pensées et y accède de manière sécurisée à partir de n'importe quel appareil.

## Installation et configuration

L'`API Joplin` nécessite un jeton d'accès.
Vous pouvez trouver les instructions d'installation [ici](https://joplinapp.org/api/references/rest_api/).
