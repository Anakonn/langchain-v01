# Wikipédia

>[Wikipédia](https://wikipedia.org/) est une encyclopédie en ligne multilingue gratuite, écrite et maintenue par une communauté de bénévoles, connus sous le nom de Wikipédiens, par le biais d'une collaboration ouverte et en utilisant un système d'édition wiki appelé MediaWiki. `Wikipédia` est l'ouvrage de référence le plus vaste et le plus lu de l'histoire.

## Installation et configuration

```bash
pip install wikipedia
```
