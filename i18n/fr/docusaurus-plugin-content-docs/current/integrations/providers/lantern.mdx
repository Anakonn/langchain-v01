# Lanterne

Cette page couvre comment utiliser la [Lanterne](https://github.com/lanterndata/lantern) dans LangChain.
Elle est divisée en deux parties : la configuration et les références aux wrappers Lanterne spécifiques.

## Configuration

1. La première étape consiste à créer une base de données avec l'extension `lantern` installée.

    Suivez les étapes du [Guide d'installation de Lanterne](https://github.com/lanterndata/lantern#-quick-install) pour installer la base de données et l'extension. L'image Docker est le moyen le plus simple pour commencer.

## Wrappers

### VectorStore

Il existe un wrapper autour des bases de données vectorielles Postgres, vous permettant de l'utiliser comme un vectorstore, que ce soit pour la recherche sémantique ou la sélection d'exemples.

Pour importer ce vectorstore :

```python
<!--IMPORTS:[{"imported": "Lantern", "source": "langchain_community.vectorstores", "docs": "https://api.python.langchain.com/en/latest/vectorstores/langchain_community.vectorstores.lantern.Lantern.html", "title": "Lantern"}]-->
from langchain_community.vectorstores import Lantern
```
