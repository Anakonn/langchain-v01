# Facebook - Meta

>[Meta Platforms, Inc.](https://www.facebook.com/), faisant affaire sous le nom de `Meta`, anciennement
> appelée `Facebook, Inc.`, et `TheFacebook, Inc.`, est une entreprise technologique
> multinationale américaine. La société possède et exploite `Facebook`, `Instagram`, `Threads`,
> et `WhatsApp`, entre autres produits et services.

## Modèles d'intégration

### LASER

>[LASER](https://github.com/facebookresearch/LASER) est une bibliothèque Python développée par
> l'équipe de recherche `Meta AI` et utilisée pour
> créer des embeddings de phrases multilingues pour
> [plus de 147 langues à partir du 25/2/2024](https://github.com/facebookresearch/flores/blob/main/flores200/README.md#languages-in-flores-200)

```bash

```

## Magasins de vecteurs

### Facebook Faiss

>[Facebook AI Similarity Search (Faiss)](https://engineering.fb.com/2017/03/29/data-infrastructure/faiss-a-library-for-efficient-similarity-search/)
> est une bibliothèque pour la recherche et le regroupement efficaces de vecteurs denses. Elle contient des algorithmes qui
> recherchent dans des ensembles de vecteurs de toute taille, jusqu'à ceux qui ne tiennent peut-être pas en RAM. Elle contient également du code
> d'assistance pour l'évaluation et le réglage des paramètres.

[Documentation Faiss](https://faiss.ai/).

Nous devons installer le package python `faiss`.

```bash
pip install faiss-gpu # For CUDA 7.5+ supported GPU's.
```

OU

```bash
pip install faiss-cpu # For CPU Installation
```

Voir un [exemple d'utilisation](/docs/integrations/vectorstores/faiss).
