# Activeloop Deep Lake

>[Activeloop Deep Lake](https://docs.activeloop.ai/) est un data lake pour les applications d'apprentissage profond, vous permettant de l'utiliser comme un magasin de vecteurs.

## Pourquoi Deep Lake ?

- Plus qu'un simple magasin de vecteurs (multimodal). Vous pouvez ensuite utiliser l'ensemble de données pour affiner vos propres modèles LLM.
- Stocke non seulement les embeddings, mais aussi les données d'origine avec un contrôle de version automatique.
- Vraiment serverless. Ne nécessite pas un autre service et peut être utilisé avec les principaux fournisseurs de cloud (`AWS S3`, `GCS`, etc.)

`Activeloop Deep Lake` prend en charge `SelfQuery Retrieval` :
[Activeloop Deep Lake Self Query Retrieval](/docs/integrations/retrievers/self_query/activeloop_deeplake_self_query)
