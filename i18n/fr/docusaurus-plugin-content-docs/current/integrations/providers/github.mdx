# GitHub

>[GitHub](https://github.com/) est une plateforme de développement qui permet aux développeurs de créer,
> stocker, gérer et partager leur code. Elle utilise le logiciel `Git`, offrant le
> contrôle de version distribué de Git ainsi que le contrôle d'accès, le suivi des bogues,
> les demandes de fonctionnalités logicielles, la gestion des tâches, l'intégration continue et les wikis pour chaque projet.

## Installation et configuration

Pour accéder à l'API GitHub, vous avez besoin d'un [jeton d'accès personnel](https://github.com/settings/tokens).
