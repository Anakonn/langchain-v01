# ElevenLabs

>[ElevenLabs](https://elevenlabs.io/about) est une entreprise de recherche et de déploiement d'IA vocale
> avec pour mission de rendre le contenu universellement accessible dans n'importe quelle langue et voix.
>
>`ElevenLabs` crée l'audio IA le plus réaliste, polyvalent et sensible au contexte,
> offrant la possibilité de générer de la parole dans des centaines de
> nouvelles et existantes voix dans 29 langues.

## Installation et configuration

Tout d'abord, vous devez configurer un compte ElevenLabs. Vous pouvez suivre les
[instructions ici](https://docs.elevenlabs.io/welcome/introduction).

Installez le package Python :

```bash
pip install elevenlabs
```
