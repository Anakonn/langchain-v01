# Label Studio

>[Label Studio](https://labelstud.io/guide/get_started) est une plateforme open-source de labellisation de données qui offre à LangChain une flexibilité lorsqu'il s'agit d'étiqueter des données pour l'affinage de modèles de langage à grande échelle (LLM). Elle permet également la préparation de données d'entraînement personnalisées et la collecte et l'évaluation des réponses grâce à des commentaires humains.

## Installation et configuration

Consultez le [guide d'installation de Label Studio](https://labelstud.io/guide/install) pour connaître les options d'installation.

Nous devons installer les packages Python `label-studio` et `label-studio-sdk-python` :

```bash
pip install label-studio label-studio-sdk
```
