# Anyscale

>[Anyscale](https://www.anyscale.com) est une plateforme pour exécuter, affiner et mettre à l'échelle les LLM via des API prêtes pour la production.
> [Anyscale Endpoints](https://docs.anyscale.com/endpoints/overview) servent de nombreux modèles open-source de manière rentable.

`Anyscale` fournit également [un exemple](https://docs.anyscale.com/endpoints/model-serving/examples/langchain-integration)
de la façon de configurer `LangChain` avec `Anyscale` pour des agents de chat avancés.

## Installation et configuration

- Obtenez une URL de service Anyscale, une route et une clé API, puis définissez-les comme variables d'environnement (`ANYSCALE_SERVICE_URL`,`ANYSCALE_SERVICE_ROUTE`, `ANYSCALE_SERVICE_TOKEN`).
- Veuillez consulter [la documentation Anyscale](https://www.anyscale.com/get-started) pour plus de détails.

Nous devons installer le package `openai` :

```bash
pip install openai
```
