# OpenWeatherMap

>[OpenWeatherMap](https://openweathermap.org/api/) fournit toutes les données météorologiques essentielles pour un emplacement spécifique :
>- Météo actuelle
>- Prévisions à la minute pour 1 heure
>- Prévisions horaires pour 48 heures
>- Prévisions quotidiennes pour 8 jours
>- Alertes météorologiques nationales
>- Données météorologiques historiques remontant à plus de 40 ans

Cette page explique comment utiliser l'`API OpenWeatherMap` dans LangChain.

## Installation et configuration

- Installez les prérequis avec

```bash
pip install pyowm
```

- Rendez-vous sur OpenWeatherMap et inscrivez-vous pour obtenir votre clé API [ici](https://openweathermap.org/api/)
- Définissez votre clé API en tant que variable d'environnement `OPENWEATHERMAP_API_KEY`

## Wrappers

### Utilitaire

Il existe un utilitaire `OpenWeatherMapAPIWrapper` qui encapsule cette API. Pour importer cet utilitaire :

```python
<!--IMPORTS:[{"imported": "OpenWeatherMapAPIWrapper", "source": "langchain_community.utilities.openweathermap", "docs": "https://api.python.langchain.com/en/latest/utilities/langchain_community.utilities.openweathermap.OpenWeatherMapAPIWrapper.html", "title": "OpenWeatherMap"}]-->
from langchain_community.utilities.openweathermap import OpenWeatherMapAPIWrapper
```

Pour une présentation détaillée de ce wrapper, consultez [ce notebook](/docs/integrations/tools/openweathermap).

### Outil

Vous pouvez également charger facilement ce wrapper en tant qu'Outil (à utiliser avec un Agent).
Vous pouvez le faire avec :

```python
<!--IMPORTS:[{"imported": "load_tools", "source": "langchain.agents", "docs": "https://api.python.langchain.com/en/latest/agent_toolkits/langchain_community.agent_toolkits.load_tools.load_tools.html", "title": "OpenWeatherMap"}]-->

tools = load_tools(["openweathermap-api"])
```

Pour plus d'informations sur les outils, consultez [cette page](/docs/modules/tools/).
