# Kinetica

[Kinetica](https://www.kinetica.com/) est une base de données en temps réel conçue pour permettre
l'analyse et l'IA générative sur les données de séries temporelles et spatiales.

## Modèle de discussion

L'enveloppe LLM Kinetica utilise le [Kinetica SqlAssist
LLM](https://docs.kinetica.com/7.2/sql-gpt/concepts/) pour transformer le langage naturel en
SQL afin de simplifier le processus de récupération des données.

Voir [Kinetica SqlAssist LLM Demo](/docs/integrations/chat/kinetica) pour l'utilisation.

```python
<!--IMPORTS:[{"imported": "ChatKinetica", "source": "langchain_community.chat_models.kinetica", "docs": "https://api.python.langchain.com/en/latest/chat_models/langchain_community.chat_models.kinetica.ChatKinetica.html", "title": "Kinetica"}]-->

```
