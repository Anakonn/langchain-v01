# IA confiante

>[IA confiante](https://confident-ai.com) est un créateur de `DeepEval`.
>
>[DeepEval](https://github.com/confident-ai/deepeval) est un package pour les tests unitaires des LLM.
> En utilisant `DeepEval`, tout le monde peut construire des modèles de langage robustes grâce à des itérations plus rapides
> en utilisant à la fois les tests unitaires et les tests d'intégration. `DeepEval` fournit un support pour chaque étape de l'itération
> de la création de données synthétiques aux tests.

## Installation et configuration

Vous devez obtenir les [identifiants de l'API DeepEval](https://app.confident-ai.com).

Vous devez installer le package Python `DeepEval` :

```bash
pip install deepeval
```
