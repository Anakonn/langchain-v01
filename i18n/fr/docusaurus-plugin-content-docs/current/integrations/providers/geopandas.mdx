# Geopandas

>[GeoPandas](https://geopandas.org/) est un projet open source visant à faciliter le travail
> avec les données géospatiales en python. `GeoPandas` étend les types de données utilisés par
> `pandas` pour permettre des opérations spatiales sur les types géométriques.
> Les opérations géométriques sont effectuées par `shapely`.

## Installation et configuration

Nous devons installer plusieurs packages python.

```bash
pip install -U sodapy pandas geopandas
```
