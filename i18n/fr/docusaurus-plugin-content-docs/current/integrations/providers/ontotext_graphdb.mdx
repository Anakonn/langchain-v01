# Ontotext GraphDB

>[Ontotext GraphDB](https://graphdb.ontotext.com/) est une base de données de graphes et un outil de découverte de connaissances conforme à RDF et SPARQL.

## Dépendances

Installez le package [rdflib](https://github.com/RDFLib/rdflib) avec

```bash
pip install rdflib==7.0.0
```
