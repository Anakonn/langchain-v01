# Vespa

>[Vespa](https://vespa.ai/) est un moteur de recherche et une base de données vectorielle entièrement fonctionnels.
> Il prend en charge la recherche vectorielle (ANN), la recherche lexicale et la recherche dans les données structurées, le tout dans la même requête.

## Installation et configuration

```bash
pip install pyvespa
```
