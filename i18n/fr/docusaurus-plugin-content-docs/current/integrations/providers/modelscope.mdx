# ModelScope

>[ModelScope](https://www.modelscope.cn/home) est un grand référentiel de modèles et de jeux de données.

Cette page couvre comment utiliser l'écosystème modelscope dans LangChain.
Elle est divisée en deux parties : l'installation et la configuration, puis les références aux wrappers modelscope spécifiques.

## Installation et configuration

Installez le package `modelscope`.

```bash
pip install modelscope
```
