# Chaindesk

>[Chaindesk](https://chaindesk.ai) est une plateforme de récupération de documents [open-source](https://github.com/gmpetrov/databerry) qui permet de connecter vos données personnelles aux modèles de langage à grande échelle.

## Installation et configuration

Nous devons nous inscrire à Chaindesk, créer un magasin de données, ajouter des données et obtenir l'URL de l'endpoint de l'API de votre magasin de données.
Nous avons besoin de la [clé API](https://docs.chaindesk.ai/api-reference/authentication).
