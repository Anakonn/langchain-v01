# Alibaba Cloud

>[Alibaba Group Holding Limited (Wikipédia)](https://en.wikipedia.org/wiki/Alibaba_Group), ou `Alibaba`
> (chinois : 阿里巴巴), est une entreprise technologique multinationale chinoise spécialisée dans le
> e-commerce, la vente au détail, Internet et la technologie.
>
> [Alibaba Cloud (Wikipédia)](https://en.wikipedia.org/wiki/Alibaba_Cloud), également connu sous le nom de `Aliyun`
> (chinois : 阿里云 ; pinyin : Ālǐyún ; littéralement 'Nuage Ali'), est une entreprise de cloud computing,
> une filiale du groupe `Alibaba`. `Alibaba Cloud` fournit des services de cloud computing aux entreprises
> en ligne et à l'écosystème de commerce électronique d'Alibaba.
