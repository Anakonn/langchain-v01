# Minimax

>[Minimax](https://api.minimax.chat) est une startup chinoise qui fournit des modèles de traitement du langage naturel
> pour les entreprises et les particuliers.

## Installation et configuration

Obtenez une [clé d'API Minimax](https://api.minimax.chat/user-center/basic-information/interface-key) et définissez-la comme une variable d'environnement (`MINIMAX_API_KEY`)
Obtenez un [ID de groupe Minimax](https://api.minimax.chat/user-center/basic-information) et définissez-le comme une variable d'environnement (`MINIMAX_GROUP_ID`)
