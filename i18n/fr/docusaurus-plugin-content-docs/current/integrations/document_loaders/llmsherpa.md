# LLM Sherpa

Ce cahier couvre comment utiliser `LLM Sherpa` pour charger des fichiers de nombreux types. `LLM Sherpa` prend en charge différents formats de fichiers, notamment DOCX, PPTX, HTML, TXT et XML.

`LLMSherpaFileLoader` utilise LayoutPDFReader, qui fait partie de la bibliothèque LLMSherpa. Cet outil est conçu pour analyser les PDF tout en préservant leurs informations de mise en page, souvent perdues lors de l'utilisation de la plupart des analyseurs de texte PDF.

Voici quelques-unes des principales fonctionnalités de LayoutPDFReader :

* Il peut identifier et extraire des sections et des sous-sections ainsi que leurs niveaux.
* Il combine les lignes pour former des paragraphes.
* Il peut identifier les liens entre les sections et les paragraphes.
* Il peut extraire des tableaux ainsi que la section dans laquelle ils se trouvent.
* Il peut identifier et extraire des listes et des listes imbriquées.
* Il peut joindre le contenu réparti sur plusieurs pages.
* Il peut supprimer les en-têtes et les pieds de page répétitifs.
* Il peut supprimer les filigranes.

vérifier [llmsherpa](https://llmsherpa.readthedocs.io/en/latest/) documentation.

`INFO : cette bibliothèque échoue avec certains fichiers PDF, donc utilisez-la avec prudence.`

## LLMSherpaFileLoader

Sous le capot, LLMSherpaFileLoader a défini quelques stratégies pour charger le contenu du fichier : ["sections", "chunks", "html", "text"], configurer [nlm-ingestor](https://github.com/nlmatics/nlm-ingestor) pour obtenir `llmsherpa_api_url` ou utiliser la valeur par défaut.

### stratégie des sections : renvoyer le fichier analysé en sections

```python
from langchain_community.document_loaders.llmsherpa import LLMSherpaFileLoader

loader = LLMSherpaFileLoader(
    file_path="https://arxiv.org/pdf/2402.14207.pdf",
    new_indent_parser=True,
    apply_ocr=True,
    strategy="sections",
    llmsherpa_api_url="http://localhost:5010/api/parseDocument?renderFormat=all",
)
docs = loader.load()
```

```python
docs[1]
```
