# OpenAI

Toutes les fonctionnalités liées à OpenAI

>[OpenAI](https://en.wikipedia.org/wiki/OpenAI) est un laboratoire de recherche en intelligence artificielle (IA) américain
> composé de l'organisation à but non lucratif `OpenAI Incorporated`
> et de sa filiale à but lucratif `OpenAI Limited Partnership`.
> `OpenAI` mène des recherches en IA dans le but déclaré de promouvoir et de développer une IA bienveillante.
> Les systèmes `OpenAI` fonctionnent sur une plateforme de supercalculateurs basée sur `Azure` de `Microsoft`.

>L'[API OpenAI](https://platform.openai.com/docs/models) est alimentée par un ensemble diversifié de modèles avec différentes capacités et tarifs.
>
>[ChatGPT](https://chat.openai.com) est le chatbot d'intelligence artificielle (IA) développé par `OpenAI`.

## Installation et configuration

Installez le package d'intégration avec

```bash
pip install langchain-openai
```

Obtenez une clé d'API OpenAI et définissez-la comme variable d'environnement (`OPENAI_API_KEY`)

## Tokenizer

Il existe plusieurs endroits où vous pouvez utiliser le tokenizer `tiktoken`. Par défaut, il est utilisé pour compter les jetons
pour les LLM OpenAI.

Vous pouvez également l'utiliser pour compter les jetons lors de la division des documents avec

```python
<!--IMPORTS:[{"imported": "CharacterTextSplitter", "source": "langchain.text_splitter", "docs": "https://api.python.langchain.com/en/latest/character/langchain_text_splitters.character.CharacterTextSplitter.html", "title": "OpenAI"}]-->
from langchain.text_splitter import CharacterTextSplitter
CharacterTextSplitter.from_tiktoken_encoder(...)
```

Pour un guide plus détaillé de cela, voir [ce notebook](/docs/modules/data_connection/document_transformers/split_by_token#tiktoken)
