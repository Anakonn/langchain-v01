# Google

Toutes les fonctionnalités liées à [Google Cloud Platform](https://cloud.google.com/) et autres produits `Google`.

## LLMs

Nous recommandons aux développeurs individuels de commencer avec l'API Gemini (`langchain-google-genai`) et de passer à Vertex AI (`langchain-google-vertexai`) lorsqu'ils ont besoin d'un support commercial et de limites de taux plus élevées. Si vous êtes déjà familier avec le Cloud ou natif du Cloud, vous pouvez commencer directement avec Vertex AI.
Veuillez trouver plus d'informations [ici](https://ai.google.dev/gemini-api/docs/migrate-to-cloud).

## Modèles de chat

### Google Generative AI

Accédez aux modèles `Gemini` de GoogleAI tels que `gemini-pro` et `gemini-pro-vision` via la classe `ChatGoogleGenerativeAI`.

```bash
pip install -U langchain-google-genai
```

Configurez votre clé API.

```bash
export GOOGLE_API_KEY=your-api-key
```

```python
from langchain_google_genai import ChatGoogleGenerativeAI

llm = ChatGoogleGenerativeAI(model="gemini-pro")
llm.invoke("Sing a ballad of LangChain.")
```

Le modèle Gemini vision prend en charge les entrées d'image lorsqu'il fournit un seul message de chat.

```python
<!--IMPORTS:[{"imported": "HumanMessage", "source": "langchain_core.messages", "docs": "https://api.python.langchain.com/en/latest/messages/langchain_core.messages.human.HumanMessage.html", "title": "Google"}]-->
from langchain_core.messages import HumanMessage
from langchain_google_genai import ChatGoogleGenerativeAI

llm = ChatGoogleGenerativeAI(model="gemini-pro-vision")

message = HumanMessage(
    content=[
        {
            "type": "text",
            "text": "What's in this image?",
        },  # You can optionally provide text parts
        {"type": "image_url", "image_url": "https://picsum.photos/seed/picsum/200/300"},
    ]
)
llm.invoke([message])
```

La valeur de image_url peut être l'un des éléments suivants :

- Une URL d'image publique
- Un fichier gcs (par exemple, "gcs://path/to/file.png")
- Un chemin de fichier local
- Une image encodée en base64 (par exemple, data:image/png;base64,abcd124)
- Une image PIL

### ScaNN

>[Google ScaNN](https://github.com/google-research/google-research/tree/master/scann)
> (Scalable Nearest Neighbors) est un paquet python.
>
>`ScaNN` est une méthode pour une recherche de similarité vectorielle efficace à grande échelle.

>`ScaNN` inclut l'élagage de l'espace de recherche et la quantification pour la recherche du produit intérieur maximum et supporte également d'autres fonctions de distance telles que
> la distance Euclidienne. L'implémentation est optimisée pour les processeurs x86
> avec support AVX2. Voir son [github Google Research](https://github.com/google-research/google-research/tree/master/scann)
> pour plus de détails.
