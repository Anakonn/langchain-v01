# स्थापना

## आधिकारिक रिलीज़

LangChain को स्थापित करने के लिए निम्नलिखित कमांड चलाएं:

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
import CodeBlock from "@theme/CodeBlock";

<Tabs>
  <TabItem value="pip" label="Pip" default>
    <CodeBlock language="bash">pip install langchain</CodeBlock>
  </TabItem>
  <TabItem value="conda" label="Conda">
    <CodeBlock language="bash">conda install langchain -c conda-forge</CodeBlock>
  </TabItem>
</Tabs>

यह LangChain के न्यूनतम आवश्यकताओं को स्थापित करेगा।
LangChain का अधिक मूल्य तब आता है जब इसे विभिन्न मॉडल प्रदाताओं, डेटास्टोर आदि के साथ एकीकृत किया जाता है।
डिफ़ॉल्ट रूप से, ऐसा करने के लिए आवश्यक डिपेंडेंसी स्थापित नहीं होती है। आपको विशिष्ट एकीकरणों के लिए डिपेंडेंसी अलग से स्थापित करनी होगी।

## स्रोत से

यदि आप स्रोत से स्थापित करना चाहते हैं, तो आप रिपोजिटरी क्लोन कर सकते हैं और सुनिश्चित करें कि निर्देशिका `PATH/TO/REPO/langchain/libs/langchain` है और निम्नलिखित कमांड चलाएं:

```bash
pip install -e .
```

## LangChain कोर

`langchain-core` पैकेज में LangChain पारिस्थितिकी तंत्र द्वारा उपयोग की जाने वाली आधारभूत अमूर्तताएं और LangChain अभिव्यक्ति भाषा शामिल हैं। यह `langchain` द्वारा स्वचालित रूप से स्थापित किया जाता है, लेकिन अलग से भी उपयोग किया जा सकता है। इसे निम्नलिखित कमांड से स्थापित करें:

```bash
pip install langchain-core
```

## LangChain समुदाय

`langchain-community` पैकेज में तृतीय-पक्ष एकीकरण शामिल हैं। यह `langchain` द्वारा स्वचालित रूप से स्थापित किया जाता है, लेकिन अलग से भी उपयोग किया जा सकता है। इसे निम्नलिखित कमांड से स्थापित करें:

```bash
pip install langchain-community
```

## LangChain प्रयोगात्मक

`langchain-experimental` पैकेज में अनुसंधान और प्रयोगात्मक उपयोगों के लिए इरादे वाला प्रयोगात्मक LangChain कोड शामिल है।
इसे निम्नलिखित कमांड से स्थापित करें:

```bash
pip install langchain-experimental
```

## LangGraph

`langgraph` LLM के साथ स्थिर, बहु-अभिनेता अनुप्रयोग बनाने के लिए एक लाइब्रेरी है, जो LangChain पर निर्मित है और उसके साथ उपयोग करने के लिए इरादा किया गया है।
इसे निम्नलिखित कमांड से स्थापित करें:

```bash
pip install langgraph
```

## LangServe

LangServe डेवलपर्स को LangChain रनेबल्स और श्रृंखलाओं को REST API के रूप में तैनात करने में मदद करता है।
LangServe LangChain CLI द्वारा स्वचालित रूप से स्थापित किया जाता है।
यदि LangChain CLI का उपयोग नहीं कर रहे हैं, तो इसे निम्नलिखित कमांड से स्थापित करें:

```bash
pip install "langserve[all]"
```

क्लाइंट और सर्वर दोनों डिपेंडेंसी के लिए। या `pip install "langserve[client]"` क्लाइंट कोड के लिए, और `pip install "langserve[server]"` सर्वर कोड के लिए।

## LangChain CLI

LangChain CLI LangChain टेम्प्लेट और अन्य LangServe परियोजनाओं के साथ काम करने के लिए उपयोगी है।
इसे निम्नलिखित कमांड से स्थापित करें:

```bash
pip install langchain-cli
```

## LangSmith SDK

LangSmith SDK LangChain द्वारा स्वचालित रूप से स्थापित किया जाता है।
यदि आप LangChain का उपयोग नहीं कर रहे हैं, तो इसे निम्नलिखित कमांड से स्थापित करें:

```bash
pip install langsmith
```
