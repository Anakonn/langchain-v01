# टाइगरग्राफ

>[टाइगरग्राफ](https://www.tigergraph.com/tigergraph-db/) एक नेटिवली वितरित और उच्च-प्रदर्शन ग्राफ डेटाबेस है।
> डेटा को शीर्षक और किनारों के ग्राफ प्रारूप में संग्रहीत करने से समृद्ध संबंध होते हैं,
> एलएलएम प्रतिक्रियाओं के लिए आदर्श।

`टाइगरग्राफ` और `लैंगचेन` एकीकरण का एक बड़ा उदाहरण [यहां प्रस्तुत किया गया है](https://github.com/tigergraph/graph-ml-notebooks/blob/main/applications/large_language_models/TigerGraph_LangChain_Demo.ipynb)।

## स्थापना और सेटअप

[कैसे `टाइगरग्राफ` डेटाबेस से कनेक्ट करें](https://docs.tigergraph.com/pytigergraph/current/getting-started/connection) के निर्देशों का पालन करें।

पायथन एसडीके स्थापित करें:

```bash
pip install pyTigerGraph
```

## उदाहरण

`टाइगरग्राफ InquiryAI` कार्यक्षमता का उपयोग करने के लिए, आप `langchain_community.graphs` से `टाइगरग्राफ` आयात कर सकते हैं।

```python
<!--IMPORTS:[{"imported": "TigerGraph", "source": "langchain_community.graphs", "docs": "https://api.python.langchain.com/en/latest/graphs/langchain_community.graphs.tigergraph_graph.TigerGraph.html", "title": "TigerGraph"}]-->
import pyTigerGraph as tg

conn = tg.TigerGraphConnection(host="DATABASE_HOST_HERE", graphname="GRAPH_NAME_HERE", username="USERNAME_HERE", password="PASSWORD_HERE")

### ==== CONFIGURE INQUIRYAI HOST ====
conn.ai.configureInquiryAIHost("INQUIRYAI_HOST_HERE")

from langchain_community.graphs import TigerGraph

graph = TigerGraph(conn)
result = graph.query("How many servers are there?")
print(result)
```
