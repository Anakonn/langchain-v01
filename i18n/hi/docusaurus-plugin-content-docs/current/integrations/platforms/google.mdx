# गूगल

[Google Cloud Platform](https://cloud.google.com/) और अन्य `Google` उत्पादों से संबंधित सभी कार्यक्षमता।

## LLMs

हम व्यक्तिगत डेवलपर्स को Gemini API (`langchain-google-genai`) के साथ शुरू करने और Vertex AI (`langchain-google-vertexai`) पर जाने की सलाह देते हैं जब उन्हें व्यावसायिक समर्थन और उच्च दर सीमाओं तक पहुँच की आवश्यकता होती है। यदि आप पहले से ही Cloud-friendly या Cloud-native हैं, तो आप सीधे Vertex AI में शुरू कर सकते हैं।
कृपया अधिक जानकारी [यहां](https://ai.google.dev/gemini-api/docs/migrate-to-cloud) पाएं।

## चैट मॉडल

### गूगल जनरेटिव AI

`ChatGoogleGenerativeAI` क्लास के माध्यम से GoogleAI `Gemini` मॉडल जैसे `gemini-pro` और `gemini-pro-vision` तक पहुंचें।

```bash
pip install -U langchain-google-genai
```

अपने API कुंजी को कॉन्फ़िगर करें।

```bash
export GOOGLE_API_KEY=your-api-key
```

```python
from langchain_google_genai import ChatGoogleGenerativeAI

llm = ChatGoogleGenerativeAI(model="gemini-pro")
llm.invoke("Sing a ballad of LangChain.")
```

Gemini vision मॉडल एकल चैट संदेश प्रदान करते समय छवि इनपुट का समर्थन करता है।

```python
<!--IMPORTS:[{"imported": "HumanMessage", "source": "langchain_core.messages", "docs": "https://api.python.langchain.com/en/latest/messages/langchain_core.messages.human.HumanMessage.html", "title": "Google"}]-->
from langchain_core.messages import HumanMessage
from langchain_google_genai import ChatGoogleGenerativeAI

llm = ChatGoogleGenerativeAI(model="gemini-pro-vision")

message = HumanMessage(
    content=[
        {
            "type": "text",
            "text": "What's in this image?",
        },  # You can optionally provide text parts
        {"type": "image_url", "image_url": "https://picsum.photos/seed/picsum/200/300"},

```

image_url का मान निम्न में से कोई भी हो सकता है:

- एक सार्वजनिक छवि URL
- एक gcs फ़ाइल (उदा., "gcs://path/to/file.png")
- एक स्थानीय फ़ाइल पथ
- एक base64 एन्कोडेड छवि (उदा., data:image/png;base64,abcd124)
- एक PIL छवि

### BigQuery Vector Search

> [Google Cloud BigQuery](https://cloud.google.com/bigquery),
> BigQuery Google Cloud में एक सर्वरलेस और किफायती एंटरप्राइज़ डेटा वेयरहाउस है।
>
> [Google Cloud BigQuery Vector Search](https://cloud.google.com/bigquery/docs/vector-search-intro)
> BigQuery वेक्टर खोज आपको GoogleSQL का उपयोग करके सेमांटिक खोज करने की अनुमति देती है, जो तेज़ लेकिन अनुमानित परिणामों के लिए वेक्टर इंडेक्स का उपयोग करती है, या सटीक परिणामों के लिए ब्रूट फोर्स का उपयोग करती है।

> यह Euclidean या Cosine दूरी की गणना कर सकता है। LangChain के साथ, हम डिफ़ॉल्ट रूप से Euclidean दूरी का उपयोग करते हैं।
