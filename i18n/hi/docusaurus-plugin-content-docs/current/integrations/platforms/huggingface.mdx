# Hugging Face

[Hugging Face प्लेटफ़ॉर्म](https://huggingface.co/) से संबंधित सभी कार्यक्षमता।

## चैट मॉडल

### Hugging Face से मॉडल

हम `Hugging Face` LLM वर्गों का उपयोग कर सकते हैं या सीधे `ChatHuggingFace` वर्ग का उपयोग कर सकते हैं।

हमें कई पायथन पैकेज स्थापित करने की आवश्यकता है।

```bash
pip install huggingface_hub
pip install transformers
```

## LLMs

### Hugging Face स्थानीय पाइपलाइन

Hugging Face मॉडल को `HuggingFacePipeline` वर्ग के माध्यम से स्थानीय रूप से चलाया जा सकता है।

हमें `transformers` पायथन पैकेज स्थापित करने की आवश्यकता है।

```bash
pip install transformers
```

[उपयोग उदाहरण](/docs/integrations/llms/huggingface_pipelines) देखें।

```python
<!--IMPORTS:[{"imported": "HuggingFacePipeline", "source": "langchain_community.llms.huggingface_pipeline", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.huggingface_pipeline.HuggingFacePipeline.html", "title": "Hugging Face"}]-->
from langchain_community.llms.huggingface_pipeline import HuggingFacePipeline
```

स्थानीय पाइपलाइन रैपर में OpenVINO बैकएंड का उपयोग करने के लिए, कृपया optimum लाइब्रेरी स्थापित करें और HuggingFacePipeline के बैकएंड को `openvino` के रूप में सेट करें:

```bash
pip install --upgrade-strategy eager "optimum[openvino,nncf]"
```

[उपयोग उदाहरण](/docs/integrations/llms/huggingface_pipelines) देखें

CLI के साथ OpenVINO IR प्रारूप में अपने मॉडल को निर्यात करने के लिए:

```bash
optimum-cli export openvino --model gpt2 ov_model
```

[वजन-केवल क्वांटीकरण](https://github.com/huggingface/optimum-intel?tab=readme-ov-file#export) लागू करने के लिए अपने मॉडल को निर्यात करते समय।

## एम्बेडिंग मॉडल

### Hugging Face हब

>[Hugging Face हब](https://huggingface.co/docs/hub/index) एक प्लेटफ़ॉर्म है
> जहां 350,000 से अधिक मॉडल, 75,000 डेटासेट और 150,000 डेमो ऐप (स्पेस) हैं, सभी ओपन सोर्स
> और सार्वजनिक रूप से उपलब्ध, एक ऑनलाइन प्लेटफ़ॉर्म में जहां लोग आसानी से
> मशीन लर्निंग के साथ सहयोग और निर्माण कर सकते हैं। हब एक केंद्रीय स्थान के रूप में काम करता है जहां कोई भी
> मशीन लर्निंग के साथ खोज, प्रयोग, सहयोग और निर्माण कर सकता है।

हमें `sentence_transformers` पायथन पैकेज स्थापित करने की आवश्यकता है।

```bash
pip install sentence_transformers
```

### Hugging Face Text Embeddings Inference (TEI)

>[Hugging Face Text Embeddings Inference (TEI)](https://huggingface.co/docs/text-generation-inference/index) ओपन-सोर्स
> पाठ एम्बेडिंग और अनुक्रम वर्गीकरण मॉडल को तैनात और सर्व करने के लिए एक टूलकिट है। `TEI` सबसे लोकप्रिय मॉडलों के लिए उच्च-प्रदर्शन निकालने को सक्षम बनाता है,
>जिसमें `FlagEmbedding`, `Ember`, `GTE` और `E5` शामिल हैं।

