# OpenAI

OpenAI से संबंधित सभी कार्यक्षमता

>[OpenAI](https://en.wikipedia.org/wiki/OpenAI) एक अमेरिकी कृत्रिम बुद्धिमत्ता (AI) अनुसंधान प्रयोगशाला है
> जिसमें गैर-लाभकारी `OpenAI Incorporated`
> और इसकी लाभकारी सहायक कंपनी `OpenAI Limited Partnership` शामिल हैं।
> `OpenAI` मैत्रीपूर्ण AI को बढ़ावा देने और विकसित करने के घोषित उद्देश्य के साथ AI अनुसंधान करता है।
> `OpenAI` प्रणालियां `Microsoft` के `Azure`-आधारित सुपरकंप्यूटिंग प्लेटफॉर्म पर चलती हैं।

>[OpenAI API](https://platform.openai.com/docs/models) विभिन्न क्षमताओं और मूल्य बिंदुओं वाले मॉडलों के विविध सेट द्वारा संचालित है।
>
>[ChatGPT](https://chat.openai.com) `OpenAI` द्वारा विकसित एक कृत्रिम बुद्धिमत्ता (AI) चैटबॉट है।

## स्थापना और सेटअप

एकीकरण पैकेज को इंस्टॉल करें

```bash
pip install langchain-openai
```

एक OpenAI api कुंजी प्राप्त करें और इसे एक पर्यावरण चर (`OPENAI_API_KEY`) के रूप में सेट करें

## टोकनाइज़र

`tiktoken` टोकनाइज़र का उपयोग कई जगहों पर किया जा सकता है। डिफ़ॉल्ट रूप से, इसका उपयोग OpenAI LLM के लिए टोकन गिनने के लिए किया जाता है।

आप इसका उपयोग दस्तावेज़ को विभाजित करते समय टोकन गिनने के लिए भी कर सकते हैं

```python
<!--IMPORTS:[{"imported": "CharacterTextSplitter", "source": "langchain.text_splitter", "docs": "https://api.python.langchain.com/en/latest/character/langchain_text_splitters.character.CharacterTextSplitter.html", "title": "OpenAI"}]-->
```

इस बारे में अधिक विस्तृत वॉकथ्रू के लिए, [इस नोटबुक](/docs/modules/data_connection/document_transformers/split_by_token#tiktoken) देखें
