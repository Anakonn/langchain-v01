# AWS

[Amazon AWS](https://aws.amazon.com/) प्लेटफ़ॉर्म से संबंधित `LangChain` एकीकरण।

`langchain_aws` पैकेज में प्रथम-पक्ष AWS एकीकरण उपलब्ध हैं।

```bash
pip install langchain-aws
```

और `langchain_community` पैकेज में `boto3` वैकल्पिक निर्भरता के साथ कुछ समुदाय एकीकरण भी उपलब्ध हैं।

```bash
pip install langchain-community boto3
```

## LLMs

### Bedrock

>[Amazon Bedrock](https://aws.amazon.com/bedrock/) एक पूरी तरह से प्रबंधित सेवा है जो `AI21 Labs`, `Anthropic`, `Cohere`, `Meta`, `Stability AI` और `Amazon` जैसी अग्रणी AI कंपनियों से उच्च-प्रदर्शन वाले फाउंडेशन मॉडल (FM) का चयन प्रदान करती है, साथ ही सुरक्षा, गोपनीयता और जिम्मेदार AI के साथ जनरेटिव AI अनुप्रयोग बनाने के लिए आवश्यक व्यापक क्षमताएं भी प्रदान करती है। `Amazon Bedrock` का उपयोग करके, आप शीर्ष FM को अपने उपयोग मामले के लिए आसानी से प्रयोग और मूल्यांकन कर सकते हैं, तकनीकों जैसे फाइन-ट्यूनिंग और `Retrieval Augmented Generation` (`RAG`) का उपयोग करके उन्हें निजी रूप से अनुकूलित कर सकते हैं, और अपने उद्यम प्रणालियों और डेटा स्रोतों का उपयोग करके कार्य निष्पादन करने वाले एजेंटों का निर्माण कर सकते हैं। `Amazon Bedrock` सर्वरलेस होने के कारण, आपको किसी भी बुनियादी ढांचे का प्रबंधन नहीं करना पड़ता है, और आप AWS सेवाओं का उपयोग करके अपने अनुप्रयोगों में जनरेटिव AI क्षमताओं को सुरक्षित रूप से एकीकृत और तैनात कर सकते हैं।

[उपयोग उदाहरण](/docs/integrations/llms/bedrock) देखें।

```python
from langchain_aws import BedrockLLM
```

### Amazon API Gateway

>[Amazon API Gateway](https://aws.amazon.com/api-gateway/) एक पूरी तरह से प्रबंधित सेवा है जो डेवलपर्स को किसी भी स्केल पर API बनाने, प्रकाशित करने, बनाए रखने, मॉनिटर करने और सुरक्षित करने में आसान बनाती है। API आवेदनों के लिए डेटा, व्यवसाय तर्क या पृष्ठभूमि सेवाओं से सुविधा प्रदान करने के लिए "मुख्य द्वार" का काम करते हैं। `API Gateway` का उपयोग करके, आप RESTful API और WebSocket API बना सकते हैं जो वास्तविक समय की दो-तरफा संचार अनुप्रयोगों को सक्षम करते हैं। `API Gateway` कंटेनरीकृत और सर्वरलेस कार्यभार, साथ ही वेब अनुप्रयोगों का समर्थन करता है।
>
> `API Gateway` सैकड़ों हजारों समकालिक API कॉल स्वीकार करने और प्रसंस्करण करने से जुड़े सभी कार्यों को संभालता है, जिसमें ट्रैफ़िक प्रबंधन, CORS समर्थन, प्राधिकरण और पहुंच नियंत्रण, थ्रोटलिंग, मॉनिटरिंग और API संस्करण प्रबंधन शामिल हैं। `API Gateway` में कोई न्यूनतम शुल्क या स्टार्टअप लागत नहीं है। आप प्राप्त API कॉल और स्थानांतरित डेटा की मात्रा के लिए भुगतान करते हैं, और `API Gateway` स्तरीय मूल्य मॉडल के साथ, जैसे-जैसे आपका API उपयोग बढ़ता है, आप अपनी लागत को कम कर सकते हैं।

### Amazon DocumentDB Vector Search

>[Amazon DocumentDB (with MongoDB Compatibility)](https://docs.aws.amazon.com/documentdb/) क्लाउड में MongoDB-संगत डेटाबेस को सेट अप, संचालित और स्केल करने में आसान बनाता है।
> Amazon DocumentDB के साथ, आप वही एप्लिकेशन कोड चला सकते हैं और वही ड्राइवर और टूल का उपयोग कर सकते हैं जो आप MongoDB के साथ उपयोग करते हैं।
> Amazon DocumentDB के लिए वेक्टर सर्च JSON-आधारित दस्तावेज़ डेटाबेस की लचीलापन और समृद्ध क्वेरी क्षमता को वेक्टर सर्च की शक्ति के साथ जोड़ता है।

### Amazon Kendra

> [Amazon Kendra](https://docs.aws.amazon.com/kendra/latest/dg/what-is-kendra.html) एक बुद्धिमान खोज सेवा है
> जो `Amazon Web Services` (`AWS`) द्वारा प्रदान की जाती है। यह उन्नत प्राकृतिक भाषा प्रसंस्करण (NLP) और मशीन
> लर्निंग एल्गोरिदम का उपयोग करके संगठन के भीतर विभिन्न डेटा स्रोतों में शक्तिशाली खोज क्षमताएं प्रदान करता है।
> `Kendra` उपयोगकर्ताओं को जल्दी और सटीक जानकारी खोजने में मदद करने के लिए डिज़ाइन किया गया है,
> उत्पादकता और निर्णय लेने में सुधार करता है।

> `Kendra` के साथ, हम विभिन्न प्रकार के सामग्री जैसे दस्तावेज़, FAQs, नॉलेज बेस,
> मैनुअल और वेबसाइटों में खोज कर सकते हैं। यह कई भाषाओं का समर्थन करता है और जटिल क्वेरी, समानार्थी शब्द और
> संदर्भगत अर्थों को समझने के लिए डिज़ाइन किया गया है ताकि अत्यधिक प्रासंगिक खोज परिणाम प्रदान किए जा सकें।
