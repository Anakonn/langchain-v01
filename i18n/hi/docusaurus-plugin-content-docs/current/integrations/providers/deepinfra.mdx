# DeepInfra

>[DeepInfra](https://deepinfra.com/docs) हमें [नवीनतम मशीन लर्निंग मॉडल](https://deepinfra.com/models) को आसानी से चलाने में मदद करता है।
> DeepInfra मॉडल को चलाने, स्केल करने और मॉनिटर करने से जुड़े सभी कठिन कार्यों को संभालता है। उपयोगकर्ता अपने एप्लिकेशन पर ध्यान केंद्रित कर सकते हैं और सरल REST API कॉल के माध्यम से मॉडल को एकीकृत कर सकते हैं।

>DeepInfra LangChain के साथ एकीकरण के [उदाहरण](https://deepinfra.com/docs/advanced/langchain) प्रदान करता है।

यह पृष्ठ `LangChain` के भीतर `DeepInfra` पारिस्थितिकी तंत्र का उपयोग करने के बारे में कवर करता है।
यह दो भागों में विभाजित है: स्थापना और सेटअप, और फिर विशिष्ट DeepInfra रैपर्स के संदर्भ।

## स्थापना और सेटअप

- यहां से अपना DeepInfra api कुंजी प्राप्त करें [यहां](https://deepinfra.com/)।
- एक DeepInfra api कुंजी प्राप्त करें और इसे एक पर्यावरण चर (`DEEPINFRA_API_TOKEN`) के रूप में सेट करें

## उपलब्ध मॉडल

DeepInfra तैनाती के लिए तैयार ओपन सोर्स एलएलएम का एक श्रृंखला प्रदान करता है।

आप [पाठ-जनन](https://deepinfra.com/models?type=text-generation) और [एम्बेडिंग](https://deepinfra.com/models?type=embeddings) के लिए समर्थित मॉडल देख सकते हैं।

आप [अनुरोध और प्रतिक्रिया पैरामीटरों की एक सूची](https://deepinfra.com/meta-llama/Llama-2-70b-chat-hf/api) देख सकते हैं।

चैट मॉडल [openai api का पालन करते हैं](https://deepinfra.com/meta-llama/Llama-2-70b-chat-hf/api?example=openai-http)
