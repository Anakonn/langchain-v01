# Docugami

>[Docugami](https://docugami.com) व्यावसायिक दस्तावेजों को एक दस्तावेज XML ज्ञान ग्राफ में परिवर्तित करता है, जिससे पूरे दस्तावेजों का एक वृक्ष प्रतिनिधित्व बनता है। यह एक समृद्ध प्रतिनिधित्व है जो दस्तावेज के विभिन्न खंडों की语义और संरचनात्मक विशेषताओं को एक XML वृक्ष के रूप में शामिल करता है।

## स्थापना और सेटअप

```bash
pip install dgml-utils
pip install docugami-langchain
```

## दस्तावेज लोडर

एक [उपयोग उदाहरण](/docs/integrations/document_loaders/docugami) देखें।

```python
from docugami_langchain.document_loaders import DocugamiLoader
```
