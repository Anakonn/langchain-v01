# Airbyte

>[Airbyte](https://github.com/airbytehq/airbyte) एक डेटा एकीकरण प्लेटफॉर्म है जो API, डेटाबेस और फ़ाइलों से वेयरहाउस और लेक्स तक के ELT पाइपलाइन के लिए है। इसके पास डेटा वेयरहाउस और डेटाबेस के सबसे बड़े ELT कनेक्टर कैटलॉग है।

## स्थापना और सेटअप

```bash
pip install -U langchain-airbyte
```

:::note

वर्तमान में, `langchain-airbyte` लाइब्रेरी Pydantic v2 का समर्थन नहीं करती है।
इस पैकेज का उपयोग करने के लिए कृपया Pydantic v1 में डाउनग्रेड करें।

यह पैकेज वर्तमान में Python 3.10+ की आवश्यकता है।

:::

एकीकरण पैकेज को कोई वैश्विक पर्यावरण चर सेट करने की आवश्यकता नहीं है, लेकिन कुछ एकीकरण (जैसे `source-github`) को पारित किए जाने वाले क्रेडेंशियल की आवश्यकता हो सकती है।
