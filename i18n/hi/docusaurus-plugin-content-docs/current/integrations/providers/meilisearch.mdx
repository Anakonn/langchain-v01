# मेलीसर्च

> [मेलीसर्च](https://meilisearch.com) एक ओपन-सोर्स, बिजली की तरह तेज़ और अत्यधिक प्रासंगिक खोज इंजन है।
> यह डेवलपर्स को तेज़ खोज अनुभव बनाने में मदद करने के लिए महान डिफ़ॉल्ट के साथ आता है।
>
> आप [स्वयं होस्ट मेलीसर्च](https://www.meilisearch.com/docs/learn/getting_started/installation#local-installation) कर सकते हैं
> या [मेलीसर्च क्लाउड](https://www.meilisearch.com/pricing) पर चला सकते हैं।
>
>`मेलीसर्च v1.3` वेक्टर खोज का समर्थन करता है।

## इंस्टॉलेशन और सेटअप

विस्तृत कॉन्फ़िगरेशन निर्देशों के लिए एक [उपयोग उदाहरण](/docs/integrations/vectorstores/meilisearch) देखें।

हमें `meilisearch` पायथन पैकेज इंस्टॉल करना होगा।

```bash
pip install meilisearch
```
