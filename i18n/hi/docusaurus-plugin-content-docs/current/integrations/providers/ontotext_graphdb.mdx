# ऑन्टोटेक्स्ट ग्राफ़डीबी

>[ऑन्टोटेक्स्ट ग्राफ़डीबी](https://graphdb.ontotext.com/) एक ग्राफ़ डेटाबेस और आरडीएफ और एसपीएआरक्यूएल के अनुरूप ज्ञान खोज उपकरण है।

## निर्भरताएं

[rdflib](https://github.com/RDFLib/rdflib) पैकेज को इंस्टॉल करें

```bash
pip install rdflib==7.0.0
```
