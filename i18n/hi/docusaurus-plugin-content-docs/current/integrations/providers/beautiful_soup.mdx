# सुंदर सूप

>[सुंदर सूप](https://www.crummy.com/software/BeautifulSoup/) एक Python पैकेज है जो HTML और XML दस्तावेजों (गलत चिह्नित मार्कअप, यानी गैर-बंद टैग वाले) को पार्स करने के लिए है, जिसे टैग सूप के नाम से जाना जाता है।
> यह पार्स किए गए पृष्ठों के लिए एक पार्स ट्री बनाता है जिसका उपयोग HTML से डेटा निकालने के लिए किया जा सकता है,[3] जो वेब स्क्रैपिंग के लिए उपयोगी है।

## स्थापना और सेटअप

```bash
pip install beautifulsoup4
```

## दस्तावेज़ रूपांतरक

एक [उपयोग उदाहरण](/docs/integrations/document_transformers/beautiful_soup) देखें।

```python
from langchain_community.document_loaders import BeautifulSoupTransformer
```
