# मिनीमैक्स

>[मिनीमैक्स](https://api.minimax.chat) एक चीनी स्टार्टअप है जो कंपनियों और व्यक्तियों के लिए प्राकृतिक भाषा प्रसंस्करण मॉडल प्रदान करता है।

## इंस्टॉलेशन और सेटअप

एक [मिनीमैक्स एपीआई कुंजी](https://api.minimax.chat/user-center/basic-information/interface-key) प्राप्त करें और इसे एक पर्यावरण चर (`MINIMAX_API_KEY`) के रूप में सेट करें
एक [मिनीमैक्स समूह आईडी](https://api.minimax.chat/user-center/basic-information) प्राप्त करें और इसे एक पर्यावरण चर (`MINIMAX_GROUP_ID`) के रूप में सेट करें

## एलएलएम

एक मिनीमैक्स एलएलएम रैपर मौजूद है, जिसका उपयोग आप कर सकते हैं
[उपयोग उदाहरण](/docs/integrations/llms/minimax) देखें।

