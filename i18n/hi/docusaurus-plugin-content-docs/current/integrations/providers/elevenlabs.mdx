# ElevenLabs

>[ElevenLabs](https://elevenlabs.io/about) एक वॉयस एआई अनुसंधान और तैनाती कंपनी है
> जिसका मिशन किसी भी भाषा और आवाज में सामग्री को सार्वभौमिक रूप से सुलभ बनाना है।
>
>`ElevenLabs` सबसे यथार्थवादी, बहुमुखी और संदर्भ-जागरूक
> एआई ऑडियो बनाता है, जो 29 भाषाओं में सैकड़ों नई और मौजूदा आवाजों में वाक्य उत्पन्न करने की क्षमता प्रदान करता है।

## स्थापना और सेटअप

पहले, आपको एक ElevenLabs खाता सेट करना होगा। आप यहां दिए गए
[निर्देशों](https://docs.elevenlabs.io/welcome/introduction) का पालन कर सकते हैं।

Python पैकेज स्थापित करें:

```bash
pip install elevenlabs
```
