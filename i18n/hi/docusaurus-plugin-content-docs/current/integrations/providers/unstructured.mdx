# अनरचित

> [Unstructured.IO](https://www.unstructured.io/) से `अनरचित` पैकेज PDF और Word दस्तावेज़ जैसे कच्चे स्रोत दस्तावेज़ों से साफ़ पाठ निकालता है।
यह पृष्ठ LangChain में [`अनरचित`](https://github.com/Unstructured-IO/unstructured) पारिस्थितिकी तंत्र का उपयोग करने को कवर करता है।

## स्थापना और सेटअप

यदि आप स्थानीय रूप से चलने वाले लोडर का उपयोग कर रहे हैं, तो `अनरचित` और इसकी निर्भरताओं को स्थानीय रूप से चलाने के लिए निम्नलिखित चरणों का उपयोग करें।

- `pip install अनरचित` के साथ Python SDK स्थापित करें।
    - आप दस्तावेज़ विशिष्ट आवश्यकताओं को अतिरिक्त के साथ स्थापित कर सकते हैं, यानी `pip install "अनरचित[docx]"`।
    - सभी दस्तावेज़ प्रकारों के लिए आवश्यकताओं को स्थापित करने के लिए `pip install "अनरचित[all-docs]"` का उपयोग करें।
- यदि वे आपके सिस्टम पर पहले से उपलब्ध नहीं हैं, तो निम्नलिखित सिस्टम निर्भरताएं स्थापित करें। आप किन दस्तावेज़ प्रकारों का पार्सिंग कर रहे हैं, इस पर निर्भर करता है कि आपको इनमें से सभी की आवश्यकता नहीं हो सकती है।
    - `libmagic-dev` (फ़ाइल प्रकार का पता लगाना)
    - `poppler-utils` (छवियां और PDF)
    - `tesseract-ocr`(छवियां और PDF)
    - `libreoffice` (MS Office दस्तावेज़)
    - `pandoc` (EPUB)

यदि आप कम सेटअप के साथ तेजी से चालू होना चाहते हैं, तो आप `pip install अनरचित` चला सकते हैं और `UnstructuredAPIFileLoader` या `UnstructuredAPIFileIOLoader` का उपयोग कर सकते हैं। यह होस्ट किए गए Unstructured API का उपयोग करके आपके दस्तावेज़ को प्रोसेस करेगा।

`Unstructured API` अनुरोध करने के लिए API कुंजियों की आवश्यकता होती है।
आप [यहाँ](https://unstructured.io/api-key-hosted) से एक API कुंजी का अनुरोध कर सकते हैं और आज ही इसका उपयोग शुरू कर सकते हैं!
API कॉल करने की शुरुआत करने के लिए README [यहाँ](https://github.com/Unstructured-IO/unstructured-api) देखें।
हम आपके प्रतिक्रिया सुनने के लिए उत्सुक हैं, हमारे [समुदाय स्लैक](https://join.slack.com/t/unstructuredw-kbe4326/shared_invite/zt-1x7cgo0pg-PTptXWylzPQF9xZolzCnwQ) में इसके बारे में बताएं।
और गुणवत्ता और प्रदर्शन में सुधार के लिए तैयार रहें!
यदि आप Unstructured API को स्वयं होस्ट करना या इसे स्थानीय रूप से चलाना चाहते हैं, तो [यहाँ](https://github.com/Unstructured-IO/unstructured-api#dizzy-instructions-for-using-the-docker-image) दिए गए निर्देशों का पालन करें।
