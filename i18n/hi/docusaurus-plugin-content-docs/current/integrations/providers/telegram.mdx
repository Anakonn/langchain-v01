# टेलीग्राम

>[टेलीग्राम मैसेंजर](https://web.telegram.org/a/) एक वैश्विक रूप से उपलब्ध फ्रीमियम, क्रॉस-प्लेटफॉर्म, एन्क्रिप्टेड, क्लाउड-आधारित और केंद्रीकृत तत्काल संदेश सेवा है। एप्लिकेशन वैकल्पिक अंत से अंत एन्क्रिप्टेड चैट और वीडियो कॉलिंग, VoIP, फ़ाइल शेयरिंग और कई अन्य सुविधाएं भी प्रदान करता है।
