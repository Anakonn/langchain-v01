# Huawei

>[Huawei Technologies Co., Ltd.](https://www.huawei.com/) एक चीनी बहुराष्ट्रीय
> डिजिटल संचार प्रौद्योगिकी निगम है।
>
>[Huawei Cloud](https://www.huaweicloud.com/intl/en-us/product/) एक व्यापक वैश्विक क्लाउड कंप्यूटिंग सेवाओं का समग्र सूट प्रदान करता है।

## स्थापना और सेटअप

`Huawei Cloud` का उपयोग करने के लिए, आपको एक एक्सेस टोकन की आवश्यकता है।

आपको एक पायथन लाइब्रेरी भी स्थापित करनी होगी:

```bash
pip install -U esdk-obs-python
```
