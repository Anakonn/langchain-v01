# डेटाडॉग लॉग्स

>[डेटाडॉग](https://www.datadoghq.com/) एक मॉनिटरिंग और एनालिटिक्स प्लेटफॉर्म है जो क्लाउड-स्केल एप्लिकेशन के लिए है।

## इंस्टॉलेशन और सेटअप

```bash
pip install datadog_api_client
```

हमें लोडर को डेटाडॉग API कुंजी और APP कुंजी के साथ इनिशियलाइज़ करना होगा, और हमें वांछित लॉग्स को एक्सट्रैक्ट करने के लिए क्वेरी सेट करने की आवश्यकता है।
