# scikit-learn

>[scikit-learn](https://scikit-learn.org/stable/) एक ओपन-सोर्स मशीन लर्निंग एल्गोरिदम का संग्रह है,
> जिसमें [k निकटतम पड़ोसी](https://scikit-learn.org/stable/modules/generated/sklearn.neighbors.NearestNeighbors.html) का कुछ कार्यान्वयन शामिल है। `SKLearnVectorStore` इस कार्यान्वयन को लपेटता है और JSON, BSON (बाइनरी JSON) या Apache Parquet प्रारूप में वेक्टर स्टोर को स्थायी करने की संभावना जोड़ता है।

## स्थापना और सेटअप

- `pip install scikit-learn` के साथ Python पैकेज स्थापित करें

## वेक्टर स्टोर

`SKLearnVectorStore` scikit-learn पैकेज में निकटतम पड़ोसी कार्यान्वयन के इर्द-गिर्द एक सरल रैपर प्रदान करता है, जिससे आप इसका उपयोग एक वेक्टर स्टोर के रूप में कर सकते हैं।

इस वेक्टर स्टोर को आयात करने के लिए:
