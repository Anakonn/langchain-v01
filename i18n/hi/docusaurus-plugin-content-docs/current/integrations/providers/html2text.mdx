# HTML से पाठ

>[html2text](https://github.com/Alir3z4/html2text/) एक Python पैकेज है जो `HTML` पृष्ठ को साफ, पढ़ने में आसान सादे `ASCII पाठ` में बदल देता है।

ASCII वास्तव में एक वैध `Markdown` (पाठ से HTML प्रारूप) भी है।

## स्थापना और सेटअप

```bash
pip install html2text
```

## दस्तावेज़ रूपांतरक

एक [उपयोग उदाहरण](/docs/integrations/document_transformers/html2text) देखें।

```python
from langchain_community.document_loaders import Html2TextTransformer
```
