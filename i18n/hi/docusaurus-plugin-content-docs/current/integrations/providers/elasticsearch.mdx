# Elasticsearch

> [Elasticsearch](https://www.elastic.co/elasticsearch/) एक वितरित, RESTful खोज और विश्लेषण इंजन है।
> यह एक वितरित, बहु-किराएदार-क्षमता वाला पूर्ण-पाठ खोज इंजन प्रदान करता है जिसमें एक HTTP वेब इंटरफ़ेस और स्कीमा-मुक्त JSON दस्तावेज़ हैं।

## स्थापना और सेटअप

Elasticsearch शुरू करने के दो तरीके हैं:

#### अपने स्थानीय मशीन पर Docker के माध्यम से Elasticsearch स्थापित करें

उदाहरण: सुरक्षा को अक्षम करके एकल-नोड Elasticsearch इंस्टांस चलाएं। यह उत्पादन उपयोग के लिए अनुशंसित नहीं है।

```bash
    docker run -p 9200:9200 -e "discovery.type=single-node" -e "xpack.security.enabled=false" -e "xpack.security.http.ssl.enabled=false" docker.elastic.co/elasticsearch/elasticsearch:8.9.0
```

#### Elastic Cloud पर Elasticsearch तैनात करें

Elastic Cloud एक प्रबंधित Elasticsearch सेवा है। [मुक्त परीक्षण](https://cloud.elastic.co/registration?utm_source=langchain&utm_content=documentation) के लिए साइन अप करें।

### क्लाइंट स्थापित करें

```bash
pip install elasticsearch
pip install langchain-elasticsearch
```
