# ड्रॉपबॉक्स

>[ड्रॉपबॉक्स](https://en.wikipedia.org/wiki/Dropbox) एक फ़ाइल होस्टिंग सेवा है जो सब कुछ - परंपरागत फ़ाइलों, क्लाउड सामग्री और वेब शॉर्टकट को एक ही जगह लाती है।

## इंस्टॉलेशन और सेटअप

विस्तृत [इंस्टॉलेशन गाइड](/docs/integrations/document_loaders/dropbox#prerequisites) देखें।

```bash
pip install -U dropbox
```
