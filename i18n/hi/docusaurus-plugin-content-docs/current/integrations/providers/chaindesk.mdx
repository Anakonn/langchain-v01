# चेनडेस्क

>[चेनडेस्क](https://chaindesk.ai) एक [ओपन-सोर्स](https://github.com/gmpetrov/databerry) दस्तावेज़ पुनर्प्राप्ति प्लेटफ़ॉर्म है जो आपके व्यक्तिगत डेटा को बड़े भाषा मॉडलों से जोड़ने में मदद करता है।

## इंस्टॉलेशन और सेटअप

हमें चेनडेस्क के लिए साइन अप करना होगा, एक डेटास्टोर बनाना होगा, कुछ डेटा जोड़ना होगा और आपका डेटास्टोर एपीआई एंडपॉइंट यूआरएल प्राप्त करना होगा।
हमें [एपीआई कुंजी](https://docs.chaindesk.ai/api-reference/authentication) की आवश्यकता है।
