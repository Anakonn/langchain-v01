# TiDB

> [TiDB Cloud](https://tidbcloud.com/), एक व्यापक डेटाबेस-एज-ए-सर्विस (DBaaS) समाधान है,
> जो समर्पित और सर्वरलेस विकल्प प्रदान करता है। `TiDB Serverless` अब MySQL परिदृश्य में एक निर्मित-इन वेक्टर खोज को एकीकृत कर रहा है। इस सुधार के साथ, आप `TiDB Serverless` का उपयोग करके एआई अनुप्रयोग विकसित कर सकते हैं
> बिना नए डेटाबेस या अतिरिक्त तकनीकी स्टैक की आवश्यकता के। [प्राइवेट बीटा](https://tidb.cloud/ai) के लिए प्रतीक्षा सूची में शामिल होकर इसका अनुभव करें।

## स्थापना और सेटअप

आपको TiDB डेटाबेस के कनेक्शन विवरण प्राप्त करने होंगे।
[TiDB Cloud](https://tidbcloud.com/) पर जाकर कनेक्शन विवरण प्राप्त करें।
