# एनीस्केल

>[एनीस्केल](https://www.anyscale.com) उत्पादन-तैयार एपीआई के माध्यम से एलएलएम को चलाने, फाइन ट्यून करने और पैमाने पर लाने के लिए एक प्लेटफॉर्म है।
>[एनीस्केल एंडपॉइंट](https://docs.anyscale.com/endpoints/overview) लागत-प्रभावी तरीके से कई ओपन-सोर्स मॉडल प्रदान करते हैं।

`एनीस्केल` भी [एक उदाहरण](https://docs.anyscale.com/endpoints/model-serving/examples/langchain-integration) प्रदान करता है
उन्नत चैट एजेंटों के लिए `LangChain` को `एनीस्केल` के साथ सेट अप करने का।

## इंस्टॉलेशन और सेटअप

- एक एनीस्केल सर्विस यूआरएल, रूट और एपीआई कुंजी प्राप्त करें और उन्हें पर्यावरण चर (`ANYSCALE_SERVICE_URL`,`ANYSCALE_SERVICE_ROUTE`, `ANYSCALE_SERVICE_TOKEN`) के रूप में सेट करें।
- अधिक जानकारी के लिए कृपया [एनीस्केल दस्तावेज़](https://www.anyscale.com/get-started) देखें।

हमें `openai` पैकेज इंस्टॉल करना होगा:

```bash
pip install openai
```
