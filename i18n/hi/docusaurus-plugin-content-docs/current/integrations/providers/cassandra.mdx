# कैसेंड्रा

> [Apache Cassandra®](https://cassandra.apache.org/) एक NoSQL, पंक्ति-उन्मुख, अत्यधिक स्केलेबल और अत्यधिक उपलब्ध डेटाबेस है।
> संस्करण 5.0 से शुरू, डेटाबेस में [वेक्टर खोज क्षमताएं](https://cassandra.apache.org/doc/trunk/cassandra/vector-search/overview.html) शामिल हैं।

इस पृष्ठ में वर्णित एकीकरण का उपयोग `Cassandra` के साथ-साथ अन्य CQL-संगत डेटाबेसों के साथ भी किया जा सकता है,
यानी जो `Cassandra Query Language` प्रोटोकॉल का उपयोग करते हैं।

## स्थापना और सेटअप

निम्नलिखित Python पैकेज स्थापित करें:

```bash
pip install "cassio>=0.1.6"
```
