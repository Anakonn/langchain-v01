# रोबोकॉर्प

>[रोबोकॉर्प](https://robocorp.com/) पायथन वर्कर्स को बनाने और संचालित करने में मदद करता है जो कहीं भी और किसी भी स्केल पर बेझिझक चलते हैं

## इंस्टॉलेशन और सेटअप

आपको `langchain-robocorp` पायथन पैकेज इंस्टॉल करना होगा:

```bash
pip install langchain-robocorp
```

आपके एजेंट एप्लिकेशन से संवाद करने के लिए आपको एक चल रहा एक्शन सर्वर का इंस्टेंस होना चाहिए। एक्शन सर्वर सेट करने और अपने एक्शन बनाने के बारे में [रोबोकॉर्प त्वरित शुरुआत](https://github.com/robocorp/robocorp#quickstart) देखें।

आप एक्शन सर्वर `new` कमांड का उपयोग करके एक नया प्रोजेक्ट बूटस्ट्रैप कर सकते हैं।

```bash
action-server new
cd ./your-project-name
action-server start
```
