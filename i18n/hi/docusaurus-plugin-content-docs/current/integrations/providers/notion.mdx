# Notion DB

>[Notion](https://www.notion.so/) एक सहयोग प्लेटफ़ॉर्म है जिसमें संशोधित Markdown समर्थन है जो कैनबन बोर्ड, कार्य, विकि और डेटाबेस को एकीकृत करता है। यह नोट लेने, ज्ञान और डेटा प्रबंधन, और परियोजना और कार्य प्रबंधन के लिए एक सर्वोत्तम कार्यस्थल है।

## स्थापना और सेटअप

सभी निर्देश उदाहरणों में दिए गए हैं।
