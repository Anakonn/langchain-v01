# लेकFS

>[लेकFS](https://docs.lakefs.io/) डेटा लेक पर स्केलेबल वर्जन नियंत्रण प्रदान करता है, और उन संस्करणों को बनाने और एक्सेस करने के लिए Git-जैसी semantics का उपयोग करता है।

## इंस्टॉलेशन और सेटअप

`ENDPOINT`, `LAKEFS_ACCESS_KEY`, और `LAKEFS_SECRET_KEY` प्राप्त करें।
आप इंस्टॉलेशन निर्देश यहां [पर](https://docs.lakefs.io/quickstart/launch.html) पा सकते हैं।
