# StochasticAI

यह पृष्ठ LangChain में StochasticAI पारिस्थितिकी तंत्र का उपयोग करने के बारे में कवर करता है।
यह दो भागों में विभाजित है: स्थापना और सेटअप, और फिर विशिष्ट StochasticAI रैपर्स के संदर्भ।

## स्थापना और सेटअप

- `pip install stochasticx` के साथ स्थापित करें
- एक StochasticAI api कुंजी प्राप्त करें और इसे एक पर्यावरण चर (`STOCHASTICAI_API_KEY`) के रूप में सेट करें
