# CerebriumAI

>[Cerebrium](https://docs.cerebrium.ai/cerebrium/getting-started/introduction) एक सर्वरलेस जीपीयू इन्फ्रास्ट्रक्चर प्रदाता है।
> यह कई एलएलएम मॉडल्स के लिए एपीआई एक्सेस प्रदान करता है।

[CerebriumAI दस्तावेज़](https://docs.cerebrium.ai/examples/langchain) में उदाहरण देखें।

## इंस्टॉलेशन और सेटअप

- एक पायथन पैकेज इंस्टॉल करें:

```bash
pip install cerebrium
```

- [एक CerebriumAI एपीआई कुंजी प्राप्त करें](https://docs.cerebrium.ai/cerebrium/getting-started/installation) और इसे एक पर्यावरण चर (`CEREBRIUMAI_API_KEY`) के रूप में सेट करें
