# लेखक

यह पृष्ठ LangChain में लेखक पारिस्थितिकी तंत्र का उपयोग करने के बारे में कवर करता है।
यह दो भागों में विभाजित है: स्थापना और सेटअप, और फिर विशिष्ट लेखक रैपर संदर्भ।

## स्थापना और सेटअप

- एक लेखक API कुंजी प्राप्त करें और इसे एक पर्यावरण चर (`WRITER_API_KEY`) के रूप में सेट करें

## रैपर

### LLM
