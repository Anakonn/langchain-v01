# Trello

>[Trello](https://www.atlassian.com/software/trello) एक वेब-आधारित परियोजना प्रबंधन और सहयोग उपकरण है जो व्यक्तियों और टीमों को अपने कार्यों और परियोजनाओं को संगठित और ट्रैक करने की अनुमति देता है। यह एक "बोर्ड" नामक दृश्यात्मक इंटरफ़ेस प्रदान करता है जहां उपयोगकर्ता अपने कार्यों और गतिविधियों को दर्शाने के लिए सूचियां और कार्ड बना सकते हैं।
>TrelloLoader हमें `Trello` बोर्ड से कार्ड लोड करने की अनुमति देता है।

## स्थापना और सेटअप

```bash
pip install py-trello beautifulsoup4
```
