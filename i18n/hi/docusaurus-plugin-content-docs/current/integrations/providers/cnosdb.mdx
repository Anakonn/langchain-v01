# CnosDB

> [CnosDB](https://github.com/cnosdb/cnosdb) एक ओपन-सोर्स वितरित टाइम सीरीज डेटाबेस है जो उच्च प्रदर्शन, उच्च संपीड़न दर और उच्च उपयोग में आसानी प्रदान करता है।

## इंस्टॉलेशन और सेटअप

```python
pip install cnos-connector
```

## CnosDB से कनेक्ट करना

आप `SQLDatabase.from_cnosdb()` विधि का उपयोग करके CnosDB से कनेक्ट कर सकते हैं।

### वाक्यविन्यास

```python
def SQLDatabase.from_cnosdb(url: str = "127.0.0.1:8902",
                              user: str = "root",
                              password: str = "",
                              tenant: str = "cnosdb",
                              database: str = "public")
```

तर्क:
1. url (str): CnosDB सेवा का HTTP कनेक्शन होस्ट नाम और पोर्ट नंबर, "http://" या "https://" को छोड़कर, डिफ़ॉल्ट मान "127.0.0.1:8902" है।
2. user (str): CnosDB सेवा से कनेक्ट करने के लिए उपयोग किया जाने वाला उपयोगकर्ता नाम, डिफ़ॉल्ट मान "root" है।
3. password (str): CnosDB सेवा से कनेक्ट करने वाले उपयोगकर्ता का पासवर्ड, डिफ़ॉल्ट मान "" है।
4. tenant (str): CnosDB सेवा से कनेक्ट करने के लिए उपयोग किया जाने वाला टेनेंट का नाम, डिफ़ॉल्ट मान "cnosdb" है।
5. database (str): CnosDB टेनेंट में डेटाबेस का नाम।

### SQL डेटाबेस श्रृंखला

यह उदाहरण CnosDB पर एक प्रश्न का उत्तर देने के लिए SQL श्रृंखला का उपयोग दर्शाता है।

```python
from langchain_community.utilities import SQLDatabaseChain

db_chain = SQLDatabaseChain.from_llm(llm, db, verbose=True)

db_chain.run(
    "What is the average temperature of air at station XiaoMaiDao between October 19, 2022 and Occtober 20, 2022?"
)
```
