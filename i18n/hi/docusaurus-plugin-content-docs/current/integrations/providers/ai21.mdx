# AI21 प्रयोगशाला

>[AI21 प्रयोगशाला](https://www.ai21.com/about) प्राकृतिक भाषा प्रसंस्करण (एनएलपी) में विशेषज्ञ कंपनी है, जो ऐसे AI प्रणाली विकसित करती है जो प्राकृतिक भाषा को समझ और उत्पन्न कर सकती है।

यह पृष्ठ `LangChain` में `AI21` पारिस्थितिकी तंत्र का उपयोग करने के बारे में कवर करता है।

## स्थापना और सेटअप

- एक AI21 api कुंजी प्राप्त करें और इसे एक पर्यावरण चर (`AI21_API_KEY`) के रूप में सेट करें
- Python पैकेज स्थापित करें:

```bash
pip install langchain-ai21
```
