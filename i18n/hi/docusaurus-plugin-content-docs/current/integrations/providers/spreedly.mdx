# स्प्रेडली

>[स्प्रेडली](https://docs.spreedly.com/) एक सेवा है जो आपको क्रेडिट कार्ड को सुरक्षित रूप से संग्रहीत करने और किसी भी संख्या में भुगतान गेटवे और तृतीय पक्ष API के खिलाफ लेनदेन करने में सक्षम बनाती है। यह एक साथ कार्ड टोकनाइज़ेशन/वॉल्ट सेवा और गेटवे और प्राप्तकर्ता एकीकरण सेवा प्रदान करके ऐसा करता है। स्प्रेडली द्वारा टोकनाइज़ किए गए भुगतान विधियों को `स्प्रेडली` पर संग्रहीत किया जाता है, जो आपको एक कार्ड को स्वतंत्र रूप से संग्रहीत करने और फिर अपनी व्यावसायिक आवश्यकताओं के आधार पर उस कार्ड को विभिन्न अंतिम बिंदुओं पर पास करने की अनुमति देता है।
