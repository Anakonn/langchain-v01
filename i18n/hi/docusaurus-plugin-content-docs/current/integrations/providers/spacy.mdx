# spaCy

>[spaCy](https://spacy.io/) एक उन्नत प्राकृतिक भाषा प्रसंस्करण के लिए एक ओपन-सोर्स सॉफ्टवेयर लाइब्रेरी है, जिसे प्रोग्रामिंग भाषाओं Python और Cython में लिखा गया है।

## इंस्टॉलेशन और सेटअप

```bash
pip install spacy
```
