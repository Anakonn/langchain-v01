# PromptLayer

>[PromptLayer](https://docs.promptlayer.com/introduction) एक प्रॉम्प्ट इंजीनियरिंग प्लेटफॉर्म है।
>यह एलएलएम (LLM) ऑब्जर्वेबिलिटी को भी मदद करता है ताकि अनुरोधों, वर्जन प्रॉम्प्ट्स और उपयोग को देखा जा सके।
>
>`PromptLayer` के पास एलएंगचेन (LangChain) के साथ सीधे एकीकृत एलएलएम (LLMs) हैं (जैसे
> [`PromptLayerOpenAI`](https://docs.promptlayer.com/languages/langchain)),
> लेकिन एलएंगचेन (LangChain) के साथ `PromptLayer` को एकीकृत करने का अनुशंसित तरीका कॉलबैक का उपयोग करना है।

## स्थापना और सेटअप

`PromptLayer` के साथ काम करने के लिए, हमें:
- एक `PromptLayer` खाता बनाना होगा
- एक एपीआई टोकन बनाना होगा और इसे एक पर्यावरण चर (`PROMPTLAYER_API_KEY`) के रूप में सेट करना होगा

एक Python पैकेज स्थापित करें:

```bash
pip install promptlayer
```
