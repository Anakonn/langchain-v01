# एनएलपीक्लाउड

>[एनएलपी क्लाउड](https://docs.nlpcloud.com/#introduction) एक कृत्रिम बुद्धिमत्ता प्लेटफॉर्म है जो आपको सबसे उन्नत एआई इंजनों का उपयोग करने और अपने डेटा के साथ अपने स्वयं के इंजनों को प्रशिक्षित करने की अनुमति देता है।

## स्थापना और सेटअप

- `nlpcloud` पैकेज स्थापित करें।

```bash
pip install nlpcloud
```

- एक एनएलपीक्लाउड एपीआई कुंजी प्राप्त करें और इसे एक पर्यावरण चर (`NLPCLOUD_API_KEY`) के रूप में सेट करें।
