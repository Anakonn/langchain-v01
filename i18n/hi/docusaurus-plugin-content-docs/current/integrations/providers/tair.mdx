# टेयर

>[Alibaba Cloud Tair](https://www.alibabacloud.com/help/en/tair/latest/what-is-tair) एक क्लाउड नेटिव इन-मेमोरी डेटाबेस सर्विस है
> जिसे `Alibaba Cloud` द्वारा विकसित किया गया है। यह समृद्ध डेटा मॉडल और उद्यम-स्तरीय क्षमताएं प्रदान करता है
> आपके रियल-टाइम ऑनलाइन सценारियो को समर्थन देने के लिए, पूर्ण संगतता बनाए रखते हुए `Redis` के साथ।
> `Tair` नए गैर-प्रवासी मेमोरी (एनवीएम) स्टोरेज माध्यम पर आधारित स्थायी मेमोरी-अनुकूलित इंस्टेंस भी पेश करता है।

## इंस्टॉलेशन और सेटअप

Tair Python SDK स्थापित करें:

```bash
pip install tair
```
