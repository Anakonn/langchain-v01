# Facebook - Meta

>[Meta Platforms, Inc.](https://www.facebook.com/), जो `Meta` के रूप में व्यवसाय करता है, पहले `Facebook, Inc.` और `TheFacebook, Inc.` के नाम से जाना जाता था, एक अमेरिकी बहुराष्ट्रीय प्रौद्योगिकी कंग्लोमरेट है। कंपनी `Facebook`, `Instagram`, `Threads` और `WhatsApp` सहित अन्य उत्पादों और सेवाओं का स्वामित्व और संचालन करती है।

## एम्बेडिंग मॉडल

### LASER

>[LASER](https://github.com/facebookresearch/LASER) एक Python लाइब्रेरी है जिसे `Meta AI Research` टीम द्वारा विकसित किया गया है और इसका उपयोग [2/25/2024 तक](https://github.com/facebookresearch/flores/blob/main/flores200/README.md#languages-in-flores-200) 147 से अधिक भाषाओं के लिए बहुभाषीय वाक्य एम्बेडिंग बनाने के लिए किया जाता है।

```bash
pip install laser_encoders
```

## वेक्टर स्टोर

### Facebook Faiss

>[Facebook AI Similarity Search (Faiss)](https://engineering.fb.com/2017/03/29/data-infrastructure/faiss-a-library-for-efficient-similarity-search/) घनत्व वेक्टरों के कुशल समानता खोज और क्लस्टरिंग के लिए एक लाइब्रेरी है। यह उन सेटों में खोज करने वाले एल्गोरिदम शामिल करता है जिनका आकार किसी भी हो, यहां तक कि वे जो RAM में नहीं समा सकते। यह मूल्यांकन और पैरामीटर ट्यूनिंग के लिए सहायक कोड भी शामिल करता है।

[Faiss दस्तावेज़](https://faiss.ai/)।

हमें `faiss` Python पैकेज इंस्टॉल करना होगा।

```bash
pip install faiss-gpu # For CUDA 7.5+ supported GPU's.
```

या

```bash
pip install faiss-cpu # For CPU Installation
```

[उपयोग उदाहरण](/docs/integrations/vectorstores/faiss) देखें।
