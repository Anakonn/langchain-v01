# ArcGIS

>[ArcGIS](https://www.esri.com/en-us/arcgis/about-arcgis/overview) एक परिवार है क्लाइंट, सर्वर और ऑनलाइन भौगोलिक सूचना प्रणाली सॉफ़्टवेयर का जिसका विकास और रखरखाव [Esri](https://www.esri.com/) द्वारा किया जाता है।

>`ArcGISLoader` का उपयोग `arcgis` पैकेज करता है।
> `arcgis` एक पायथन लाइब्रेरी है वेक्टर और रास्टर विश्लेषण, भू-कोडिंग, मानचित्र बनाने,
> रूटिंग और दिशाओं के लिए। यह प्रशासित करता है, संगठित करता है और उपयोगकर्ताओं को प्रबंधित करता है,
> आपके GIS में समूह और सूचना मद।
>यह `Esri` और अन्य प्राधिकृत स्रोतों से तैयार-उपयोग के मानचित्रों और क्यूरेटेड भौगोलिक डेटा तक पहुंच प्रदान करता है, और आपके खुद के डेटा के साथ भी काम करता है।

## स्थापना और सेटअप

हमें `arcgis` पैकेज स्थापित करना होगा।

```bash
pip install -U arcgis
```
