# SingleStoreDB

>[SingleStoreDB](https://singlestore.com/) एक उच्च-प्रदर्शन वितरित SQL डेटाबेस है जो [क्लाउड](https://www.singlestore.com/cloud/) और ऑन-प्रिमाइसेस दोनों में तैनाती का समर्थन करता है। यह वेक्टर स्टोरेज और वेक्टर फंक्शंस प्रदान करता है जिसमें [dot_product](https://docs.singlestore.com/managed-service/en/reference/sql-reference/vector-functions/dot_product.html) और [euclidean_distance](https://docs.singlestore.com/managed-service/en/reference/sql-reference/vector-functions/euclidean_distance.html) शामिल हैं, जिससे उन एआई एप्लिकेशन का समर्थन होता है जिन्हें टेक्स्ट समानता मिलान की आवश्यकता होती है।

## इंस्टॉलेशन और सेटअप

डेटाबेस से [कनेक्शन](https://singlestoredb-python.labs.singlestore.com/generated/singlestoredb.connect.html) स्थापित करने के कई तरीके हैं। आप या तो एनवायरनमेंट वेरिएबल्स सेट कर सकते हैं या `SingleStoreDB constructor` को नामित पैरामीटर पास कर सकते हैं। वैकल्पिक रूप से, आप इन पैरामीटर्स को `from_documents` और `from_texts` मेथड्स को भी प्रदान कर सकते हैं।

```bash
pip install singlestoredb
```
