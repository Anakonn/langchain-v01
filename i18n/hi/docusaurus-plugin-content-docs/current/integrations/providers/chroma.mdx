# क्रोमा

>[क्रोमा](https://docs.trychroma.com/getting-started) एम्बेडिंग के साथ AI अनुप्रयोग बनाने के लिए एक डेटाबेस है।

## स्थापना और सेटअप

```bash
pip install langchain-chroma
```

## VectorStore

क्रोमा वेक्टर डेटाबेस के इर्द-गिर्द एक रैपर मौजूद है, जो आपको इसका उपयोग वेक्टर स्टोर के रूप में करने की अनुमति देता है,
चाहे वह语义 खोज या उदाहरण चयन के लिए हो।
