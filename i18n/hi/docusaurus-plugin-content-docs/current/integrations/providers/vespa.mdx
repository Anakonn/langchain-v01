# वेस्पा

>[वेस्पा](https://vespa.ai/) एक पूरी तरह से सुविधाजनक खोज इंजन और वेक्टर डेटाबेस है।
> यह वेक्टर खोज (एएनएन), लेक्सिकल खोज और संरचित डेटा में खोज सभी को एक ही क्वेरी में समर्थित करता है।

## इंस्टॉलेशन और सेटअप

```bash
pip install pyvespa
```
