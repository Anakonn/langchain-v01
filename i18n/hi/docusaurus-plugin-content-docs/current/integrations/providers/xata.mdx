# Xata

> [Xata](https://xata.io) एक सर्वरलेस डेटा प्लेटफॉर्म है, जो `PostgreSQL` पर आधारित है।
> यह आपके डेटाबेस के साथ बातचीत करने के लिए एक Python SDK और अपने डेटा को प्रबंधित करने के लिए एक UI प्रदान करता है।
> `Xata` में एक नेटिव वेक्टर प्रकार है, जिसे किसी भी तालिका में जोड़ा जा सकता है, और यह समानता खोज का समर्थन करता है। LangChain `Xata` में सीधे वेक्टर डालता है, और किसी दिए गए वेक्टर के निकटतम पड़ोसियों को खोजने के लिए इसका प्रश्न पूछता है, ताकि आप `Xata` के साथ सभी LangChain एम्बेडिंग एकीकरण का उपयोग कर सकें।

## स्थापना और सेटअप

हमें `xata` पायथन पैकेज को स्थापित करना होगा।

```bash
pip install xata==1.0.0a7
```

## वेक्टर स्टोर

एक [उपयोग उदाहरण](/docs/integrations/vectorstores/xata) देखें।

```python

```
