# ArangoDB

>[ArangoDB](https://github.com/arangodb/arangodb) एक स्केलेबल ग्राफ डेटाबेस सिस्टम है जो कनेक्टेड डेटा से तेजी से मूल्य प्राप्त करने में मदद करता है। नेटिव ग्राफ, एकीकृत खोज इंजन और JSON समर्थन, एक ही क्वेरी भाषा के माध्यम से। ArangoDB ऑन-प्रेम, क्लाउड में - कहीं भी चलता है।

## स्थापना और सेटअप

[ArangoDB Python Driver](https://github.com/ArangoDB-Community/python-arango) पैकेज को इंस्टॉल करें

```bash
pip install python-arango
```
