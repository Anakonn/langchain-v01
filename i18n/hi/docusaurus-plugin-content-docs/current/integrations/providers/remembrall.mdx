# रेमेम्बरॉल

>[रेमेम्बरॉल](https://remembrall.dev/) एक प्लेटफॉर्म है जो एक भाषा मॉडल को दीर्घकालिक मेमोरी, पुनर्प्राप्ति सहायक जनरेशन और पूर्ण अवलोकनक्षमता प्रदान करता है।

## इंस्टॉलेशन और सेटअप

शुरू करने के लिए, [गिटहब पर साइन इन करें और रेमेम्बरॉल प्लेटफॉर्म पर](https://remembrall.dev/login) अपना [सेटिंग पृष्ठ से API कुंजी कॉपी करें](https://remembrall.dev/dashboard/settings)।
