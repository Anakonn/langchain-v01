# वीवाइट

>[वीवाइट](https://weaviate.io/) एक ओपन-सोर्स वेक्टर डेटाबेस है। यह आपको डेटा ऑब्जेक्ट और अपने पसंदीदा एमएल मॉडल से वेक्टर एम्बेडिंग्स को संग्रहित करने और अरबों डेटा ऑब्जेक्ट्स में सुचारू रूप से स्केल करने की अनुमति देता है।

`वीवाइट` क्या है?
- वीवाइट एक ओपन-सोर्स ​वेक्टर सर्च इंजन ​प्रकार का डेटाबेस है।
- वीवाइट आपको JSON दस्तावेज़ों को एक वर्ग गुण-सा तरीके से संग्रहित करने और इन दस्तावेज़ों को वेक्टर स्पेस में प्रतिनिधित्व करने के लिए मशीन लर्निंग वेक्टर संलग्न करने की अनुमति देता है।
- वीवाइट को स्टैंडअलोन (यानी अपने वेक्टर लाओ) या विभिन्न मॉड्यूलों के साथ इस्तेमाल किया जा सकता है जो वेक्टराइज़ेशन कर सकते हैं और कोर क्षमताओं का विस्तार कर सकते हैं।
- वीवाइट में आपके डेटा तक आसानी से पहुंचने के लिए एक GraphQL-API है।
- हम केवल मिलीसेकंड में क्वेरी करने के लिए आपके वेक्टर सर्च सेटअप को उत्पादन में लाने का लक्ष्य रखते हैं (देखें कि क्या वीवाइट आपके उपयोग मामले के लिए उपयुक्त है [ओपन-सोर्स बेंचमार्क](https://weaviate.io/developers/weaviate/current/benchmarks/))।
- [मूल शुरुआत गाइड](https://weaviate.io/developers/weaviate/current/core-knowledge/basics.html) में पांच मिनट में वीवाइट के बारे में जानें।

**वीवाइट का विस्तार से विवरण:**

`वीवाइट` एक कम-लेटेंसी वेक्टर सर्च इंजन है जिसमें अलग-अलग मीडिया प्रकारों (पाठ, छवि आदि) के लिए आउट-ऑफ-द-बॉक्स समर्थन है। यह语义搜索, प्रश्न-उत्तर निष्कर्षण, वर्गीकरण, कस्टमाइज़ेबल मॉडल (PyTorch/TensorFlow/Keras) आदि प्रदान करता है। Go में से शुरू किया गया, वीवाइट ऑब्जेक्ट और वेक्टर दोनों को संग्रहित करता है, जिससे वेक्टर सर्च को संरचित फ़िल्टरिंग और क्लाउड-नेटिव डेटाबेस की त्रुटि सहिष्णुता के साथ जोड़ा जा सकता है। यह सब GraphQL, REST और विभिन्न क्लाइंट-साइड प्रोग्रामिंग भाषाओं के माध्यम से पहुंचने योग्य है।

## स्थापना और सेटअप

Python SDK स्थापित करें:

```bash
pip install langchain-weaviate
```

## वेक्टर स्टोर

`वीवाइट` इंडेक्स के इर्द-गिर्द एक रैपर मौजूद है, जिससे आप इसका उपयोग वेक्टर स्टोर के रूप में कर सकते हैं, चाहे वह सेमेंटिक सर्च या उदाहरण चयन के लिए हो।

इस वेक्टर स्टोर को आयात करने के लिए:

```python
from langchain_weaviate import WeaviateVectorStore
```

वीवाइट रैपर के बारे में अधिक विस्तृत वॉकथ्रू के लिए, [यह नोटबुक](/docs/integrations/vectorstores/weaviate) देखें
