# मनोवैज्ञानिक

:::warning
यह प्रदाता अब अनुरक्षित नहीं है, और काम न कर सकता है। सावधानी से उपयोग करें।
:::

>[मनोवैज्ञानिक](https://www.psychic.dev/) एक प्लेटफॉर्म है जो `Notion`, `Zendesk`, `Confluence` और `Google Drive` जैसे SaaS टूल्स के साथ OAuth के माध्यम से एकीकरण और इन एप्लिकेशनों से अनरचित डेटा को आपके SQL या वेक्टर डेटाबेस में सिंक करने के लिए है। आप इसे अनरचित डेटा के लिए Plaid की तरह सोच सकते हैं।

## स्थापना और सेटअप

```bash
pip install psychicapi
```

मनोवैज्ञानिक को सेट अप करना आसान है - आप `react` लाइब्रेरी आयात करते हैं और इसे अपने `Sidekick API` कुंजी के साथ कॉन्फ़िगर करते हैं, जिसे आप [मनोवैज्ञानिक डैशबोर्ड](https://dashboard.psychic.dev/) से प्राप्त करते हैं। जब आप एप्लिकेशन्स को कनेक्ट करते हैं, तो आप डैशबोर्ड से इन कनेक्शन्स को देख सकते हैं और सर्वर-साइड लाइब्रेरीज का उपयोग करके डेटा प्राप्त कर सकते हैं।

1. [डैशबोर्ड](https://dashboard.psychic.dev/) में एक खाता बनाएं।
2. अपने फ्रंटएंड रिएक्ट ऐप में मनोवैज्ञानिक लिंक मॉडल जोड़ने के लिए [रिएक्ट लाइब्रेरी](https://docs.psychic.dev/sidekick-link) का उपयोग करें। आप इसका उपयोग SaaS ऐप्स को कनेक्ट करने के लिए करेंगे।
3. एक कनेक्शन बना लेने के बाद, आप [उदाहरण नोटबुक](/docs/integrations/document_loaders/psychic) का पालन करके `PsychicLoader` का उपयोग कर सकते हैं

## अन्य डॉक्यूमेंट लोडर्स के मुकाबले लाभ

1.	**सार्वभौमिक API:** हर SaaS ऐप के लिए OAuth प्रवाह और API सीखने के बजाय, आप एक बार मनोवैज्ञानिक को एकीकृत करते हैं और हमारे सार्वभौमिक API का उपयोग करके डेटा प्राप्त करते हैं।
2.	**डेटा सिंक:** आपके ग्राहकों के SaaS ऐप्स में डेटा जल्दी ही पुराना हो सकता है। मनोवैज्ञानिक के साथ, आप दैनिक या रियल-टाइम आधार पर अपने दस्तावेजों को अद्यतन रखने के लिए वेबहुक कॉन्फ़िगर कर सकते हैं।
3.	**सरलीकृत OAuth:** मनोवैज्ञानिक OAuth को एंड-टू-एंड संभालता है ताकि आपको प्रत्येक एकीकरण के लिए OAuth क्लाइंट बनाने, एक्सेस टोकन को ताजा रखने और OAuth रीडायरेक्ट लॉजिक को संभालने का समय नहीं लगाना पड़े।
