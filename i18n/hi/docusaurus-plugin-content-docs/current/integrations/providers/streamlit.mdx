# स्ट्रीमलिट

>[स्ट्रीमलिट](https://streamlit.io/) डेटा ऐप्स को बनाने और साझा करने का एक तेज़ तरीका है।
>`स्ट्रीमलिट` डेटा स्क्रिप्ट को मिनटों में साझा करने योग्य वेब ऐप्स में बदल देता है। सभी पायथन में। कोई फ्रंट-एंड अनुभव की आवश्यकता नहीं है।
>[streamlit.io/generative-ai](https://streamlit.io/generative-ai) पर और उदाहरण देखें।

## इंस्टॉलेशन और सेटअप

हमें `streamlit` पायथन पैकेज को इंस्टॉल करना होगा:

```bash
pip install streamlit
```
