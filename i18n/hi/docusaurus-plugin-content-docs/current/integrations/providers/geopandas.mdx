# Geopandas

>[GeoPandas](https://geopandas.org/) एक ओपन सोर्स प्रोजेक्ट है जो पायथन में भौगोलिक डेटा के साथ काम करना आसान बनाता है। `GeoPandas` `pandas` द्वारा उपयोग किए जाने वाले डेटा प्रकारों का विस्तार करता है ताकि ज्यामितीय प्रकारों पर स्थानिक ऑपरेशन किए जा सकें। ज्यामितीय ऑपरेशन `shapely` द्वारा किए जाते हैं।

## स्थापना और सेटअप

हमें कई पायथन पैकेज स्थापित करने होंगे।

```bash
pip install -U sodapy pandas geopandas
```
