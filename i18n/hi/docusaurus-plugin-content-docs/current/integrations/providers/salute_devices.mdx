# सलूट डिवाइस

सलूट डिवाइस जीगाचैट एलएलएम मॉडल प्रदान करता है।

जीगाचैट तक पहुंचने के लिए अधिक जानकारी के लिए [यहां](https://developers.sber.ru/docs/ru/gigachat/api/integration) देखें।

## इंस्टॉलेशन और सेटअप

जीगाचैट पैकेज पीआईपी से पीआईपीआई से इंस्टॉल किया जा सकता है:

```bash
pip install gigachat
```
