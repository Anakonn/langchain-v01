# टाइगरग्राफ

>[टाइगरग्राफ](https://www.tigergraph.com/tigergraph-db/) एक मूल रूप से वितरित और उच्च-प्रदर्शन ग्राफ डेटाबेस है।
> डेटा को शीर्षक और किनारों के ग्राफ प्रारूप में संग्रहीत करने से समृद्ध संबंध होते हैं,
> एलएलएम प्रतिक्रियाओं को आधार देने के लिए आदर्श।

## स्थापना और सेटअप

[कैसे `टाइगरग्राफ` डेटाबेस से कनेक्ट करें](https://docs.tigergraph.com/pytigergraph/current/getting-started/connection) के निर्देशों का पालन करें।

पायथन एसडीके स्थापित करें:

```bash
pip install pyTigerGraph
```
