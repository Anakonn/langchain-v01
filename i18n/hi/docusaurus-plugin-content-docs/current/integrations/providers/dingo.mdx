# DingoDB

यह पृष्ठ LangChain में DingoDB पारिस्थितिकी तंत्र का उपयोग करने के बारे में कवर करता है।
यह दो भागों में विभाजित है: स्थापना और सेटअप, और फिर विशिष्ट DingoDB रैपर्स के संदर्भ।

## स्थापना और सेटअप

- `pip install dingodb` के साथ Python SDK को स्थापित करें

## VectorStore

DingoDB इंडेक्स के इर्द-गिर्द एक रैपर मौजूद है, जिससे आप इसका उपयोग एक वेक्टर स्टोर के रूप में कर सकते हैं,
चाहे वह语义 खोज या उदाहरण चयन के लिए हो।

इस वेक्टर स्टोर को आयात करने के लिए:

```python
<!--IMPORTS:[{"imported": "Dingo", "source": "langchain_community.vectorstores", "docs": "https://api.python.langchain.com/en/latest/vectorstores/langchain_community.vectorstores.dingo.Dingo.html", "title": "DingoDB"}]-->
from langchain_community.vectorstores import Dingo
```

DingoDB रैपर के बारे में अधिक विस्तृत वॉकथ्रू के लिए, [इस नोटबुक](/docs/integrations/vectorstores/dingo) देखें
