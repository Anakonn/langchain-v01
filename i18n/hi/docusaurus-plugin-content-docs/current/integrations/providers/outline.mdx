# रूपरेखा

> [रूपरेखा](https://www.getoutline.com/) एक ओपन-सोर्स सहयोगात्मक ज्ञान आधार प्लेटफॉर्म है जो टीम सूचना साझाकरण के लिए डिज़ाइन किया गया है।

## सेटअप

आपको पहले अपने Outline इंस्टेंस के लिए एक [api कुंजी बनाना](https://www.getoutline.com/developers#section/Authentication) होगा। फिर आपको निम्नलिखित पर्यावरण चर सेट करने की आवश्यकता है:

```python
import os

os.environ["OUTLINE_API_KEY"] = "xxx"
os.environ["OUTLINE_INSTANCE_URL"] = "https://app.getoutline.com"
```
