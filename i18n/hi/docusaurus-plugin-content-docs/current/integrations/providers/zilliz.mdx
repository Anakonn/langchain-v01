# जिल्लिज़

>[जिल्लिज़ क्लाउड](https://zilliz.com/doc/quick_start) एलएफ एआई मिल्व्यूस®के लिए क्लाउड पर पूरी तरह से प्रबंधित सेवा है,

## इंस्टॉलेशन और सेटअप

पायथन एसडीके इंस्टॉल करें:

```bash
pip install pymilvus
```
