# ओरेकल क्लाउड इंफ्रास्ट्रक्चर (OCI)

[ओरेकल क्लाउड इंफ्रास्ट्रक्चर](https://www.oracle.com/artificial-intelligence/) से संबंधित `LangChain` एकीकरण।

## LLMs

### OCI जनरेटिव एआई

> ओरेकल क्लाउड इंफ्रास्ट्रक्चर (OCI) [जनरेटिव एआई](https://docs.oracle.com/en-us/iaas/Content/generative-ai/home.htm) एक पूरी तरह से प्रबंधित सेवा है जो विभिन्न उपयोग मामलों को कवर करने वाले अत्याधुनिक, अनुकूलनीय बड़े भाषा मॉडल (LLM) का एक सेट प्रदान करती है, जो एक एकल API के माध्यम से उपलब्ध हैं।
> OCI जनरेटिव एआई सेवा का उपयोग करके आप तैयार-उपयोग के पूर्व प्रशिक्षित मॉडल का उपयोग कर सकते हैं, या अपने डेटा पर आधारित अपने कस्टम मॉडल को बना और होस्ट कर सकते हैं।

उपयोग करने के लिए, आपके पास नवीनतम `oci` पायथन एसडीके इंस्टॉल होना चाहिए।

```bash
pip install -U oci
```

[उपयोग उदाहरणों](/docs/integrations/llms/oci_generative_ai) देखें।

### OCI डेटा साइंस मॉडल डिप्लॉयमेंट एंडपॉइंट

> [OCI डेटा साइंस](https://docs.oracle.com/en-us/iaas/data-science/using/home.htm) डेटा साइंस टीमों के लिए एक पूरी तरह से प्रबंधित और सर्वरलेस प्लेटफॉर्म है। OCI डेटा साइंस प्लेटफॉर्म का उपयोग करके आप मशीन लर्निंग मॉडल को बना, प्रशिक्षित और प्रबंधित कर सकते हैं, और फिर उन्हें
> [OCI डेटा साइंस मॉडल डिप्लॉयमेंट सेवा](https://docs.oracle.com/en-us/iaas/data-science/using/model-dep-about.htm) का उपयोग करके OCI मॉडल डिप्लॉयमेंट एंडपॉइंट के रूप में तैनात कर सकते हैं।

यदि आपने VLLM या TGI फ्रेमवर्क के साथ एक LLM तैनात किया है, तो आप `OCIModelDeploymentVLLM` या `OCIModelDeploymentTGI` क्लास का उपयोग करके इससे बातचीत कर सकते हैं।

उपयोग करने के लिए, आपके पास नवीनतम `oracle-ads` पायथन एसडीके इंस्टॉल होना चाहिए।

```bash
pip install -U oracle-ads
```

[उपयोग उदाहरणों](/docs/integrations/llms/oci_model_deployment_endpoint) देखें।
