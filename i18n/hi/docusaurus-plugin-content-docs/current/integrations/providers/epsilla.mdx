# एप्सिला

यह पृष्ठ LangChain में [एप्सिला](https://github.com/epsilla-cloud/vectordb) का उपयोग करने के बारे में कवर करता है।
यह दो भागों में विभाजित है: स्थापना और सेटअप, और फिर विशिष्ट एप्सिला रैपर्स के संदर्भ।

## स्थापना और सेटअप

- `pip/pip3 install pyepsilla` के साथ पायथन एसडीके स्थापित करें

## रैपर्स

### वेक्टर स्टोर

एप्सिला वेक्टर डेटाबेस के इर्द-गिर्द एक रैपर मौजूद है, जो आपको इसका उपयोग एक वेक्टर स्टोर के रूप में करने की अनुमति देता है,
चाहे वह语义 खोज या उदाहरण चयन के लिए हो।

इस वेक्टर स्टोर को आयात करने के लिए:

```python
<!--IMPORTS:[{"imported": "Epsilla", "source": "langchain_community.vectorstores", "docs": "https://api.python.langchain.com/en/latest/vectorstores/langchain_community.vectorstores.epsilla.Epsilla.html", "title": "Epsilla"}]-->
from langchain_community.vectorstores import Epsilla
```

एप्सिला रैपर के बारे में अधिक विस्तृत वॉकथ्रू के लिए, [यह नोटबुक](/docs/integrations/vectorstores/epsilla) देखें
