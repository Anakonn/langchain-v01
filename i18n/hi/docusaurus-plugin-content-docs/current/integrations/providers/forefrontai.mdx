# ForefrontAI

यह पृष्ठ LangChain में ForefrontAI पारिस्थितिकी तंत्र का उपयोग करने के बारे में कवर करता है।
यह दो भागों में विभाजित है: स्थापना और सेटअप, और फिर विशिष्ट ForefrontAI रैपर्स के संदर्भ।

## स्थापना और सेटअप

- एक ForefrontAI api कुंजी प्राप्त करें और इसे एक पर्यावरण चर (`FOREFRONTAI_API_KEY`) के रूप में सेट करें

## रैपर्स

### LLM

ForefrontAI LLM रैपर मौजूद है, जिसका उपयोग आप इस प्रकार कर सकते हैं।
