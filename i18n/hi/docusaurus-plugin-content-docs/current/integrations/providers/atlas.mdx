# एटलस

>[नोमिक एटलस](https://docs.nomic.ai/index.html) एक ऐसा प्लेटफॉर्म है जिसका उपयोग छोटे और इंटरनेट स्केल के अनरचित डेटासेट के साथ इंटरैक्ट करने के लिए किया जा सकता है।

## इंस्टॉलेशन और सेटअप

- `pip install nomic` का उपयोग करके पायथन पैकेज इंस्टॉल करें
- `नोमिक` को लैंगचेन्स पोएट्री एक्स्ट्रास में भी शामिल किया गया है `poetry install -E all`
