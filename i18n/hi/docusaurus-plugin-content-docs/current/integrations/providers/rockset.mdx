# रॉकसेट

>[रॉकसेट](https://rockset.com/product/) एक रियल-टाइम एनालिटिक्स डेटाबेस सर्विस है जो स्केल पर कम लेटेंसी और उच्च कंकरेंसी एनालिटिकल क्वेरी को सर्व करता है। यह एक कन्वर्ज्ड इंडेक्स™ बनाता है जो संरचित और अर्ध-संरचित डेटा पर काम करता है और वेक्टर एम्बेडिंग के लिए एक कुशल स्टोर प्रदान करता है। स्कीमाहीन डेटा पर SQL चलाने का इसका समर्थन इसे मेटाडेटा फ़िल्टर के साथ वेक्टर सर्च चलाने के लिए एक आदर्श विकल्प बनाता है।

## इंस्टॉलेशन और सेटअप

सुनिश्चित करें कि आपके पास रॉकसेट खाता है और वेब कंसोल पर जाकर API कुंजी प्राप्त करें। विवरण [वेबसाइट](https://rockset.com/docs/rest-api/) पर मिल सकते हैं।

```bash
pip install rockset
```
