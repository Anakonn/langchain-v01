# Figma

>[Figma](https://www.figma.com/) एक सहयोगी वेब एप्लिकेशन है जो इंटरफ़ेस डिज़ाइन के लिए है।

## स्थापना और सेटअप

Figma API को एक `access token`, `node_ids`, और एक `file key` की आवश्यकता होती है।

`file key` को URL से प्राप्त किया जा सकता है। https://www.figma.com/file/{filekey}/sampleFilename

`Node IDs` भी URL में उपलब्ध हैं। किसी भी चीज़ पर क्लिक करें और '?node-id=\{node_id}' पैरामीटर को देखें।

`Access token` [निर्देश](https://help.figma.com/hc/en-us/articles/8085703771159-Manage-personal-access-tokens)।
