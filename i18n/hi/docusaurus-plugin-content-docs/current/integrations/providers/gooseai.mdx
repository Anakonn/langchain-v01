# GooseAI

यह पृष्ठ LangChain में GooseAI पारिस्थितिकी तंत्र का उपयोग करने के बारे में कवर करता है।
यह दो भागों में विभाजित है: स्थापना और सेटअप, और फिर विशिष्ट GooseAI रैपर्स के संदर्भ।

## स्थापना और सेटअप

- `pip install openai` के साथ Python SDK स्थापित करें
- यहां से अपना GooseAI api कुंजी प्राप्त करें [यहां](https://goose.ai/)।
- पर्यावरण चर (`GOOSEAI_API_KEY`) सेट करें।

```python
import os
os.environ["GOOSEAI_API_KEY"] = "YOUR_API_KEY"
```
