# मोटरहेड

>[मोटरहेड](https://github.com/getmetal/motorhead) एक रस्ट में कार्यान्वित मेमोरी सर्वर है। यह पृष्ठभूमि में स्वचालित रूप से क्रमिक सारांशीकरण को संभालता है और स्थिरता रहित अनुप्रयोगों की अनुमति देता है।
