# बैडू

>[बैडू क्लाउड](https://cloud.baidu.com/) बैडू, इंक. द्वारा प्रदान किया जाने वाला एक क्लाउड सेवा है, जिसका मुख्यालय बीजिंग में है। यह क्लाउड स्टोरेज सेवा, क्लाइंट सॉफ्टवेयर, फ़ाइल प्रबंधन, संसाधन साझाकरण और तृतीय पक्ष एकीकरण प्रदान करता है।

## इंस्टॉलेशन और सेटअप

`Qianfan` `AK` और `SK` कुंजियां यहाँ से पंजीकरण और प्राप्त करें [यहाँ](https://cloud.baidu.com/product/wenxinworkshop)।
