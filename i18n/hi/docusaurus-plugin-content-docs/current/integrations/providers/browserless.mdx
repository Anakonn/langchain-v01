# ब्राउज़रलेस

>[ब्राउज़रलेस](https://www.browserless.io/docs/start) एक सेवा है जो आपको क्लाउड में हेडलेस क्रोम इंस्टेंस चलाने की अनुमति देती है। यह स्केल पर ब्राउज़र-आधारित ऑटोमेशन चलाने का एक शानदार तरीका है, बिना अपने अपने बुनियादी ढांचे का प्रबंधन करने की चिंता किए।

## इंस्टॉलेशन और सेटअप

हमें [यहां](https://www.browserless.io/pricing/) से API कुंजी प्राप्त करनी होगी।
