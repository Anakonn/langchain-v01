# इयूगू

>[इयूगू](https://www.iugu.com/) एक ब्राजीलियाई सेवाएं और सॉफ्टवेयर-ऐज-ए-सर्विस (SaaS) कंपनी है। यह ई-कॉमर्स वेबसाइटों और मोबाइल एप्लिकेशनों के लिए भुगतान-प्रोसेसिंग सॉफ्टवेयर और एप्लिकेशन प्रोग्रामिंग इंटरफ़ेस प्रदान करती है।

## इंस्टॉलेशन और सेटअप

`इयूगू API` को एक एक्सेस टोकन की आवश्यकता होती है, जिसे `इयूगू` डैशबोर्ड के अंदर पाया जा सकता है।
