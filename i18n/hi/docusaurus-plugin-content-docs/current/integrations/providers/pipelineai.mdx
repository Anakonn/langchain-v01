# पाइपलाइनएआई

यह पृष्ठ LangChain में पाइपलाइनएआई पारिस्थितिकी तंत्र का उपयोग करने के बारे में कवर करता है।
यह दो भागों में विभाजित है: स्थापना और सेटअप, और फिर विशिष्ट पाइपलाइनएआई रैपर्स के संदर्भ।

## स्थापना और सेटअप

- `pip install pipeline-ai` के साथ स्थापित करें
- एक पाइपलाइन क्लाउड api कुंजी प्राप्त करें और इसे एक पर्यावरण चर (`PIPELINE_API_KEY`) के रूप में सेट करें
