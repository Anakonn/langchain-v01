# गिट

>[गिट](https://en.wikipedia.org/wiki/Git) एक वितरित संस्करण नियंत्रण प्रणाली है जो किसी भी कंप्यूटर फ़ाइलों के सेट में परिवर्तनों को ट्रैक करता है, आमतौर पर सॉफ्टवेयर विकास के दौरान सहयोगी रूप से सोर्स कोड पर काम करने वाले प्रोग्रामरों के बीच समन्वय के लिए उपयोग किया जाता है।

## इंस्टॉलेशन और सेटअप

पहले, आपको `GitPython` पायथन पैकेज इंस्टॉल करना होगा।

```bash
pip install GitPython
```
