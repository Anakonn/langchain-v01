# कूचबेस

>[कूचबेस](http://couchbase.com/) एक पुरस्कार विजेता वितरित NoSQL क्लाउड डेटाबेस है
> जो अपरिमित बहुमुखता, प्रदर्शन, पैमाना और वित्तीय मूल्य प्रदान करता है
> आपके सभी क्लाउड, मोबाइल, AI और एज कंप्यूटिंग एप्लिकेशन के लिए।

## इंस्टॉलेशन और सेटअप

हमें `couchbase` पैकेज इंस्टॉल करना होगा।

```bash
pip install couchbase
```
