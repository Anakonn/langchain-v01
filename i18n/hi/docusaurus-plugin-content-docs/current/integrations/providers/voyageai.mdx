# VoyageAI

VoyageAI से संबंधित सभी कार्यक्षमता

>[VoyageAI](https://www.voyageai.com/) Voyage AI आपके डोमेन और कंपनी के लिए अनुकूलित एम्बेडिंग मॉडल बनाता है, बेहतर पुनर्प्राप्ति गुणवत्ता के लिए।

## स्थापना और सेटअप

एकीकरण पैकेज को इंस्टॉल करें

```bash
pip install langchain-voyageai
```

एक VoyageAI API कुंजी प्राप्त करें और इसे एक पर्यावरण चर (`VOYAGE_API_KEY`) के रूप में सेट करें

## पाठ एम्बेडिंग मॉडल

एक [उपयोग उदाहरण](/docs/integrations/text_embedding/voyageai) देखें

