# टाइग्रिस

> [टाइग्रिस](https://tigrisdata.com) एक ओपन-सोर्स सर्वरलेस NoSQL डेटाबेस और सर्च प्लेटफॉर्म है जो उच्च-प्रदर्शन वेक्टर सर्च एप्लिकेशन बनाने को सरल बनाने के लिए डिज़ाइन किया गया है।
> `टाइग्रिस` आपको महत्वपूर्ण एप्लिकेशन बनाने पर ध्यान केंद्रित करने की अनुमति देता है, क्योंकि यह कई उपकरणों को प्रबंधित, संचालित और समन्वयित करने की बुनियादी ढांचा की जटिलता को समाप्त कर देता है।

## इंस्टॉलेशन और सेटअप

```bash
pip install tigrisdb openapi-schema-pydantic
```
