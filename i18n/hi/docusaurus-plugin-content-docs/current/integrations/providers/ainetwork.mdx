# AINetwork

>[AI नेटवर्क](https://www.ainetwork.ai/build-on-ain) एक लेयर 1 ब्लॉकचेन है जो बड़े पैमाने पर AI मॉडल्स को समायोजित करने के लिए डिज़ाइन किया गया है, जो [$AIN टोकन](https://www.ainetwork.ai/token) द्वारा संचालित एक विकेंद्रीकृत GPU नेटवर्क का उपयोग करता है, जो AI-संचालित `NFTs` (`AINFTs`) को समृद्ध करता है।

## स्थापना और सेटअप

आपको `ain-py` पायथन पैकेज स्थापित करना होगा।

```bash
pip install ain-py
```

आपको `AIN_BLOCKCHAIN_ACCOUNT_PRIVATE_KEY` पर्यावरण चर को अपने AIN ब्लॉकचेन खाते के निजी कुंजी से सेट करना होगा।
