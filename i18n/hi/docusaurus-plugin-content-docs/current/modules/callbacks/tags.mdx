# टैग

आप अपने कॉलबैक में टैग जोड़ सकते हैं `call()`/`run()`/`apply()` विधियों में `tags` तर्क पास करके। यह आपके लॉग को फ़िल्टर करने के लिए उपयोगी है, उदाहरण के लिए, यदि आप किसी विशिष्ट `LLMChain` को किए गए सभी अनुरोधों को लॉग करना चाहते हैं, तो आप एक टैग जोड़ सकते हैं, और फिर उस टैग द्वारा अपने लॉग को फ़िल्टर कर सकते हैं। आप कंस्ट्रक्टर और अनुरोध कॉलबैक दोनों में टैग पास कर सकते हैं, विवरण के लिए ऊपर दिए गए उदाहरणों को देखें। ये टैग फिर "शुरू" कॉलबैक विधियों के `tags` तर्क में पारित किए जाते हैं, अर्थात् `on_llm_start`, `on_chat_model_start`, `on_chain_start`, `on_tool_start`।
