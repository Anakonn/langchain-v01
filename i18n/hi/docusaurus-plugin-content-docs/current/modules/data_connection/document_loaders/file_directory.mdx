यह कवरता है कि सभी दस्तावेजों को एक निर्देशिका में कैसे लोड किया जाए।

छिपे हुए, डिफ़ॉल्ट रूप से यह [UnstructuredLoader](/docs/integrations/document_loaders/unstructured_file) का उपयोग करता है।

```python
<!--IMPORTS:[{"imported": "DirectoryLoader", "source": "langchain_community.document_loaders", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.directory.DirectoryLoader.html", "title": "File Directory"}]-->
from langchain_community.document_loaders import DirectoryLoader
```

हम `glob` पैरामीटर का उपयोग कर सकते हैं कि कौन से फ़ाइलें लोड की जाएं। ध्यान दें कि यहां यह `.rst` फ़ाइल या `.html` फ़ाइलों को लोड नहीं करता है।

```python
loader = DirectoryLoader('../', glob="**/*.md")
```

```python
docs = loader.load()
```

```python
len(docs)
```

```output
    1
```

## प्रगति पट्टी दिखाएं

डिफ़ॉल्ट रूप से प्रगति पट्टी नहीं दिखाई जाएगी। प्रगति पट्टी दिखाने के लिए, `tqdm` लाइब्रेरी (उदा. `pip install tqdm`) इंस्टॉल करें, और `show_progress` पैरामीटर को `True` पर सेट करें।

```python
loader = DirectoryLoader('../', glob="**/*.md", show_progress=True)
docs = loader.load()
```

```output

    0it [00:00, ?it/s]
```

## मल्टीथ्रेडिंग का उपयोग करें

डिफ़ॉल्ट रूप से लोडिंग एक थ्रेड में होती है। कई थ्रेड का उपयोग करने के लिए, `use_multithreading` फ्लैग को सच पर सेट करें।

```python
loader = DirectoryLoader('../', glob="**/*.md", use_multithreading=True)
docs = loader.load()
```

## लोडर क्लास बदलें

डिफ़ॉल्ट रूप से यह `UnstructuredLoader` क्लास का उपयोग करता है। हालांकि, आप लोडर के प्रकार को आसानी से बदल सकते हैं।

```python
<!--IMPORTS:[{"imported": "TextLoader", "source": "langchain_community.document_loaders", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.text.TextLoader.html", "title": "File Directory"}]-->
from langchain_community.document_loaders import TextLoader
```

```python
loader = DirectoryLoader('../', glob="**/*.md", loader_cls=TextLoader)
```

```python
docs = loader.load()
```

```python
len(docs)
```

```output
    1
```

यदि आप पायथन स्रोत कोड फ़ाइलें लोड करना चाहते हैं, तो `PythonLoader` का उपयोग करें।

```python
<!--IMPORTS:[{"imported": "PythonLoader", "source": "langchain_community.document_loaders", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.python.PythonLoader.html", "title": "File Directory"}]-->
from langchain_community.document_loaders import PythonLoader
```

```python
loader = DirectoryLoader('../../../../../', glob="**/*.py", loader_cls=PythonLoader)
```

```python
docs = loader.load()
```

```python
len(docs)
```

```output
    691
```

## TextLoader के साथ फ़ाइल एन्कोडिंग का स्वत: पता लगाएं

इस उदाहरण में हम कुछ ऐसी रणनीतियों को देखेंगे जो एक निर्देशिका से कई मनमाने फ़ाइलों को लोड करते समय `TextLoader` क्लास का उपयोग करने में उपयोगी हो सकती हैं।

समस्या को दर्शाने के लिए, पहले आइए कई पाठों को मनमाने एन्कोडिंग के साथ लोड करने का प्रयास करें।

```python
path = '../../../../../tests/integration_tests/examples'
loader = DirectoryLoader(path, glob="**/*.txt", loader_cls=TextLoader)
```

### B. मौन विफलता

हम `DirectoryLoader` को `silent_errors` पैरामीटर पास कर सकते हैं ताकि वह वे फ़ाइलें छोड़ दे जिन्हें लोड नहीं किया जा सका और लोड प्रक्रिया जारी रख सके।

```python
loader = DirectoryLoader(path, glob="**/*.txt", loader_cls=TextLoader, silent_errors=True)
docs = loader.load()
```

```output
    Error loading ../../../../../tests/integration_tests/examples/example-non-utf8.txt
```

```python
doc_sources = [doc.metadata['source']  for doc in docs]
doc_sources
```

```output
     '../../../../../tests/integration_tests/examples/example-utf8.txt']
```

### C. एन्कोडिंग का स्वत: पता लगाना

हम `TextLoader` से फ़ाइल एन्कोडिंग का स्वत: पता लगवाने के लिए कह सकते हैं, `autodetect_encoding` पैरामीटर को लोडर क्लास में पास करके।

```python
text_loader_kwargs={'autodetect_encoding': True}
loader = DirectoryLoader(path, glob="**/*.txt", loader_cls=TextLoader, loader_kwargs=text_loader_kwargs)
docs = loader.load()
```
