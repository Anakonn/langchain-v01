# वेक्टर स्टोर

:::info
[एकीकरण](/docs/integrations/vectorstores/) पर दस्तावेज़ के लिए 3rd-पार्टी वेक्टर स्टोर के साथ बनाए गए एकीकरणों पर जाएं।
:::

अनरूपित डेटा को संग्रहीत और खोजने का सबसे आम तरीका यह है कि इसे एम्बेड किया जाए और प्राप्त एम्बेडिंग वेक्टर को संग्रहीत किया जाए, और फिर क्वेरी समय पर अनरूपित क्वेरी को एम्बेड किया जाए और उन एम्बेडिंग वेक्टर को पुनः प्राप्त किया जाए जो एम्बेड क्वेरी के 'सबसे समान' हैं। एक वेक्टर स्टोर आपके लिए एम्बेड किए गए डेटा को संग्रहीत करने और वेक्टर खोज करने का काम करता है।

![वेक्टर स्टोर की प्रक्रिया को दर्शाने वाला आरेख: 1. स्रोत डेटा लोड करें, 2. वेक्टर स्टोर क्वेरी करें, 3. 'सबसे समान' परिणाम पुनः प्राप्त करें।](/img/vector_stores.jpg "वेक्टर स्टोर प्रक्रिया आरेख")

## शुरू करें

यह वॉकथ्रू वेक्टर स्टोर से संबंधित मूलभूत कार्यक्षमता को प्रदर्शित करता है। वेक्टर स्टोर के साथ काम करने का एक महत्वपूर्ण हिस्सा वेक्टर बनाना है, जिसे आमतौर पर एम्बेडिंग के माध्यम से बनाया जाता है। इसलिए, इसमें गहराई से जाने से पहले [पाठ एम्बेडिंग मॉडल](/docs/modules/data_connection/text_embedding/) इंटरफ़ेस से परिचित होना अनुशंसित है।

import Tabs from '@theme/Tabs';

### वेक्टर द्वारा समानता खोज

वेक्टर के एक पैरामीटर के रूप में एक एम्बेडिंग वेक्टर का उपयोग करके दस्तावेजों की समानता खोज करना भी संभव है `similarity_search_by_vector`।

```python
embedding_vector = OpenAIEmbeddings().embed_query(query)
docs = db.similarity_search_by_vector(embedding_vector)
print(docs[0].page_content)
```

## असिंक्रोनस ऑपरेशन

वेक्टर स्टोर आमतौर पर एक अलग सेवा के रूप में चलते हैं जिसमें कुछ आईओ ऑपरेशन की आवश्यकता होती है, और इसलिए उन्हें असिंक्रोनस रूप से कहा जा सकता है। इससे प्रदर्शन लाभ होता है क्योंकि आप बाहरी सेवाओं से प्रतिक्रिया प्राप्त करने का समय नहीं गंवाते। यह तब भी महत्वपूर्ण हो सकता है यदि आप एक असिंक्रोनस फ्रेमवर्क, जैसे [FastAPI](https://fastapi.tiangolo.com/) के साथ काम करते हैं।

LangChain वेक्टर स्टोर पर असिंक्रोनस ऑपरेशन का समर्थन करता है। सभी विधियों को उनके असिंक्रोनस समकक्षों का उपयोग करके कहा जा सकता है, `a` उपसर्ग के साथ, जिसका अर्थ है `async`।

`Qdrant` एक वेक्टर स्टोर है, जो सभी असिंक्रोनस ऑपरेशन का समर्थन करता है, इसलिए इसका उपयोग इस वॉकथ्रू में किया जाएगा।

```bash
pip install qdrant-client
```

## अधिकतम सीमांत प्रासंगिकता खोज (MMR)

अधिकतम सीमांत प्रासंगिकता क्वेरी के प्रति समानता **और** चयनित दस्तावेजों में विविधता के लिए अनुकूलित होती है। यह भी असिंक्रोनस एपीआई में समर्थित है।

```python
query = "What did the president say about Ketanji Brown Jackson"
found_docs = await qdrant.amax_marginal_relevance_search(query, k=2, fetch_k=10)
for i, doc in enumerate(found_docs):
    print(f"{i + 1}.", doc.page_content, "\n")
```
