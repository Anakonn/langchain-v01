# प्रश्न

भाषा मॉडल के लिए एक प्रश्न उपयोगकर्ता द्वारा प्रदान किया गया निर्देशों या इनपुट का एक सेट है जो मॉडल के प्रतिक्रिया को मार्गदर्शन करता है, इसे संदर्भ को समझने और प्रासंगिक और सुसंगत भाषा-आधारित आउटपुट जैसे प्रश्नों का उत्तर देना, वाक्य पूरा करना या एक वार्तालाप में शामिल होना में मदद करता है।

## [त्वरित शुरुआत](./quick_start)

यह [त्वरित शुरुआत](./quick_start) प्रश्नों के साथ काम करने का एक मूलभूत अवलोकन प्रदान करता है।
