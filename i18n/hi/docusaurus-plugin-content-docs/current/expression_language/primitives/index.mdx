# प्राथमिक

LCEL के साथ उपयोग करने योग्य विभिन्न [घटकों](/docs/modules) के अलावा, LangChain में डेटा को आगे-पीछे करने और प्रारूपित करने, तर्क बाइंड करने, कस्टम लॉजिक को इनवोक करने और अधिक के लिए विभिन्न प्राथमिक भी शामिल हैं।

यह खंड इन घटकों में से कुछ के उपयोग और उपयोगिता पर अधिक गहराई से जाता है।
