# Preguntas frecuentes

## Pull Requests (PRs)

### ¿Cómo puedo permitir que los mantenedores editen mi PR?

Cuando envías una solicitud de extracción, puede ser necesario realizar cambios adicionales antes de fusionarla. A menudo, es más eficiente que los mantenedores realicen estos cambios ellos mismos antes de la fusión, en lugar de pedirte que lo hagas durante la revisión del código.

De forma predeterminada, la mayoría de las solicitudes de extracción tendrán una insignia `✅ Los mantenedores tienen permiso para editar esta solicitud de extracción.` en la barra lateral derecha.

Si no ves esta insignia, es posible que hayas desactivado esta configuración en el fork desde el que estás solicitando la extracción. Consulta [esta página de documentación de Github](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/allowing-changes-to-a-pull-request-branch-created-from-a-fork) para obtener más información.

Cabe destacar que Github no permite habilitar esta configuración para forks en **organizaciones** ([issue](https://github.com/orgs/community/discussions/5634))). Si estás trabajando en una organización, te recomendamos enviar tu PR desde un fork personal para poder habilitar esta configuración.
