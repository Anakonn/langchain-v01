# Extendiendo LangChain

Se recomienda extender las abstracciones base de LangChain, ya sea que esté planeando contribuir al repositorio de código abierto o construir una integración interna a medida.

Consulte estas guías para construir sus propias clases personalizadas para los siguientes módulos:

- [Modelos de chat](/docs/modules/model_io/chat/custom_chat_model) para interactuar con modelos de lenguaje adaptados a chat.
- [LLMs](/docs/modules/model_io/llms/custom_llm) para interactuar con modelos de lenguaje de texto.
- [Analizadores de salida](/docs/modules/model_io/output_parsers/custom) para manejar las salidas de los modelos de lenguaje.
