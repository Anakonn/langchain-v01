# Directorio de archivos

Este documento cubre cómo cargar todos los documentos en un directorio.

Internamente, por defecto, esto usa el [UnstructuredLoader](/docs/integrations/document_loaders/unstructured_file).

```python
<!--IMPORTS:[{"imported": "DirectoryLoader", "source": "langchain_community.document_loaders", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.directory.DirectoryLoader.html", "title": "File Directory"}]-->
from langchain_community.document_loaders import DirectoryLoader
```

Podemos usar el parámetro `glob` para controlar qué archivos cargar. Tenga en cuenta que aquí no carga el archivo `.rst` ni los archivos `.html`.

```python
loader = DirectoryLoader('../', glob="**/*.md")
```

```python
docs = loader.load()
```

```python
len(docs)
```

```output
    1
```

## Mostrar una barra de progreso

De forma predeterminada, no se mostrará una barra de progreso. Para mostrar una barra de progreso, instale la biblioteca `tqdm` (por ejemplo, `pip install tqdm`) y establezca el parámetro `show_progress` en `True`.

```python
loader = DirectoryLoader('../', glob="**/*.md", show_progress=True)
docs = loader.load()
```

```output
    Requirement already satisfied: tqdm in /Users/jon/.pyenv/versions/3.9.16/envs/microbiome-app/lib/python3.9/site-packages (4.65.0)


    0it [00:00, ?it/s]
```

## Usar subprocesos múltiples

De forma predeterminada, la carga se realiza en un solo subproceso. Para utilizar varios subprocesos, establezca el indicador `use_multithreading` en verdadero.

```python
loader = DirectoryLoader('../', glob="**/*.md", use_multithreading=True)
docs = loader.load()
```

## Cambiar la clase de cargador

De forma predeterminada, se utiliza la clase `UnstructuredLoader`. Sin embargo, puede cambiar fácilmente el tipo de cargador.

```python
<!--IMPORTS:[{"imported": "TextLoader", "source": "langchain_community.document_loaders", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.text.TextLoader.html", "title": "File Directory"}]-->
from langchain_community.document_loaders import TextLoader
```

```python
loader = DirectoryLoader('../', glob="**/*.md", loader_cls=TextLoader)
```

```python
docs = loader.load()
```

```python
len(docs)
```

```output
    1
```

Si necesita cargar archivos de código fuente de Python, use `PythonLoader`.

```python
<!--IMPORTS:[{"imported": "PythonLoader", "source": "langchain_community.document_loaders", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.python.PythonLoader.html", "title": "File Directory"}]-->
from langchain_community.document_loaders import PythonLoader
```

```python
loader = DirectoryLoader('../../../../../', glob="**/*.py", loader_cls=PythonLoader)
```

```python
docs = loader.load()
```

```python
len(docs)
```

```output
    691
```

El archivo `example-non-utf8.txt` usa una codificación diferente, por lo que la función `load()` falla con un mensaje útil que indica qué archivo no se pudo decodificar.

Con el comportamiento predeterminado de `TextLoader`, cualquier error al cargar alguno de los documentos hará que falle todo el proceso de carga y no se cargará ningún documento.

### B. Fallo silencioso

Podemos pasar el parámetro `silent_errors` al `DirectoryLoader` para omitir los archivos que no se pudieron cargar y continuar el proceso de carga.

```python
loader = DirectoryLoader(path, glob="**/*.txt", loader_cls=TextLoader, silent_errors=True)
docs = loader.load()
```

```output
    Error loading ../../../../../tests/integration_tests/examples/example-non-utf8.txt
```

```python
doc_sources = [doc.metadata['source']  for doc in docs]
doc_sources
```

```output
    ['../../../../../tests/integration_tests/examples/whatsapp_chat.txt',
     '../../../../../tests/integration_tests/examples/example-utf8.txt']
```

### C. Detección automática de codificaciones

También podemos pedirle a `TextLoader` que detecte automáticamente la codificación del archivo antes de fallar, pasando `autodetect_encoding` a la clase del cargador.

```python
text_loader_kwargs={'autodetect_encoding': True}
loader = DirectoryLoader(path, glob="**/*.txt", loader_cls=TextLoader, loader_kwargs=text_loader_kwargs)
docs = loader.load()
```

```python
doc_sources = [doc.metadata['source']  for doc in docs]
doc_sources
```

```output
    ['../../../../../tests/integration_tests/examples/example-non-utf8.txt',
     '../../../../../tests/integration_tests/examples/whatsapp_chat.txt',
     '../../../../../tests/integration_tests/examples/example-utf8.txt']
```
