# LLMs

Los Modelos de Lenguaje de Gran Tamaño (LLMs) son un componente fundamental de LangChain.
LangChain no proporciona sus propios LLMs, sino que más bien ofrece una interfaz estándar para interactuar con una gran variedad de LLMs diferentes. Para ser más específicos, esta interfaz toma como entrada una cadena de texto y devuelve una cadena de texto.

Hay muchos proveedores de LLM (OpenAI, Cohere, Hugging Face, etc.) - la clase `LLM` está diseñada para proporcionar una interfaz estándar para todos ellos.
