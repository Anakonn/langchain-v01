# Indicaciones

Una indicación para un modelo de lenguaje es un conjunto de instrucciones o entrada proporcionada por un usuario para
guiar la respuesta del modelo, ayudándolo a entender el contexto y generar un lenguaje relevante y coherente
y salida basada en el lenguaje, como responder preguntas, completar oraciones
o participar en una conversación.
