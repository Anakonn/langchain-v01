# Casos de uso

Esta sección contiene recorridos y técnicas para tareas de uso final comunes.

Si está buscando construir algo específico o es más un aprendiz práctico, ¡pruebe uno!
Si bien hacen referencia a bloques de construcción que se explican con más detalle en otras secciones, alentamos
a las personas a comenzar a recorrerlos y descomponer el código en un contexto del mundo real.

O, si prefiere mirar los fundamentos primero, puede consultar las secciones sobre [Expression Language](/docs/expression_language/get_started)
y los diversos [componentes](/docs/modules) que LangChain proporciona para obtener más conocimientos básicos.
