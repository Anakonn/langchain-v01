# Primitivos

Además de varios [componentes](/docs/modules) que se pueden usar con LCEL, LangChain también incluye varios primitivos
que ayudan a pasar y dar formato a los datos, vincular argumentos, invocar lógica personalizada y más.

Esta sección profundiza más en dónde y cómo algunos de estos componentes son útiles.
