# Robocorp

>[Robocorp](https://robocorp.com/) ayuda a construir y operar trabajadores de Python que se ejecutan sin problemas en cualquier lugar y a cualquier escala

## Instalación y configuración

Necesitas instalar el paquete de python `langchain-robocorp`:

```bash
pip install langchain-robocorp
```

Necesitarás una instancia en ejecución del Action Server para comunicarte con ella desde tu aplicación de agente. Consulta la [Guía de inicio rápido de Robocorp](https://github.com/robocorp/robocorp#quickstart) sobre cómo configurar Action Server y crear tus Acciones.

Puedes inicializar un nuevo proyecto usando el comando `new` de Action Server.

```bash
action-server new
cd ./your-project-name
action-server start
```
