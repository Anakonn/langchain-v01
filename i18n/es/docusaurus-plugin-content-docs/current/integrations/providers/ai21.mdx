# AI21 Labs

>[AI21 Labs](https://www.ai21.com/about) es una empresa especializada en Procesamiento de Lenguaje Natural (NLP), que desarrolla sistemas de IA que pueden entender y generar lenguaje natural.

Esta página cubre cómo usar el ecosistema `AI21` dentro de `LangChain`.

## Instalación y configuración

- Obtén una clave de API de AI21 y configúrala como una variable de entorno (`AI21_API_KEY`)
- Instala el paquete de Python:

```bash
pip install langchain-ai21
```
