# Vespa

>[Vespa](https://vespa.ai/) es un motor de búsqueda y base de datos de vectores completamente equipado.
> Admite búsqueda de vectores (ANN), búsqueda léxica y búsqueda en datos estructurados, todo en la misma consulta.

## Instalación y configuración

```bash
pip install pyvespa
```
