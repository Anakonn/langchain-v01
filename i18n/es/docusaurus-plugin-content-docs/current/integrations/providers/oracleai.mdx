# Búsqueda de vectores de Oracle AI

La búsqueda de vectores de Oracle AI está diseñada para cargas de trabajo de Inteligencia Artificial (IA) que le permiten consultar datos en función de la semántica, en lugar de palabras clave. Uno de los mayores beneficios de la búsqueda de vectores de Oracle AI es que la búsqueda semántica en datos no estructurados se puede combinar con la búsqueda relacional en datos empresariales en un solo sistema.
Esto no solo es poderoso, sino también significativamente más efectivo porque no necesita agregar una base de datos de vectores especializada, eliminando el dolor de la fragmentación de datos entre múltiples sistemas.

Además, dado que Oracle ha estado construyendo tecnologías de base de datos durante tanto tiempo, sus vectores pueden beneficiarse de las características más poderosas de Oracle Database, como las siguientes:

 * Soporte de particionamiento
 * Escalabilidad de Real Application Clusters
 * Escaneos inteligentes de Exadata
 * Procesamiento de fragmentación en bases de datos distribuidas geográficamente
 * Transacciones
 * SQL paralelo
 * Recuperación ante desastres
 * Seguridad
 * Oracle Machine Learning
 * Oracle Graph Database
 * Oracle Spatial and Graph
 * Oracle Blockchain
 * JSON
