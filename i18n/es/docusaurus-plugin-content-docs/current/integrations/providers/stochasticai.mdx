# StochasticAI

Esta página cubre cómo usar el ecosistema StochasticAI dentro de LangChain.
Se divide en dos partes: instalación y configuración, y luego referencias a envoltorios específicos de StochasticAI.

## Instalación y configuración

- Instalar con `pip install stochasticx`
- Obtener una clave de API de StochasticAI y establecerla como una variable de entorno (`STOCHASTICAI_API_KEY`)

## Envoltorios

### LLM

Existe un envoltorio LLM de StochasticAI, al que se puede acceder con

```python
<!--IMPORTS:[{"imported": "StochasticAI", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.stochasticai.StochasticAI.html", "title": "StochasticAI"}]-->
```
