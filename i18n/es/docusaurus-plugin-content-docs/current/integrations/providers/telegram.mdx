# Telegram

>[Telegram Messenger](https://web.telegram.org/a/) es un servicio de mensajería instantánea gratuito, multiplataforma, cifrado, basado en la nube y centralizado. La aplicación también proporciona chats y videollamadas opcionales con cifrado de extremo a extremo, VoIP, intercambio de archivos y varias otras funciones.
