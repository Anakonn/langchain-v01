# CnosDB

> [CnosDB](https://github.com/cnosdb/cnosdb) es una base de datos de series temporales distribuida de código abierto con alto rendimiento, alta tasa de compresión y alta facilidad de uso.

## Instalación y configuración

```python
pip install cnos-connector
```

## Conectarse a CnosDB

Puede conectarse a CnosDB usando el método `SQLDatabase.from_cnosdb()`.

### Sintaxis

```python
def SQLDatabase.from_cnosdb(url: str = "127.0.0.1:8902",
                              user: str = "root",
                              password: str = "",
                              tenant: str = "cnosdb",
                              database: str = "public")
```

Argumentos:
1. url (str): El nombre de host y el número de puerto de la conexión HTTP del servicio CnosDB, excluyendo "http://" o "https://", con un valor predeterminado de "127.0.0.1:8902".
2. user (str): El nombre de usuario utilizado para conectarse al servicio CnosDB, con un valor predeterminado de "root".
3. password (str): La contraseña del usuario que se conecta al servicio CnosDB, con un valor predeterminado de "".
4. tenant (str): El nombre del inquilino utilizado para conectarse al servicio CnosDB, con un valor predeterminado de "cnosdb".
5. database (str): El nombre de la base de datos en el inquilino CnosDB.

Este ejemplo demuestra el uso de la Cadena SQL para responder a una pregunta sobre un CnosDB.

```python
from langchain_community.utilities import SQLDatabaseChain

db_chain = SQLDatabaseChain.from_llm(llm, db, verbose=True)

db_chain.run(
    "What is the average temperature of air at station XiaoMaiDao between October 19, 2022 and Occtober 20, 2022?"
)
```
