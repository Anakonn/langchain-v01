# Grobid

GROBID es una biblioteca de aprendizaje automático para extraer, analizar y reestructurar documentos sin procesar.

Está diseñado y se espera que se use para analizar artículos académicos, donde funciona particularmente bien.

*Nota*: si los artículos suministrados a Grobid son documentos grandes (por ejemplo, disertaciones) que exceden cierto número
de elementos, es posible que no se procesen.

Esta página cubre cómo usar Grobid para analizar artículos para LangChain.

## Instalación

La instalación de grobid se describe en detalle en https://grobid.readthedocs.io/en/latest/Install-Grobid/.
Sin embargo, probablemente sea más fácil y menos problemático ejecutar grobid a través de un contenedor docker,
como se documenta [aquí](https://grobid.readthedocs.io/en/latest/Grobid-docker/).

## Usar Grobid con LangChain

Una vez que grobid esté instalado y en funcionamiento (puede verificarlo accediendo a http://localhost:8070),
estará listo para comenzar.

Ahora puede usar el GrobidParser para producir documentos

```python
<!--IMPORTS:[{"imported": "GrobidParser", "source": "langchain_community.document_loaders.parsers", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.parsers.grobid.GrobidParser.html", "title": "Grobid"}, {"imported": "GenericLoader", "source": "langchain_community.document_loaders.generic", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.generic.GenericLoader.html", "title": "Grobid"}]-->
from langchain_community.document_loaders.parsers import GrobidParser
from langchain_community.document_loaders.generic import GenericLoader

#Produce chunks from article paragraphs
loader = GenericLoader.from_filesystem(
    "/Users/31treehaus/Desktop/Papers/",
    glob="*",
    suffixes=[".pdf"],
    parser= GrobidParser(segment_sentences=False)
)
docs = loader.load()

#Produce chunks from article sentences
loader = GenericLoader.from_filesystem(
    "/Users/31treehaus/Desktop/Papers/",
    glob="*",
    suffixes=[".pdf"],
    parser= GrobidParser(segment_sentences=True)
)
docs = loader.load()
```

Los metadatos de los fragmentos incluirán cuadros delimitadores. Aunque son un poco raros de analizar,
se explican en https://grobid.readthedocs.io/en/latest/Coordinates-in-PDF/
