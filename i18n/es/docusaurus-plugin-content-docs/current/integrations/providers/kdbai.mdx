# KDB.AI

>[KDB.AI](https://kdb.ai) es una poderosa base de datos de vectores basada en conocimiento y un motor de búsqueda que le permite construir aplicaciones de IA escalables y confiables, utilizando datos en tiempo real, al proporcionar búsqueda avanzada, recomendación y personalización.

## Instalación y configuración

Instale el SDK de Python:

```bash
pip install kdbai-client
```

## Tienda de vectores

Existe un wrapper alrededor de los índices de KDB.AI, lo que le permite usarlo como un vectorstore,
ya sea para búsqueda semántica o selección de ejemplos.
