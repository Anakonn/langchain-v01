# Incrustación de Postgres

> [pg_embedding](https://github.com/neondatabase/pg_embedding) es un paquete de código abierto para
> búsqueda de similitud de vectores usando `Postgres` y el `Algoritmo Jerárquico de Mundos Pequeños Navegables`
> para la búsqueda aproximada de vecinos más cercanos.

## Instalación y Configuración

Necesitamos instalar varios paquetes de python.

```bash
pip install psycopg2-binary
```
