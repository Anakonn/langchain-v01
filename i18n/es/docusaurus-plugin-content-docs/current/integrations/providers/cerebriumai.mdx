# CerebriumAI

>[Cerebrium](https://docs.cerebrium.ai/cerebrium/getting-started/introduction) es un proveedor de infraestructura GPU sin servidor.
> Proporciona acceso a la API a varios modelos LLM.

Consulta los ejemplos en la [documentación de CerebriumAI](https://docs.cerebrium.ai/examples/langchain).

## Instalación y configuración

- Instala un paquete de python:

```bash
pip install cerebrium
```

- [Obtén una clave de API de CerebriumAI](https://docs.cerebrium.ai/cerebrium/getting-started/installation) y configúrala
  como una variable de entorno (`CEREBRIUMAI_API_KEY`)
