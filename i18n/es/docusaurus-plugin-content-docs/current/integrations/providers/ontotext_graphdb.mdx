# Ontotext GraphDB

>[Ontotext GraphDB](https://graphdb.ontotext.com/) es una base de datos de gráficos y una herramienta de descubrimiento de conocimiento que cumple con RDF y SPARQL.

## Dependencias

Instala el paquete [rdflib](https://github.com/RDFLib/rdflib) con

```bash
pip install rdflib==7.0.0
```
