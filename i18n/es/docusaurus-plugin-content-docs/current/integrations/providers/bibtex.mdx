# BibTeX

>[BibTeX](https://www.ctan.org/pkg/bibtex) es un formato de archivo y un sistema de gestión de referencias comúnmente utilizado junto con el sistema de composición tipográfica `LaTeX`. Sirve como una forma de organizar y almacenar información bibliográfica para documentos académicos e investigación.

## Instalación y configuración

Tenemos que instalar los paquetes `bibtexparser` y `pymupdf`.

```bash
pip install bibtexparser pymupdf
```
