# Geopandas

>[GeoPandas](https://geopandas.org/) es un proyecto de código abierto para facilitar el trabajo
> con datos geoespaciales en python. `GeoPandas` extiende los tipos de datos utilizados por
> `pandas` para permitir operaciones espaciales en tipos geométricos.
> Las operaciones geométricas se realizan mediante `shapely`.

## Instalación y configuración

Tenemos que instalar varios paquetes de python.

```bash
pip install -U sodapy pandas geopandas
```
