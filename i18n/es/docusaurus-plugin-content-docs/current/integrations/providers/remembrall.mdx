# Remembrall

>[Remembrall](https://remembrall.dev/) es una plataforma que le da a un modelo de lenguaje
> memoria a largo plazo, generación con recuperación aumentada y observabilidad completa.

## Instalación y configuración

Para comenzar, [inicia sesión con Github en la plataforma Remembrall](https://remembrall.dev/login)
y copia tu [clave API de la página de configuración](https://remembrall.dev/dashboard/settings).
