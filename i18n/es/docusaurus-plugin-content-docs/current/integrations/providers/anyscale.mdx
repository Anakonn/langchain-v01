# Anyscale

>[Anyscale](https://www.anyscale.com) es una plataforma para ejecutar, ajustar y escalar LLM a través de APIs listas para producción.
> [Anyscale Endpoints](https://docs.anyscale.com/endpoints/overview) sirven muchos modelos de código abierto de una manera rentable.

`Anyscale` también proporciona [un ejemplo](https://docs.anyscale.com/endpoints/model-serving/examples/langchain-integration)
de cómo configurar `LangChain` con `Anyscale` para agentes de chat avanzados.

## Instalación y configuración

- Obtén una URL de servicio de Anyscale, una ruta y una clave API, y configúralas como variables de entorno (`ANYSCALE_SERVICE_URL`,`ANYSCALE_SERVICE_ROUTE`, `ANYSCALE_SERVICE_TOKEN`).
- Consulta [la documentación de Anyscale](https://www.anyscale.com/get-started) para obtener más detalles.

Tenemos que instalar el paquete `openai`:

```bash
pip install openai
```
