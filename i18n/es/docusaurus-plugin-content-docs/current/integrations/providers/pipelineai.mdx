# PipelineAI

Esta página cubre cómo usar el ecosistema de PipelineAI dentro de LangChain.
Se divide en dos partes: instalación y configuración, y luego referencias a envolturas específicas de PipelineAI.

## Instalación y configuración

- Instalar con `pip install pipeline-ai`
- Obtener una clave de API de Pipeline Cloud y establecerla como una variable de entorno (`PIPELINE_API_KEY`)

## Envolturas

### LLM
