# Ollama

>[Ollama](https://ollama.ai/) es una biblioteca de Python. Le permite ejecutar modelos de lenguaje de código abierto a gran escala,
> como LLaMA2, de forma local.
>
>`Ollama` agrupa los pesos del modelo, la configuración y los datos en un solo paquete, definido por un archivo Modelfile.
>Optimiza los detalles de configuración y configuración, incluido el uso de GPU.
>Para obtener una lista completa de los modelos y variantes de modelos compatibles, consulte la [biblioteca de modelos Ollama](https://ollama.ai/library).

Consulte [esta guía](/docs/guides/development/local_llms#quickstart) para obtener más detalles
sobre cómo usar `Ollama` con LangChain.

## Instalación y configuración

Siga [estas instrucciones](https://github.com/jmorganca/ollama?tab=readme-ov-file#ollama)
para configurar y ejecutar una instancia local de Ollama.
Para usar, debe configurar las variables de entorno `ANYSCALE_API_BASE` y
`ANYSCALE_API_KEY`.
