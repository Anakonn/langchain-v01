# Streamlit

>[Streamlit](https://streamlit.io/) es una forma más rápida de crear y compartir aplicaciones de datos.
>`Streamlit` convierte scripts de datos en aplicaciones web compartibles en minutos. Todo en Python puro. No se requiere experiencia en front‑end.
>Consulta más ejemplos en [streamlit.io/generative-ai](https://streamlit.io/generative-ai).

## Instalación y configuración

Necesitamos instalar el paquete Python `streamlit`:

```bash
pip install streamlit
```
