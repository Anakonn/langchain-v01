# Apache Doris

>[Apache Doris](https://doris.apache.org/) es un almacén de datos moderno para análisis en tiempo real.
Ofrece análisis ultrarrápidos en datos en tiempo real a escala.

>Generalmente, `Apache Doris` se clasifica como OLAP, y ha demostrado un excelente rendimiento
> en [ClickBench — una Benchmark para DBMS Analíticos](https://benchmark.clickhouse.com/).
> Dado que tiene un motor de ejecución vectorizado súper rápido, también se puede usar como un vectordb rápido.

## Instalación y Configuración

```bash
pip install pymysql
```
