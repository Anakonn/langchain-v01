# Huawei

>[Huawei Technologies Co., Ltd.](https://www.huawei.com/) es una corporación tecnológica de comunicaciones digitales multinacional china.
>
>[Huawei Cloud](https://www.huaweicloud.com/intl/en-us/product/) ofrece un conjunto integral de servicios globales de computación en la nube.

## Instalación y configuración

Para acceder a `Huawei Cloud`, necesitas un token de acceso.

También tienes que instalar una biblioteca de Python:

```bash
pip install -U esdk-obs-python
```
