# GitHub

>[GitHub](https://github.com/) es una plataforma de desarrolladores que permite a los desarrolladores crear,
> almacenar, administrar y compartir su código. Utiliza el software `Git`, proporcionando el
> control de versiones distribuido de Git más control de acceso, seguimiento de errores,
> solicitudes de funciones de software, gestión de tareas, integración continua y wikis para cada proyecto.

## Instalación y configuración

Para acceder a la API de GitHub, necesitas un [token de acceso personal](https://github.com/settings/tokens).
