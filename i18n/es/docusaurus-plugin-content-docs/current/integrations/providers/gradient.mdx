# Gradiente

>[Gradiente](https://gradient.ai/) permite ajustar y obtener completaciones en LLM con una API web sencilla.

## Instalación y configuración

- Instala el SDK de Python:

```bash
pip install gradientai
```

Obtén un [token de acceso y un espacio de trabajo de Gradiente](https://gradient.ai/) y configúralos como variables de entorno (`Gradient_ACCESS_TOKEN`) y (`GRADIENT_WORKSPACE_ID`)
