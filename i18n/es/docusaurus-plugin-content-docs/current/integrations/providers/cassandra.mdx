# Cassandra

> [Apache Cassandra®](https://cassandra.apache.org/) es una base de datos NoSQL, orientada a filas, altamente escalable y altamente disponible.
> A partir de la versión 5.0, la base de datos se envía con [capacidades de búsqueda vectorial](https://cassandra.apache.org/doc/trunk/cassandra/vector-search/overview.html).

Las integraciones descritas en esta página se pueden usar con `Cassandra` así como con otras bases de datos compatibles con CQL, es decir, aquellas que utilizan el protocolo `Cassandra Query Language`.

## Instalación y configuración

Instala el siguiente paquete de Python:

```bash
pip install "cassio>=0.1.6"
```
