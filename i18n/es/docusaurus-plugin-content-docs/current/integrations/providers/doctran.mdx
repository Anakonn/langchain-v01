# Doctran

>[Doctran](https://github.com/psychic-api/doctran) es un paquete de Python. Utiliza LLM y bibliotecas de procesamiento de lenguaje natural de código abierto para transformar texto sin procesar en documentos limpios, estructurados y densos en información que están optimizados para la recuperación de espacios vectoriales. Puedes pensar en `Doctran` como una caja negra donde entran cadenas desordenadas y salen cadenas etiquetadas, limpias y agradables.

## Instalación y configuración

```bash
pip install doctran
```
