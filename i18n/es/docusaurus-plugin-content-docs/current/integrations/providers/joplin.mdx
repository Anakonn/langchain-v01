# Joplin

>[Joplin](https://joplinapp.org/) es una aplicación de toma de notas de código abierto. Captura tus pensamientos
> y los accede de forma segura desde cualquier dispositivo.

## Instalación y configuración

El `Joplin API` requiere un token de acceso.
Puedes encontrar las instrucciones de instalación [aquí](https://joplinapp.org/api/references/rest_api/).
