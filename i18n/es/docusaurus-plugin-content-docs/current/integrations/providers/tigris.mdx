# Tigris

> [Tigris](https://tigrisdata.com) es una base de datos NoSQL y una plataforma de búsqueda sin servidor de código abierto diseñada para simplificar la construcción de aplicaciones de búsqueda vectorial de alto rendimiento.
> `Tigris` elimina la complejidad de la infraestructura de administrar, operar y sincronizar múltiples herramientas, lo que le permite concentrarse en construir grandes aplicaciones.

## Instalación y configuración

```bash
pip install tigrisdb openapi-schema-pydantic
```
