# YouTube

>[YouTube](https://www.youtube.com/) es una plataforma de redes sociales y de intercambio de videos en línea de Google.
> Descargamos las transcripciones de `YouTube` y la información de los videos.

## Instalación y configuración

```bash
pip install youtube-transcript-api
pip install pytube
```
