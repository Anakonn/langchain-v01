# Pétalos

Esta página cubre cómo usar el ecosistema de Pétalos dentro de LangChain.
Se divide en dos partes: instalación y configuración, y luego referencias a envoltorios específicos de Pétalos.

## Instalación y configuración

- Instalar con `pip install petals`
- Obtener una clave de API de Hugging Face y establecerla como una variable de entorno (`HUGGINGFACE_API_KEY`)
