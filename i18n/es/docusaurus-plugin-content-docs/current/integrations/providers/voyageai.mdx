# VoyageAI

Toda la funcionalidad relacionada con VoyageAI

>[VoyageAI](https://www.voyageai.com/) Voyage AI construye modelos de incrustación, personalizados para su dominio y empresa, para una mejor calidad de recuperación.

## Instalación y configuración

Instale el paquete de integración con

```bash
pip install langchain-voyageai
```

Obtenga una clave de API de VoyageAI y configúrela como una variable de entorno (`VOYAGE_API_KEY`)
