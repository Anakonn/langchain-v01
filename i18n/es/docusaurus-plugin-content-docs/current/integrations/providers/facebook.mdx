# Facebook - Meta

>[Meta Platforms, Inc.](https://www.facebook.com/), que hace negocios como `Meta`, anteriormente
> llamada `Facebook, Inc.`, y `TheFacebook, Inc.`, es una empresa tecnológica multinacional
> conglomerada estadounidense. La empresa es propietaria y opera `Facebook`, `Instagram`, `Threads`,
> y `WhatsApp`, entre otros productos y servicios.

## Modelos de incrustación

### LASER

>[LASER](https://github.com/facebookresearch/LASER) es una biblioteca de Python desarrollada por
> el equipo de `Meta AI Research` y se utiliza para
> crear incrustaciones de oraciones multilingües para
> [más de 147 idiomas a partir del 25/2/2024](https://github.com/facebookresearch/flores/blob/main/flores200/README.md#languages-in-flores-200)

```bash

```

## Tiendas de vectores

### Facebook Faiss

>[Facebook AI Similarity Search (Faiss)](https://engineering.fb.com/2017/03/29/data-infrastructure/faiss-a-library-for-efficient-similarity-search/)
> es una biblioteca para la búsqueda y agrupación eficiente de vectores densos similares. Contiene algoritmos que
> buscan en conjuntos de vectores de cualquier tamaño, incluso en aquellos que posiblemente no entren en la RAM. También contiene código de apoyo
> para la evaluación y el ajuste de parámetros.

[Documentación de Faiss](https://faiss.ai/).

Necesitamos instalar el paquete de Python `faiss`.

```bash
pip install faiss-gpu # For CUDA 7.5+ supported GPU's.
```

O

```bash
pip install faiss-cpu # For CPU Installation
```
