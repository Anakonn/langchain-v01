# Eden AI

>[Eden AI](https://docs.edenai.co/docs/getting-started-with-eden-ai) interfaz de usuario (UI)
> está diseñada para manejar proyectos de IA. Con `Eden AI Portal`,
> puede realizar IA sin código utilizando los mejores motores del mercado.

## Instalación y configuración

Acceder a la API de Eden AI requiere una clave API, que puede obtener
[creando una cuenta](https://app.edenai.run/user/register) y
dirigiéndose [aquí](https://app.edenai.run/admin/account/settings).

## Herramientas

Eden AI proporciona una lista de herramientas que otorgan a su Agente la capacidad de realizar múltiples tareas, como:
* reconocimiento de voz a texto
* texto a voz
* detección de contenido explícito en texto
* detección de contenido explícito en imágenes
* detección de objetos
* análisis de facturas de OCR
* análisis de identificación de OCR
