# DeepInfra

>[DeepInfra](https://deepinfra.com/docs) nos permite ejecutar los
> [últimos modelos de aprendizaje automático](https://deepinfra.com/models) con facilidad.
> DeepInfra se encarga de todo el trabajo pesado relacionado con la ejecución, el escalado y el monitoreo
> de los modelos. Los usuarios pueden centrarse en su aplicación e integrar los modelos con simples llamadas a la API REST.

>DeepInfra proporciona [ejemplos](https://deepinfra.com/docs/advanced/langchain) de integración con LangChain.

Esta página cubre cómo usar el ecosistema `DeepInfra` dentro de `LangChain`.
Se divide en dos partes: instalación y configuración, y luego referencias a envoltorios DeepInfra específicos.

## Instalación y configuración

- Obtenga su clave de API DeepInfra de este enlace [aquí](https://deepinfra.com/).
- Obtenga una clave de API DeepInfra y configúrela como una variable de entorno (`DEEPINFRA_API_TOKEN`)

## Modelos disponibles

DeepInfra proporciona una gama de LLM de código abierto listos para su implementación.

Puede ver los modelos compatibles para
[generación de texto](https://deepinfra.com/models?type=text-generation) y
[incrustaciones](https://deepinfra.com/models?type=embeddings).

Puede ver una [lista de parámetros de solicitud y respuesta](https://deepinfra.com/meta-llama/Llama-2-70b-chat-hf/api).

Los modelos de chat [siguen la API openai](https://deepinfra.com/meta-llama/Llama-2-70b-chat-hf/api?example=openai-http)
