# Sopa Hermosa

>[Sopa Hermosa](https://www.crummy.com/software/BeautifulSoup/) es un paquete de Python para analizar
> documentos HTML y XML (incluyendo tener marcado malformado, es decir, etiquetas no cerradas, con el nombre de sopa de etiquetas).
> Crea un árbol de análisis para las páginas analizadas que se pueden usar para extraer datos de HTML,[3] lo cual
> es útil para el raspado web.

## Instalación y configuración

```bash
pip install beautifulsoup4
```
