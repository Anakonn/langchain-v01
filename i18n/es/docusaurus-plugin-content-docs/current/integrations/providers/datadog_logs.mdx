# Registros de Datadog

>[Datadog](https://www.datadoghq.com/) es una plataforma de monitoreo y análisis para aplicaciones a escala de nube.

## Instalación y configuración

```bash
pip install datadog_api_client
```

Debemos inicializar el cargador con la clave API y la clave APP de Datadog, y necesitamos configurar la consulta para extraer los registros deseados.
