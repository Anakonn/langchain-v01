# Escritor

Esta página cubre cómo usar el ecosistema de Escritor dentro de LangChain.
Se divide en dos partes: instalación y configuración, y luego referencias a envolturas específicas de Escritor.

## Instalación y configuración

- Obtén una clave API de Escritor y configúrala como una variable de entorno (`WRITER_API_KEY`)
