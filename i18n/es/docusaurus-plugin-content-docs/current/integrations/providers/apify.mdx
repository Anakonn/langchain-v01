# Apify

>[Apify](https://apify.com) es una plataforma en la nube para el raspado web y la extracción de datos,
>que proporciona un [ecosistema](https://apify.com/store) de más de mil
>aplicaciones listas para usar llamadas *Actors* para diversos casos de uso de raspado, rastreo y extracción.

[![Apify Actors](/img/ApifyActors.png)](https://apify.com/store)

Esta integración le permite ejecutar Actors en la plataforma `Apify` y cargar sus resultados en LangChain para alimentar sus índices vectoriales
con documentos y datos de la web, por ejemplo, para generar respuestas a partir de sitios web con documentación,
blogs o bases de conocimiento.

## Instalación y configuración

- Instale el cliente de la API de Apify para Python con `pip install apify-client`
- Obtenga su [token de API de Apify](https://console.apify.com/account/integrations) y establézcalo como
  una variable de entorno (`APIFY_API_TOKEN`) o páselo a `ApifyWrapper` como `apify_api_token` en el constructor.
