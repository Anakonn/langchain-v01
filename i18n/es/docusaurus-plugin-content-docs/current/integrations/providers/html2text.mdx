# HTML a texto

>[html2text](https://github.com/Alir3z4/html2text/) es un paquete de Python que convierte una página de `HTML` en texto `ASCII` limpio y fácil de leer.

El ASCII también resulta ser un `Markdown` válido (un formato de texto a HTML).

## Instalación y configuración

```bash
pip install html2text
```

## Transformador de documentos

Consulta un [ejemplo de uso](/docs/integrations/document_transformers/html2text).

```python
from langchain_community.document_loaders import Html2TextTransformer
```
