# Activeloop Deep Lake

>[Activeloop Deep Lake](https://docs.activeloop.ai/) es un lago de datos para aplicaciones de Deep Learning, lo que le permite usarlo
> como un almacén de vectores.

## ¿Por qué Deep Lake?

- Más que un simple almacén de vectores (multimodal). Más adelante, puede utilizar el conjunto de datos para ajustar sus propios modelos de LLM.
- No solo almacena incrustaciones, sino también los datos originales con control de versiones automático.
- Verdaderamente sin servidor. No requiere otro servicio y se puede utilizar con los principales proveedores de nube (`AWS S3`, `GCS`, etc.)

`Activeloop Deep Lake` admite `SelfQuery Retrieval`:
[Activeloop Deep Lake Self Query Retrieval](/docs/integrations/retrievers/self_query/activeloop_deeplake_self_query)
