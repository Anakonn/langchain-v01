# Molestar

> [Annoy](https://github.com/spotify/annoy) (`Approximate Nearest Neighbors Oh Yeah`)
> es una biblioteca de C++ con enlaces de Python para buscar puntos en el espacio que estén
> cerca de un punto de consulta dado. También crea estructuras de datos de archivo de solo lectura de gran tamaño
> que se asignan a la memoria para que muchos procesos puedan compartir los mismos datos.

## Instalación y configuración

```bash
pip install annoy
```
