# Chroma

>[Chroma](https://docs.trychroma.com/getting-started) es una base de datos para construir aplicaciones de IA con incrustaciones.

## Instalación y configuración

```bash
pip install langchain-chroma
```

## VectorStore

Existe un wrapper alrededor de las bases de datos de vectores Chroma, lo que le permite usarla como un vectorstore,
ya sea para búsqueda semántica o selección de ejemplos.

