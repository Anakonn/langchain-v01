# Replicar

Esta página cubre cómo ejecutar modelos en Replicate dentro de LangChain.

## Instalación y configuración

- Crea una cuenta [Replicate](https://replicate.com). Obtén tu clave API y configúrala como una variable de entorno (`REPLICATE_API_TOKEN`)
- Instala el [cliente python de Replicate](https://github.com/replicate/replicate-python) con `pip install replicate`

## Llamar a un modelo

Encuentra un modelo en la [página de exploración de Replicate](https://replicate.com/explore) y luego pega el nombre del modelo y la versión en este formato: `propietario-nombre/nombre-modelo:versión`

Por ejemplo, para este [modelo dolly](https://replicate.com/replicate/dolly-v2-12b), haz clic en la pestaña API. El nombre/versión del modelo sería: `"replicate/dolly-v2-12b:ef0e1aefc61f8e096ebe4db6b2bacc297daf2ef6899f0f7e001ec445893500e5"`

Solo se requiere el parámetro `model`, pero también se pueden pasar otros parámetros del modelo con el formato `input={model_param: value, ...}`

Por ejemplo, si estuviéramos ejecutando stable diffusion y quisiéramos cambiar las dimensiones de la imagen:

```python
Replicate(model="stability-ai/stable-diffusion:db21e45d3f7023abc2a46ee38a23973f6dce16bb082a930b0c49861f96d1e5bf", input={'image_dimensions': '512x512'})
```

*Tenga en cuenta que solo se devolverá la primera salida de un modelo.*
Desde aquí, podemos inicializar nuestro modelo:

```python
llm = Replicate(model="replicate/dolly-v2-12b:ef0e1aefc61f8e096ebe4db6b2bacc297daf2ef6899f0f7e001ec445893500e5")
```

Y ejecutarlo:

```python
prompt = """
Answer the following yes/no question by reasoning step by step.
Can a dog drive a car?
"""
llm(prompt)
```

Podemos llamar a cualquier modelo Replicate (no solo LLM) usando esta sintaxis. Por ejemplo, podemos llamar a [Stable Diffusion](https://replicate.com/stability-ai/stable-diffusion):

```python
text2image = Replicate(model="stability-ai/stable-diffusion:db21e45d3f7023abc2a46ee38a23973f6dce16bb082a930b0c49861f96d1e5bf", input={'image_dimensions':'512x512'})

image_output = text2image("A cat riding a motorcycle by Picasso")
```
