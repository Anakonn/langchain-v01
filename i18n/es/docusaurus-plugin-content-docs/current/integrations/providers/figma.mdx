# Figma

>[Figma](https://www.figma.com/) es una aplicación web colaborativa para el diseño de interfaces.

## Instalación y configuración

La API de Figma requiere un `token de acceso`, `node_ids` y una `clave de archivo`.

La `clave de archivo` se puede extraer de la URL. https://www.figma.com/file/{filekey}/sampleFilename

Los `Node IDs` también están disponibles en la URL. Haga clic en cualquier cosa y busque el parámetro '?node-id=\{node_id}'.

`Token de acceso` [instrucciones](https://help.figma.com/hc/en-us/articles/8085703771159-Manage-personal-access-tokens).
