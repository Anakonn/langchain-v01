# Docugami

>[Docugami](https://docugami.com) convierte documentos comerciales en un Grafo de Conocimiento de Documento XML, generando bosques
> de árboles semánticos XML que representan documentos completos. Esta es una representación rica que incluye las características semánticas y
> estructurales de varios fragmentos del documento como un árbol XML.

## Instalación y Configuración

```bash
pip install dgml-utils
pip install docugami-langchain
```

## Cargador de Documentos

Consulte un [ejemplo de uso](/docs/integrations/document_loaders/docugami).

```python
from docugami_langchain.document_loaders import DocugamiLoader
```
