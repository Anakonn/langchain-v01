# TigerGraph

>[TigerGraph](https://www.tigergraph.com/tigergraph-db/) es una base de datos de gráficos de alto rendimiento y distribuida de forma nativa.
> El almacenamiento de datos en un formato de gráfico de vértices y bordes conduce a relaciones ricas,
> ideal para fundamentar las respuestas de LLM.

## Instalación y configuración

Siga las instrucciones [cómo conectarse a la base de datos `TigerGraph`](https://docs.tigergraph.com/pytigergraph/current/getting-started/connection).

Instalar el SDK de Python:

```bash
pip install pyTigerGraph
```
