# Tesorería Moderna

>[Tesorería Moderna](https://www.moderntreasury.com/) simplifica las operaciones de pago complejas. Es una plataforma unificada para impulsar productos y procesos que mueven dinero.
>- Conéctese a bancos y sistemas de pago
>- Realice un seguimiento de las transacciones y los saldos en tiempo real
>- Automatice las operaciones de pago para escalar

## Instalación y configuración

No hay ninguna configuración especial para ello.
