# IA Confiable

>[Confident AI](https://confident-ai.com) es un creador de `DeepEval`.
>
>[DeepEval](https://github.com/confident-ai/deepeval) es un paquete para pruebas unitarias de LLM.
> Usando `DeepEval`, todos pueden construir modelos de lenguaje robustos a través de iteraciones más rápidas
> utilizando tanto pruebas unitarias como de integración. `DeepEval` brinda soporte para cada paso en la iteración
> desde la creación de datos sintéticos hasta las pruebas.

## Instalación y Configuración

Necesitas obtener las [credenciales de la API DeepEval](https://app.confident-ai.com).

Necesitas instalar el paquete Python `DeepEval`:

```bash
pip install deepeval
```
