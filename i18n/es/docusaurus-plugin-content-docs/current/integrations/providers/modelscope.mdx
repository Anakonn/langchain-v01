# ModelScope

>[ModelScope](https://www.modelscope.cn/home) es un gran repositorio de modelos y conjuntos de datos.

Esta página cubre cómo usar el ecosistema de modelscope dentro de LangChain.
Se divide en dos partes: instalación y configuración, y luego referencias a envoltorios específicos de modelscope.

## Instalación y configuración

Instala el paquete `modelscope`.

```bash
pip install modelscope
```
