# TiDB

> [TiDB Cloud](https://tidbcloud.com/) es una solución integral de base de datos como servicio (DBaaS) que ofrece opciones dedicadas y sin servidor. `TiDB Serverless` ahora está integrando una búsqueda vectorial integrada en el panorama de MySQL. Con esta mejora, puede desarrollar aplicaciones de IA utilizando `TiDB Serverless` sin necesidad de una nueva base de datos o pilas técnicas adicionales. Sé de los primeros en experimentarlo uniéndote a la [lista de espera para la versión beta privada](https://tidb.cloud/ai).

## Instalación y configuración

Debes obtener los detalles de conexión de la base de datos TiDB.
Visita [TiDB Cloud](https://tidbcloud.com/) para obtener los detalles de conexión.
