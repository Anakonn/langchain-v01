# ForefrontAI

Esta página cubre cómo usar el ecosistema ForefrontAI dentro de LangChain.
Se divide en dos partes: instalación y configuración, y luego referencias a envolturas específicas de ForefrontAI.

## Instalación y configuración

- Obtén una clave de API de ForefrontAI y configúrala como una variable de entorno (`FOREFRONTAI_API_KEY`)

## Envolturas

### LLM

Existe una envoltura de LLM de ForefrontAI, a la que puedes acceder con

