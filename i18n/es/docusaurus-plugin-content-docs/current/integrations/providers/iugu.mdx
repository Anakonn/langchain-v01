# Iugu

>[Iugu](https://www.iugu.com/) es una empresa brasileña de servicios y software como servicio (SaaS). Ofrece software de procesamiento de pagos e interfaces de programación de aplicaciones para sitios web de comercio electrónico y aplicaciones móviles.

## Instalación y configuración

La `API de Iugu` requiere un token de acceso, que se puede encontrar dentro del panel de control de `Iugu`.
