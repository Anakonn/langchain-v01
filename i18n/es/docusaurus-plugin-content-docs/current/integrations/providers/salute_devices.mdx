# Dispositivos de saludo

Salute Devices proporciona modelos de GigaChat LLM.

Para más información sobre cómo obtener acceso a GigaChat [siga aquí](https://developers.sber.ru/docs/ru/gigachat/api/integration).

## Instalación y configuración

El paquete GigaChat se puede instalar a través de pip desde PyPI:

```bash
pip install gigachat
```
