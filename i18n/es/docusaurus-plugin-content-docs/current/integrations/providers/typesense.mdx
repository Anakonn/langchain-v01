# Typesense

> [Typesense](https://typesense.org) es un motor de búsqueda de código abierto y en memoria, que puedes [autoalojar](https://typesense.org/docs/guide/install-typesense.html#option-2-local-machine-self-hosting) o ejecutar en [Typesense Cloud](https://cloud.typesense.org/). `Typesense` se enfoca en el rendimiento al almacenar todo el índice en la RAM (con una copia de seguridad en disco) y también se enfoca en proporcionar una experiencia de desarrollador lista para usar al simplificar las opciones disponibles y establecer buenos valores predeterminados.

## Instalación y configuración

```bash
pip install typesense openapi-schema-pydantic
```
