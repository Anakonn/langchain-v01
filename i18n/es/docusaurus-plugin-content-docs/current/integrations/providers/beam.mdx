# Haz

>[Haz](https://www.beam.cloud/) es una plataforma de computación en la nube que te permite ejecutar tu código
> en servidores remotos con GPUs.

## Instalación y configuración

- [Crea una cuenta](https://www.beam.cloud/)
- Instala la CLI de Haz con `curl https://raw.githubusercontent.com/slai-labs/get-beam/main/get-beam.sh -sSfL | sh`
- Registra las claves API con `beam configure`
- Establece las variables de entorno (`BEAM_CLIENT_ID`) y (`BEAM_CLIENT_SECRET`)
- Instala el SDK de Haz:

```bash
pip install beam-sdk
```
