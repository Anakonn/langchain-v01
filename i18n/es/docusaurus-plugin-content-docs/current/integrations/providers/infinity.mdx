# Infinito

>[Infinito](https://github.com/michaelfeil/infinity) permite la creación de incrustaciones de texto.

## Modelo de incrustación de texto

Existe un modelo de incrustación de Infinito, al que se puede acceder con

```python
<!--IMPORTS:[{"imported": "InfinityEmbeddings", "source": "langchain_community.embeddings", "docs": "https://api.python.langchain.com/en/latest/embeddings/langchain_community.embeddings.infinity.InfinityEmbeddings.html", "title": "Infinity"}]-->
from langchain_community.embeddings import InfinityEmbeddings
```

Para una guía más detallada de esto, consulte [este cuaderno](/docs/integrations/text_embedding/infinity)
