# Kinetica

[Kinetica](https://www.kinetica.com/) es una base de datos en tiempo real diseñada específicamente para permitir
análisis y IA generativa en datos de series temporales y espaciales.

## Modelo de chat

El wrapper de LLM de Kinetica utiliza el [Kinetica SqlAssist
LLM](https://docs.kinetica.com/7.2/sql-gpt/concepts/) para transformar el lenguaje natural en
SQL y simplificar el proceso de recuperación de datos.

Consulte [Kinetica SqlAssist LLM Demo](/docs/integrations/chat/kinetica) para su uso.

```python
<!--IMPORTS:[{"imported": "ChatKinetica", "source": "langchain_community.chat_models.kinetica", "docs": "https://api.python.langchain.com/en/latest/chat_models/langchain_community.chat_models.kinetica.ChatKinetica.html", "title": "Kinetica"}]-->

```
