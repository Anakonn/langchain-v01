# Fauna

>[Fauna](https://fauna.com/) es una base de datos de documentos relacionales distribuida
> que combina la flexibilidad de los documentos con el poder de una base de datos
> relacional, compatible con ACID, que se escala a través de regiones, nubes o el mundo.

## Instalación y configuración

Tenemos que obtener la clave secreta.
Consulta la [guía](https://docs.fauna.com/fauna/current/learn/security_model/) detallada.

Tenemos que instalar el paquete `fauna`.

```bash
pip install -U fauna
```
