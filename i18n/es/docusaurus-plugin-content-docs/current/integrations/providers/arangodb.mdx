# ArangoDB

>[ArangoDB](https://github.com/arangodb/arangodb) es un sistema de base de datos de gráficos escalable para
> extraer valor de los datos conectados, más rápido. Gráficos nativos, un motor de búsqueda integrado y soporte JSON, a través de un solo lenguaje de consulta. ArangoDB se ejecuta en las instalaciones, en la nube, en cualquier lugar.

## Instalación y configuración

Instale el paquete [ArangoDB Python Driver](https://github.com/ArangoDB-Community/python-arango) con

```bash
pip install python-arango
```
