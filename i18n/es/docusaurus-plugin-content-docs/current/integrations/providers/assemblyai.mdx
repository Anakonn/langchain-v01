# AssemblyAI

>[AssemblyAI](https://www.assemblyai.com/) construye modelos de `Speech AI` para tareas como
transcripción de voz a texto, diarización de altavoces, resumen de discursos y más.
> Los modelos de `Speech AI` de `AssemblyAI` incluyen una precisa transcripción de voz a texto para datos de voz
> (como llamadas, reuniones virtuales y podcasts), detección de altavoces, análisis de sentimiento,
> detección de capítulos, enmascaramiento de información personal.

## Instalación y configuración

Obtén tu [clave API](https://www.assemblyai.com/dashboard/signup).

Instala el paquete `assemblyai`.

```bash
pip install -U assemblyai
```
