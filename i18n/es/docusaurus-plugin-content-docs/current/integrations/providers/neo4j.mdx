# Neo4j

>¿Qué es `Neo4j`?

>- Neo4j es un `sistema de gestión de bases de datos de código abierto` que se especializa en tecnología de bases de datos de grafos.
>- Neo4j le permite representar y almacenar datos en nodos y bordes, lo que lo hace ideal para manejar datos conectados y relaciones.
>- Neo4j proporciona un `Lenguaje de Consulta Cypher`, lo que facilita la interacción y consulta de sus datos de gráficos.
>- Con Neo4j, puede lograr `recorridos y consultas de gráficos de alto rendimiento`, adecuados para sistemas de nivel de producción.

>Comience a usar Neo4j visitando [su sitio web](https://neo4j.com/).

## Instalación y configuración

- Instale el SDK de Python con `pip install neo4j`

## Construir un grafo de conocimiento a partir de texto

Los datos de texto a menudo contienen relaciones e ideas ricas que pueden ser útiles para diversos análisis, motores de recomendación o aplicaciones de gestión del conocimiento.
La API de NLP de Diffbot permite la extracción de entidades, relaciones y significado semántico de datos de texto no estructurados.
Al combinar la API de NLP de Diffbot con Neo4j, una base de datos de gráficos, puede crear estructuras de gráficos poderosas y dinámicas basadas en la información extraída del texto.
Estas estructuras de gráficos son completamente consultables y se pueden integrar en varias aplicaciones.
