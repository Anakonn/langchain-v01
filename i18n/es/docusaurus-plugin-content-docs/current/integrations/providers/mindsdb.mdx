# MindsDB

MindsDB es la plataforma para personalizar la IA a partir de datos empresariales. Con MindsDB y sus casi 200 integraciones con [fuentes de datos](https://docs.mindsdb.com/integrations/data-overview) y [marcos de IA/ML](https://docs.mindsdb.com/integrations/ai-overview), cualquier desarrollador puede usar sus datos empresariales para personalizar la IA para su propósito, de manera más rápida y segura.

Con MindsDB, puede conectar cualquier fuente de datos a cualquier modelo de IA/ML para implementar y automatizar aplicaciones impulsadas por IA. Implementar, servir y ajustar modelos en tiempo real, utilizando datos de bases de datos, tiendas de vectores o aplicaciones. Haga todo eso usando herramientas universales que los desarrolladores ya conocen.

MindsDB se integra con LangChain, lo que permite a los usuarios:

- Implementar modelos disponibles a través de LangChain dentro de MindsDB, haciéndolos accesibles a numerosas fuentes de datos.
- Ajustar modelos disponibles a través de LangChain dentro de MindsDB utilizando datos en tiempo real y dinámicos.
- Automatizar flujos de trabajo de IA con LangChain y MindsDB.

Siga [nuestra documentación](https://docs.mindsdb.com/integrations/ai-engines/langchain) para aprender más sobre la integración de MindsDB con LangChain y ver ejemplos.
