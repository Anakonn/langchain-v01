# Atlas

>[Atlas Nomic](https://docs.nomic.ai/index.html) es una plataforma para interactuar con conjuntos de datos no estructurados a pequeña y gran escala de Internet.

## Instalación y configuración

- Instale el paquete de Python con `pip install nomic`
- `Nomic` también se incluye en los extras de langchains poetry `poetry install -E all`
