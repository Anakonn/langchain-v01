# Git

>[Git](https://en.wikipedia.org/wiki/Git) es un sistema de control de versiones distribuido que realiza un seguimiento de los cambios en cualquier conjunto de archivos de computadora, generalmente utilizado para coordinar el trabajo entre programadores que desarrollan colaborativamente el código fuente durante el desarrollo de software.

## Instalación y configuración

Primero, debe instalar el paquete de Python `GitPython`.

```bash
pip install GitPython
```
