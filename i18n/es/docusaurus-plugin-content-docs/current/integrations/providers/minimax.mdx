# Minimax

>[Minimax](https://api.minimax.chat) es una startup china que proporciona modelos de procesamiento de lenguaje natural
> para empresas e individuos.

## Instalación y configuración

Obtén una [clave de API de Minimax](https://api.minimax.chat/user-center/basic-information/interface-key) y configúrala como una variable de entorno (`MINIMAX_API_KEY`)
Obtén un [ID de grupo de Minimax](https://api.minimax.chat/user-center/basic-information) y configúralo como una variable de entorno (`MINIMAX_GROUP_ID`)
