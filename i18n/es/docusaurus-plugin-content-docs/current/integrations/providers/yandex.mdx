# Yandex

Toda la funcionalidad relacionada con Yandex Cloud

>[Yandex Cloud](https://cloud.yandex.com/en/) es una plataforma de nube pública.

## Instalación y configuración

El SDK de Yandex Cloud se puede instalar a través de pip desde PyPI:

```bash
pip install yandexcloud
```
