# GooseAI

Esta página cubre cómo usar el ecosistema de GooseAI dentro de LangChain.
Se divide en dos partes: instalación y configuración, y luego referencias a envolturas específicas de GooseAI.

## Instalación y configuración

- Instala el SDK de Python con `pip install openai`
- Obtén tu clave de API de GooseAI de este enlace [aquí](https://goose.ai/).
- Establece la variable de entorno (`GOOSEAI_API_KEY`).

```python
import os
os.environ["GOOSEAI_API_KEY"] = "YOUR_API_KEY"
```
