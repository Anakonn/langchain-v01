# lakeFS

>[lakeFS](https://docs.lakefs.io/) proporciona control de versiones escalable sobre
> el lago de datos y utiliza semántica similar a Git para crear y acceder a esas versiones.

## Instalación y configuración

Obtenga el `ENDPOINT`, `LAKEFS_ACCESS_KEY` y `LAKEFS_SECRET_KEY`.
Puede encontrar las instrucciones de instalación [aquí](https://docs.lakefs.io/quickstart/launch.html).
