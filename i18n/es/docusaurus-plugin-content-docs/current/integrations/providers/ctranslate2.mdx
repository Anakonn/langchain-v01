# CTranslate2

>[CTranslate2](https://opennmt.net/CTranslate2/quickstart.html) es una biblioteca de C++ y Python
> para inferencia eficiente con modelos Transformer.
>
>El proyecto implementa un tiempo de ejecución personalizado que aplica muchas técnicas de optimización de rendimiento
> como cuantificación de pesos, fusión de capas, reordenamiento por lotes, etc.,
> para acelerar y reducir el uso de memoria de los modelos Transformer en CPU y GPU.
>
>Una lista completa de características y modelos compatibles se incluye en el
> [repositorio del proyecto](https://opennmt.net/CTranslate2/guides/transformers.html).
> Para comenzar, consulte la [guía de inicio rápido](https://opennmt.net/CTranslate2/quickstart.html) oficial.

## Instalación y configuración

Instala el paquete de Python:

```bash
pip install ctranslate2
```
