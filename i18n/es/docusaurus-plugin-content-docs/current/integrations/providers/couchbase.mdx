# Couchbase

>[Couchbase](http://couchbase.com/) es una base de datos en la nube NoSQL distribuida galardonada
> que ofrece una versatilidad, un rendimiento, una escalabilidad y un valor financiero inigualables
> para todas sus aplicaciones de computación en la nube, móviles, de IA y periféricas.

## Instalación y configuración

Tenemos que instalar el paquete `couchbase`.

```bash
pip install couchbase
```
