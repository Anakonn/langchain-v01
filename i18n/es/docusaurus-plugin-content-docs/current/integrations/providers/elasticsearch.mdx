# Elasticsearch

> [Elasticsearch](https://www.elastic.co/elasticsearch/) es un motor de búsqueda y análisis distribuido y RESTful.
> Proporciona un motor de búsqueda de texto completo distribuido y multiinquilino con una interfaz web HTTP y documentos JSON sin esquema.

## Instalación y configuración

Hay dos formas de comenzar con Elasticsearch:

#### Instalar Elasticsearch en tu máquina local a través de docker

Ejemplo: Ejecutar una instancia de Elasticsearch de un solo nodo con la seguridad desactivada. Esto no se recomienda para uso en producción.

```bash
    docker run -p 9200:9200 -e "discovery.type=single-node" -e "xpack.security.enabled=false" -e "xpack.security.http.ssl.enabled=false" docker.elastic.co/elasticsearch/elasticsearch:8.9.0
```

#### Implementar Elasticsearch en Elastic Cloud

Elastic Cloud es un servicio de Elasticsearch administrado. Regístrate para una [prueba gratuita](https://cloud.elastic.co/registration?utm_source=langchain&utm_content=documentation).

### Instalar cliente

```bash
pip install elasticsearch
pip install langchain-elasticsearch
```
