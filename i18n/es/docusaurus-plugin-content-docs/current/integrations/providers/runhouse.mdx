# Runhouse

Esta página cubre cómo usar el ecosistema [Runhouse](https://github.com/run-house/runhouse) dentro de LangChain.
Se divide en tres partes: instalación y configuración, LLMs y Embeddings.

## Instalación y configuración

- Instala el SDK de Python con `pip install runhouse`
- Si deseas usar un clúster bajo demanda, verifica tus credenciales en la nube con `sky check`

## LLMs autohospedados

Para un LLM autohospedado básico, puedes usar la clase `SelfHostedHuggingFaceLLM`. Para LLMs más
personalizados, puedes usar la clase padre `SelfHostedPipeline`.

la clase `SelfHostedEmbedding`.
