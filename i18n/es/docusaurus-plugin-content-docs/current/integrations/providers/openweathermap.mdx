# OpenWeatherMap

>[OpenWeatherMap](https://openweathermap.org/api/) proporciona todos los datos meteorológicos esenciales para una ubicación específica:
>- Clima actual
>- Pronóstico por minuto para 1 hora
>- Pronóstico por hora para 48 horas
>- Pronóstico diario para 8 días
>- Alertas meteorológicas nacionales
>- Datos meteorológicos históricos de más de 40 años

Esta página cubre cómo usar la `OpenWeatherMap API` dentro de LangChain.

## Instalación y configuración

- Instalar los requisitos con

```bash
pip install pyowm
```

- Ir a OpenWeatherMap y registrarse para obtener su clave API [aquí](https://openweathermap.org/api/)
- Establecer su clave API como la variable de entorno `OPENWEATHERMAP_API_KEY`

## Wrappers

### Utilidad

Existe una utilidad OpenWeatherMapAPIWrapper que envuelve esta API. Para importar esta utilidad:

```python
<!--IMPORTS:[{"imported": "OpenWeatherMapAPIWrapper", "source": "langchain_community.utilities.openweathermap", "docs": "https://api.python.langchain.com/en/latest/utilities/langchain_community.utilities.openweathermap.OpenWeatherMapAPIWrapper.html", "title": "OpenWeatherMap"}]-->
from langchain_community.utilities.openweathermap import OpenWeatherMapAPIWrapper
```

Para obtener un recorrido más detallado de este wrapper, consulte [este cuaderno](/docs/integrations/tools/openweathermap).

### Herramienta

También puede cargar fácilmente este wrapper como una Herramienta (para usar con un Agente).
Puede hacer esto con:

```python
<!--IMPORTS:[{"imported": "load_tools", "source": "langchain.agents", "docs": "https://api.python.langchain.com/en/latest/agent_toolkits/langchain_community.agent_toolkits.load_tools.load_tools.html", "title": "OpenWeatherMap"}]-->
from langchain.agents import load_tools
tools = load_tools(["openweathermap-api"])
```

Para obtener más información sobre las herramientas, consulte [esta página](/docs/modules/tools/).
