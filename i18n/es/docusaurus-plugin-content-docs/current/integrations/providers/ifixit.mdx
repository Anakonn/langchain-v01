# iFixit

>[iFixit](https://www.ifixit.com) es la comunidad de reparación abierta más grande de la web. El sitio contiene casi 100k
> manuales de reparación, 200k preguntas y respuestas sobre 42k dispositivos, y todos los datos se publican bajo `CC-BY-NC-SA 3.0`.

## Instalación y configuración

No hay una configuración especial para ello.
