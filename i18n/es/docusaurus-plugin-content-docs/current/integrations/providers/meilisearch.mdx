# Meilisearch

> [Meilisearch](https://meilisearch.com) es un motor de búsqueda de código abierto, ultrarrápido y muy
> relevante.
> Viene con excelentes valores predeterminados para ayudar a los desarrolladores a construir experiencias de búsqueda ágiles.
>
> Puede [autoalojar Meilisearch](https://www.meilisearch.com/docs/learn/getting_started/installation#local-installation)
> o ejecutarlo en [Meilisearch Cloud](https://www.meilisearch.com/pricing).
>
>`Meilisearch v1.3` admite la búsqueda vectorial.

## Instalación y configuración

Consulte un [ejemplo de uso](/docs/integrations/vectorstores/meilisearch) para obtener instrucciones de configuración detalladas.

Necesitamos instalar el paquete de python `meilisearch`.

```bash
pip install meilisearch
```
