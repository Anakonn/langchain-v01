# Sin estructura

> El paquete `unstructured` de
[Unstructured.IO](https://www.unstructured.io/) extrae texto limpio de documentos fuente sin procesar como
archivos PDF y documentos de Word.
Esta página cubre cómo usar el [`unstructured`](https://github.com/Unstructured-IO/unstructured)
ecosistema dentro de LangChain.

## Instalación y configuración

Si está utilizando un cargador que se ejecuta localmente, use los siguientes pasos para obtener `unstructured` y
sus dependencias en ejecución local.

- Instale el SDK de Python con `pip install unstructured`.
    - Puede instalar dependencias específicas de documentos con extras, es decir, `pip install "unstructured[docx]"`.
    - Para instalar las dependencias de todos los tipos de documentos, use `pip install "unstructured[all-docs]"`.
- Instale las siguientes dependencias del sistema si aún no están disponibles en su sistema.
  Dependiendo de los tipos de documentos que esté analizando, es posible que no necesite todas estas.
    - `libmagic-dev` (detección de tipo de archivo)
    - `poppler-utils` (imágenes y PDF)
    - `tesseract-ocr` (imágenes y PDF)
    - `libreoffice` (documentos de MS Office)
    - `pandoc` (EPUB)

Si desea ponerse en marcha con menos configuración, puede
simplemente ejecutar `pip install unstructured` y usar `UnstructuredAPIFileLoader` o
`UnstructuredAPIFileIOLoader`. Eso procesará su documento usando la API alojada de Unstructured.

La `API de Unstructured` requiere claves API para hacer solicitudes.
Puede solicitar una clave API [aquí](https://unstructured.io/api-key-hosted) y comenzar a usarla hoy mismo.
Consulte el README [aquí](https://github.com/Unstructured-IO/unstructured-api) para comenzar a hacer llamadas a la API.
Nos encantaría escuchar sus comentarios, háganoslo saber en nuestro [slack de la comunidad](https://join.slack.com/t/unstructuredw-kbe4326/shared_invite/zt-1x7cgo0pg-PTptXWylzPQF9xZolzCnwQ).
¡Y manténgase atento a las mejoras tanto en calidad como en rendimiento!
Consulte las instrucciones
[aquí](https://github.com/Unstructured-IO/unstructured-api#dizzy-instructions-for-using-the-docker-image) si desea alojar la API de Unstructured o ejecutarla localmente.
