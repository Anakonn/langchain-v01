# Asistente LLM

Este cuaderno cubre cómo usar `LLM Sherpa` para cargar archivos de muchos tipos. `LLM Sherpa` admite diferentes formatos de archivo, incluidos DOCX, PPTX, HTML, TXT y XML.

`LLMSherpaFileLoader` usa LayoutPDFReader, que es parte de la biblioteca LLMSherpa. Esta herramienta está diseñada para analizar PDF mientras conserva su información de diseño, que a menudo se pierde cuando se usan la mayoría de los analizadores de texto PDF.

Aquí hay algunas características clave de LayoutPDFReader:

* Puede identificar y extraer secciones y subsecciones junto con sus niveles.
* Combina líneas para formar párrafos.
* Puede identificar vínculos entre secciones y párrafos.
* Puede extraer tablas junto con la sección en la que se encuentran las tablas.
* Puede identificar y extraer listas y listas anidadas.
* Puede unir el contenido distribuido en varias páginas.
* Puede eliminar encabezados y pies de página repetidos.
* Puede eliminar marcas de agua.

revisa [llmsherpa](https://llmsherpa.readthedocs.io/en/latest/) documentación.

`INFORMACIÓN: esta biblioteca falla con algunos archivos PDF, así que úsala con precaución.`

## LLMSherpaFileLoader

Debajo de LLMSherpaFileLoader se definieron algunas estrategias para cargar el contenido del archivo: ["sections", "chunks", "html", "text"], configurar [nlm-ingestor](https://github.com/nlmatics/nlm-ingestor) para obtener `llmsherpa_api_url` o usar el predeterminado.

### estrategia de secciones: devuelve el archivo analizado en secciones

```python
from langchain_community.document_loaders.llmsherpa import LLMSherpaFileLoader

loader = LLMSherpaFileLoader(
    file_path="https://arxiv.org/pdf/2402.14207.pdf",
    new_indent_parser=True,
    apply_ocr=True,
    strategy="sections",
    llmsherpa_api_url="http://localhost:5010/api/parseDocument?renderFormat=all",
)
docs = loader.load()
```

```python
docs[1]
```
