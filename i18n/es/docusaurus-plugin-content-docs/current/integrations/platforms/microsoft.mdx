### Azure Blob Storage

>[Azure Blob Storage](https://learn.microsoft.com/en-us/azure/storage/blobs/storage-blobs-introduction) es la solución de almacenamiento de objetos de Microsoft para la nube. Blob Storage está optimizado para almacenar grandes cantidades de datos no estructurados. Los datos no estructurados son datos que no se ajustan a un modelo de datos o definición en particular, como datos de texto o binarios.

>[Azure Files](https://learn.microsoft.com/en-us/azure/storage/files/storage-files-introduction) ofrece recursos compartidos de archivos totalmente administrados en la nube que se pueden acceder a través del protocolo Server Message Block (`SMB`) estándar de la industria,
> el protocolo Network File System (`NFS`) y la `API REST de Azure Files`. `Azure Files` se basa en `Azure Blob Storage`.

`Azure Blob Storage` está diseñado para:
- Servir imágenes o documentos directamente a un navegador.
- Almacenar archivos para acceso distribuido.
- Transmitir video y audio.
- Escribir en archivos de registro.
- Almacenar datos para copia de seguridad y restauración, recuperación ante desastres y archivado.
- Almacenar datos para análisis por un servicio local o alojado en Azure.

```bash
pip install azure-storage-blob
```

## Recuperadores

### Azure AI Search

>[Azure AI Search](https://learn.microsoft.com/en-us/azure/search/search-what-is-azure-search) (anteriormente conocido como `Azure Search` o `Azure Cognitive Search`) es un servicio de búsqueda en la nube que brinda a los desarrolladores infraestructura, API y herramientas para construir una experiencia de búsqueda enriquecida sobre contenido privado y heterogéneo en aplicaciones web, móviles y empresariales.

>La búsqueda es fundamental para cualquier aplicación que muestre texto a los usuarios, donde los escenarios comunes incluyen la búsqueda de catálogos o documentos, aplicaciones de comercio electrónico o la exploración de datos sobre contenido privado. Cuando creas un servicio de búsqueda, trabajarás con las siguientes capacidades:
>- Un motor de búsqueda para la búsqueda de texto completo sobre un índice de búsqueda que contiene contenido propiedad del usuario
>- Indexación enriquecida, con análisis léxico y enriquecimiento de IA opcional para la extracción y transformación de contenido
>- Sintaxis de consulta enriquecida para búsqueda de texto, búsqueda difusa, autocompletar, búsqueda geográfica y más
>- Programabilidad a través de API REST y bibliotecas de clientes en los SDK de Azure
>- Integración de Azure a nivel de datos, capa de aprendizaje automático y servicios de IA

Consulta las [instrucciones de configuración](https://learn.microsoft.com/en-us/azure/search/search-create-service-portal).
