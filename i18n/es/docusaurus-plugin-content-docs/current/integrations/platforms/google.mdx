# Google

Toda la funcionalidad relacionada con [Google Cloud Platform](https://cloud.google.com/) y otros productos de `Google`.

## LLMs

Recomendamos a los desarrolladores individuales comenzar con la API de Gemini (`langchain-google-genai`) y pasar a Vertex AI (`langchain-google-vertexai`) cuando necesiten acceso a soporte comercial y límites de tasa más altos. Si ya estás familiarizado con la nube o eres nativo de la nube, entonces puedes empezar directamente con Vertex AI.
Por favor, encuentra más información [aquí](https://ai.google.dev/gemini-api/docs/migrate-to-cloud).

Accede a los modelos `Gemini` de GoogleAI como `gemini-pro` y `gemini-pro-vision` a través de la clase `GoogleGenerativeAI`.

## Modelos de chat

### Google Generative AI

Accede a los modelos `Gemini` de GoogleAI como `gemini-pro` y `gemini-pro-vision` a través de la clase `ChatGoogleGenerativeAI`.

```bash
pip install -U langchain-google-genai
```

Configura tu clave API.

```bash
export GOOGLE_API_KEY=your-api-key
```

```python
from langchain_google_genai import ChatGoogleGenerativeAI

llm = ChatGoogleGenerativeAI(model="gemini-pro")
llm.invoke("Sing a ballad of LangChain.")
```

El modelo de visión de Gemini admite entradas de imágenes al proporcionar un solo mensaje de chat.

```python
<!--IMPORTS:[{"imported": "HumanMessage", "source": "langchain_core.messages", "docs": "https://api.python.langchain.com/en/latest/messages/langchain_core.messages.human.HumanMessage.html", "title": "Google"}]-->
from langchain_core.messages import HumanMessage
from langchain_google_genai import ChatGoogleGenerativeAI

llm = ChatGoogleGenerativeAI(model="gemini-pro-vision")

message = HumanMessage(
    content=[
        {
            "type": "text",
            "text": "What's in this image?",
        },  # You can optionally provide text parts
        {"type": "image_url", "image_url": "https://picsum.photos/seed/picsum/200/300"},
    ]
)
llm.invoke([message])
```

El valor de image_url puede ser cualquiera de los siguientes:

- Una URL pública de imagen
- Un archivo gcs (por ejemplo, "gcs://path/to/file.png")
- Una ruta de archivo local
- Una imagen codificada en base64 (por ejemplo, data:image/png;base64,abcd124)
- Una imagen PIL
