# Hugging Face

Toda la funcionalidad relacionada con la [Plataforma Hugging Face](https://huggingface.co/).

## Modelos de chat

### Modelos de Hugging Face

Podemos usar las clases LLM de `Hugging Face` o usar directamente la clase `ChatHuggingFace`.

Necesitamos instalar varios paquetes de Python.

```bash
pip install huggingface_hub
pip install transformers
```

Consulta un [ejemplo de uso](/docs/integrations/chat/huggingface).

```python
<!--IMPORTS:[{"imported": "ChatHuggingFace", "source": "langchain_community.chat_models.huggingface", "docs": "https://api.python.langchain.com/en/latest/chat_models/langchain_community.chat_models.huggingface.ChatHuggingFace.html", "title": "Hugging Face"}]-->
```

## LLMs

### Tuberías locales de Hugging Face

Los modelos de Hugging Face se pueden ejecutar localmente a través de la clase `HuggingFacePipeline`.

Necesitamos instalar el paquete de Python `transformers`.

```bash
pip install transformers
```

Consulta un [ejemplo de uso](/docs/integrations/llms/huggingface_pipelines).

```python
<!--IMPORTS:[{"imported": "HuggingFacePipeline", "source": "langchain_community.llms.huggingface_pipeline", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.huggingface_pipeline.HuggingFacePipeline.html", "title": "Hugging Face"}]-->
from langchain_community.llms.huggingface_pipeline import HuggingFacePipeline
```

Para usar el backend OpenVINO en el wrapper de la tubería local, instala la biblioteca optimum y establece el backend de HuggingFacePipeline como `openvino`:

```bash
pip install --upgrade-strategy eager "optimum[openvino,nncf]"
```

Consulta un [ejemplo de uso](/docs/integrations/llms/huggingface_pipelines).

Para exportar tu modelo al formato IR de OpenVINO con la CLI:

```bash
optimum-cli export openvino --model gpt2 ov_model
```

Para aplicar la [cuantificación solo de pesos](https://github.com/huggingface/optimum-intel?tab=readme-ov-file#export) al exportar tu modelo.

## Modelos de incrustación

### Hugging Face Hub

>El [Hugging Face Hub](https://huggingface.co/docs/hub/index) es una plataforma
> con más de 350,000 modelos, 75,000 conjuntos de datos y 150,000 aplicaciones de demostración (Spaces), todo ello de código abierto
> y de acceso público, en una plataforma en línea donde las personas pueden colaborar y construir ML juntas. El Hub funciona como un lugar central donde cualquiera
> puede explorar, experimentar, colaborar y construir tecnología con Machine Learning.

Necesitamos instalar el paquete de Python `sentence_transformers`.

```bash
pip install sentence_transformers
```

### Hugging Face Text Embeddings Inference (TEI)

>[Hugging Face Text Embeddings Inference (TEI)](https://huggingface.co/docs/text-generation-inference/index) es un kit de herramientas para implementar y servir incrustaciones de texto de código abierto
> y modelos de clasificación de secuencias. `TEI` permite la extracción de alto rendimiento para los modelos más populares,
>incluyendo `FlagEmbedding`, `Ember`, `GTE` y `E5`.

Necesitamos instalar el paquete de Python `huggingface-hub`.

```bash
pip install huggingface-hub
```

## Cargadores de documentos

### Conjunto de datos de Hugging Face

>[Hugging Face Hub](https://huggingface.co/docs/hub/index) es el hogar de más de 75,000
> [conjuntos de datos](https://huggingface.co/docs/hub/index#datasets) en más de 100 idiomas
> que se pueden utilizar para una amplia gama de tareas en NLP, Visión por Computador y Audio.
