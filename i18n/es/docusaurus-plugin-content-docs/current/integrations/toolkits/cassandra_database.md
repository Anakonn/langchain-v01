## Herramientas proporcionadas

### `cassandra_db_schema`

Recopila toda la informaciÃ³n del esquema para la base de datos conectada o un esquema especÃ­fico. Fundamental para que el agente determine las acciones.

### `cassandra_db_select_table_data`

Selecciona datos de un espacio de claves y una tabla especÃ­ficos. El agente puede pasar parÃ¡metros para un predicado y lÃ­mites en el nÃºmero de registros devueltos.

### `cassandra_db_query`

Alternativa experimental a `cassandra_db_select_table_data` que toma una cadena de consulta completamente formada por el agente en lugar de parÃ¡metros. *Advertencia*: Esto puede dar lugar a consultas inusuales que pueden no ser tan eficientes (o incluso funcionar).
