# People

LangChain 커뮤니티가 번창하는 데 기여한 전 세계의 놀라운 사람들을 소개합니다 🌐!

이 페이지는 오픈 소스 저장소에 직접 기여하고 리뷰하는 형태로 시간을 할애한 몇몇 분들을 강조합니다.

## 상위 리뷰어

LangChain이 성장함에 따라 유지보수자가 다루어야 할 범위도 확장되었습니다.

다음 분들께서는 들어오는 PR을 리뷰하는 데 있어 탁월한 노력을 기울여 주셨습니다 🙏!
