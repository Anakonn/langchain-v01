# 구글

[Google Cloud Platform](https://cloud.google.com/) 및 기타 `Google` 제품과 관련된 모든 기능.

## LLMs

개별 개발자는 Gemini API (`langchain-google-genai`)로 시작하고 상업적 지원 및 더 높은 속도 제한이 필요할 때 Vertex AI (`langchain-google-vertexai`)로 이동하는 것을 권장합니다. 이미 Cloud-friendly 또는 Cloud-native라면 Vertex AI에서 바로 시작할 수 있습니다.
자세한 정보는 [여기](https://ai.google.dev/gemini-api/docs/migrate-to-cloud)에서 찾을 수 있습니다.

### Google 생성 AI

`GoogleGenerativeAI` 클래스를 통해 `gemini-pro` 및 `gemini-pro-vision`과 같은 GoogleAI `Gemini` 모델에 접근하세요.

파이썬 패키지를 설치하세요.

```bash
pip install langchain-google-genai
```

[사용 예제](/docs/integrations/llms/google_ai)를 참조하세요.

```python
from langchain_google_genai import GoogleGenerativeAI
```

## 채팅 모델

### Google 생성 AI

`ChatGoogleGenerativeAI` 클래스를 통해 `gemini-pro` 및 `gemini-pro-vision`과 같은 GoogleAI `Gemini` 모델에 접근하세요.

```bash
pip install -U langchain-google-genai
```

API 키를 구성하세요.

```bash
export GOOGLE_API_KEY=your-api-key
```

```python
from langchain_google_genai import ChatGoogleGenerativeAI

llm = ChatGoogleGenerativeAI(model="gemini-pro")
llm.invoke("Sing a ballad of LangChain.")
```

Gemini 비전 모델은 단일 채팅 메시지를 제공할 때 이미지 입력을 지원합니다.

```python
<!--IMPORTS:[{"imported": "HumanMessage", "source": "langchain_core.messages", "docs": "https://api.python.langchain.com/en/latest/messages/langchain_core.messages.human.HumanMessage.html", "title": "Google"}]-->
from langchain_core.messages import HumanMessage
from langchain_google_genai import ChatGoogleGenerativeAI

llm = ChatGoogleGenerativeAI(model="gemini-pro-vision")

message = HumanMessage(
    content=[
        {
            "type": "text",
            "text": "What's in this image?",
        },  # You can optionally provide text parts
        {"type": "image_url", "image_url": "https://picsum.photos/seed/picsum/200/300"},
    ]
)
llm.invoke([message])
```

image_url의 값은 다음 중 하나일 수 있습니다:

- 공개 이미지 URL
- gcs 파일 (예: "gcs://path/to/file.png")
- 로컬 파일 경로
- base64로 인코딩된 이미지 (예: data:image/png;base64,abcd124)
- PIL 이미지

### YouTube

>[YouTube Search](https://github.com/joetats/youtube_search) 패키지는 사용이 제한된 API를 피하면서 `YouTube` 동영상을 검색합니다.
>
>YouTube 홈페이지의 양식을 사용하고 결과 페이지를 스크랩합니다.

Python 패키지를 설치해야 합니다.

```bash
pip install youtube_search
```

[사용 예제](/docs/integrations/tools/youtube)을 참조하세요.

```python
<!--IMPORTS:[{"imported": "YouTubeSearchTool", "source": "langchain.tools", "docs": "https://api.python.langchain.com/en/latest/tools/langchain_community.tools.youtube.search.YouTubeSearchTool.html", "title": "Google"}]-->
```

### YouTube 오디오

>[YouTube](https://www.youtube.com/) 는 `Google`이 만든 온라인 비디오 공유 및 소셜 미디어 플랫폼입니다.

`YoutubeAudioLoader`를 사용하여 오디오 파일을 가져오거나 다운로드합니다.

그런 다음, `OpenAIWhisperParser`를 사용하여 텍스트로 전사합니다.

여러 Python 패키지를 설치해야 합니다.

```bash
pip install yt_dlp pydub librosa
```

[사용 예제 및 인증 지침](/docs/integrations/document_loaders/youtube_audio)을 참조하세요.

```python
<!--IMPORTS:[{"imported": "YoutubeAudioLoader", "source": "langchain_community.document_loaders.blob_loaders.youtube_audio", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.blob_loaders.youtube_audio.YoutubeAudioLoader.html", "title": "Google"}, {"imported": "OpenAIWhisperParser", "source": "langchain_community.document_loaders.parsers", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.parsers.audio.OpenAIWhisperParser.html", "title": "Google"}]-->
```
