### Azure OpenAI

>[Microsoft Azure](https://en.wikipedia.org/wiki/Microsoft_Azure)는 `Microsoft`가 운영하는 클라우드 컴퓨팅 플랫폼으로, 전 세계 데이터 센터를 통해 애플리케이션 및 서비스에 대한 액세스, 관리 및 개발을 제공합니다. 소프트웨어 서비스(SaaS), 플랫폼 서비스(PaaS), 인프라 서비스(IaaS) 등 다양한 기능을 제공합니다. `Microsoft Azure`는 Microsoft 고유 및 타사 소프트웨어와 시스템을 포함한 다양한 프로그래밍 언어, 도구 및 프레임워크를 지원합니다.

>[Azure OpenAI](https://learn.microsoft.com/en-us/azure/cognitive-services/openai/)는 `Azure` 서비스로, `GPT-3`, `Codex` 및 `Embeddings model` 시리즈를 포함한 `OpenAI`의 강력한 언어 모델을 제공하여 콘텐츠 생성, 요약, 의미 검색 및 자연어에서 코드 변환 등을 지원합니다.

### Azure Blob Storage

>[Azure Blob Storage](https://learn.microsoft.com/en-us/azure/storage/blobs/storage-blobs-introduction)는 Microsoft의 클라우드 객체 스토리지 솔루션입니다. Blob Storage는 대량의 비정형 데이터 저장에 최적화되어 있습니다. 비정형 데이터는 특정 데이터 모델이나 정의를 따르지 않는 텍스트 또는 이진 데이터입니다.

>[Azure Files](https://learn.microsoft.com/en-us/azure/storage/files/storage-files-introduction)는 업계 표준 Server Message Block(`SMB`) 프로토콜, Network File System(`NFS`) 프로토콜 및 `Azure Files REST API`를 통해 액세스할 수 있는 완전 관리형 클라우드 파일 공유를 제공합니다. `Azure Files`는 `Azure Blob Storage`를 기반으로 합니다.

`Azure Blob Storage`는 다음과 같은 용도로 설계되었습니다:
- 브라우저에 직접 이미지 또는 문서 제공
- 분산 액세스를 위한 파일 저장
- 비디오 및 오디오 스트리밍
- 로그 파일 작성
- 백업, 재해 복구 및 보관을 위한 데이터 저장
- 온-프레미스 또는 Azure 호스팅 서비스에 의한 데이터 분석

```bash
pip install azure-storage-blob
```

> Azure Cosmos DB for MongoDB vCore의 벡터 검색을 사용하여 AI 기반 애플리케이션을 Azure Cosmos DB에 저장된 데이터와 seamlessly 통합할 수 있습니다.

## 리트리버

### Azure AI Search

>[Azure AI Search](https://learn.microsoft.com/en-us/azure/search/search-what-is-azure-search)(이전에는 `Azure Search` 또는 `Azure Cognitive Search`라고 함)는 웹, 모바일 및 엔터프라이즈 애플리케이션의 사용자에게 텍스트를 표시하는 모든 앱에 기본적인 검색 기능을 제공하는 클라우드 검색 서비스입니다.

>검색은 카탈로그 또는 문서 검색, 온라인 소매 애플리케이션 또는 독점 콘텐츠에 대한 데이터 탐색과 같은 일반적인 시나리오에 필수적입니다. 검색 서비스를 만들 때 다음과 같은 기능을 사용할 수 있습니다:
>- 사용자 소유 콘텐츠에 대한 전체 텍스트 검색을 위한 검색 엔진
>- 어휘 분석 및 선택적 AI 강화를 통한 풍부한 인덱싱, 콘텐츠 추출 및 변환
>- 텍스트 검색, 퍼지 검색, 자동 완성, 지오 검색 등을 위한 풍부한 쿼리 구문
>- Azure SDK의 REST API 및 클라이언트 라이브러리를 통한 프로그래밍 기능
>- 데이터 계층, 기계 학습 계층 및 AI(AI 서비스)에서의 Azure 통합

[설정 지침](https://learn.microsoft.com/en-us/azure/search/search-create-service-portal)을 참조하세요.
