# Hugging Face

허깅 페이스 플랫폼과 관련된 모든 기능.

## 채팅 모델

### 허깅 페이스의 모델

`Hugging Face` LLM 클래스를 사용하거나 `ChatHuggingFace` 클래스를 직접 사용할 수 있습니다.

몇 가지 Python 패키지를 설치해야 합니다.

```bash
pip install huggingface_hub
pip install transformers
```

[사용 예시](/docs/integrations/chat/huggingface)를 참고하세요.

```python
<!--IMPORTS:[{"imported": "ChatHuggingFace", "source": "langchain_community.chat_models.huggingface", "docs": "https://api.python.langchain.com/en/latest/chat_models/langchain_community.chat_models.huggingface.ChatHuggingFace.html", "title": "Hugging Face"}]-->
```

## LLMs

### 허깅 페이스 로컬 파이프라인

허깅 페이스 모델은 `HuggingFacePipeline` 클래스를 통해 로컬에서 실행할 수 있습니다.

`transformers` Python 패키지를 설치해야 합니다.

```bash
pip install transformers
```

[사용 예시](/docs/integrations/llms/huggingface_pipelines)를 참고하세요.

```python
<!--IMPORTS:[{"imported": "HuggingFacePipeline", "source": "langchain_community.llms.huggingface_pipeline", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.huggingface_pipeline.HuggingFacePipeline.html", "title": "Hugging Face"}]-->
from langchain_community.llms.huggingface_pipeline import HuggingFacePipeline
```

로컬 파이프라인 래퍼에서 OpenVINO 백엔드를 사용하려면 optimum 라이브러리를 설치하고 HuggingFacePipeline의 백엔드를 `openvino`로 설정하세요:

```bash
pip install --upgrade-strategy eager "optimum[openvino,nncf]"
```

[사용 예시](/docs/integrations/llms/huggingface_pipelines)를 참고하세요.

CLI를 사용하여 모델을 OpenVINO IR 형식으로 내보내는 방법:

```bash
optimum-cli export openvino --model gpt2 ov_model
```

[weight-only quantization](https://github.com/huggingface/optimum-intel?tab=readme-ov-file#export)을 적용하여 모델을 내보내는 방법.

## 임베딩 모델

### 허깅 페이스 허브

>허깅 페이스 허브는 35만 개 이상의 모델, 7만 5천 개 이상의 데이터셋, 15만 개 이상의 데모 앱(Spaces)이 공개되어 있는 온라인 플랫폼입니다. 이 허브는 누구나 쉽게 협업하고 기계 학습으로 기술을 구축할 수 있는 중앙 집중식 장소입니다.

`sentence_transformers` Python 패키지를 설치해야 합니다.

```bash
pip install sentence_transformers
```

#### HuggingFaceBgeEmbeddings

>BGE 모델은 [베이징 인공 지능 아카데미(BAAI)](https://en.wikipedia.org/wiki/Beijing_Academy_of_Artificial_Intelligence)에서 만들었습니다. `BAAI`는 AI 연구 및 개발에 종사하는 비영리 기관입니다.

### 허깅 페이스 텍스트 임베딩 추론(TEI)

>[허깅 페이스 텍스트 임베딩 추론(TEI)](https://huggingface.co/docs/text-generation-inference/index)은 오픈 소스 텍스트 임베딩 및 시퀀스 분류 모델을 배포하고 서비스하는 도구 키트입니다. `TEI`를 통해 `FlagEmbedding`, `Ember`, `GTE` 및 `E5`를 포함한 가장 인기 있는 모델에 대한 고성능 추출이 가능합니다.


## 문서 로더

### 허깅 페이스 데이터셋

>[허깅 페이스 허브](https://huggingface.co/docs/hub/index)에는 100개 이상의 언어로 된 7만 5천 개 이상의 [데이터셋](https://huggingface.co/docs/hub/index#datasets)이 있습니다. 이 데이터셋은 번역, 자동 음성 인식, 이미지 분류 등 다양한 작업에 사용됩니다.
