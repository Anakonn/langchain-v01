# OpenAI

OpenAI에 관련된 모든 기능

>[OpenAI](https://en.wikipedia.org/wiki/OpenAI)는 비영리 `OpenAI Incorporated`와 영리 자회사 `OpenAI Limited Partnership`로 구성된 미국의 인공지능(AI) 연구소입니다.
>`OpenAI`는 친화적인 AI 개발과 증진을 목표로 AI 연구를 수행합니다.
>`OpenAI` 시스템은 `Microsoft`의 `Azure` 기반 슈퍼컴퓨팅 플랫폼에서 실행됩니다.

>The [OpenAI API](https://platform.openai.com/docs/models)는 다양한 기능과 가격대를 가진 모델들로 구동됩니다.
>
>[ChatGPT](https://chat.openai.com)는 `OpenAI`가 개발한 인공지능 채팅봇입니다.

## 설치 및 설정

통합 패키지를 설치하려면

```bash
pip install langchain-openai
```

OpenAI API 키를 받아 환경 변수(`OPENAI_API_KEY`)로 설정하세요.

## 토크나이저

`tiktoken` 토크나이저는 여러 곳에서 사용할 수 있습니다. 기본적으로 OpenAI LLM의 토큰 수를 세는 데 사용됩니다.

문서를 분할할 때도 다음과 같이 사용할 수 있습니다:

```python
<!--IMPORTS:[{"imported": "CharacterTextSplitter", "source": "langchain.text_splitter", "docs": "https://api.python.langchain.com/en/latest/character/langchain_text_splitters.character.CharacterTextSplitter.html", "title": "OpenAI"}]-->
from langchain.text_splitter import CharacterTextSplitter
CharacterTextSplitter.from_tiktoken_encoder(...)
```

이에 대한 자세한 내용은 [이 노트북](/docs/modules/data_connection/document_transformers/split_by_token#tiktoken)을 참고하세요.
