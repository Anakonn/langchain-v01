# SerpAPI

이 페이지에서는 LangChain 내에서 SerpAPI 검색 API를 사용하는 방법을 다룹니다.
이는 설치 및 설정, 그리고 특정 SerpAPI 래퍼에 대한 참조로 구성됩니다.

## 설치 및 설정

- `pip install google-search-results`로 요구사항을 설치하세요
- SerpAPI API 키를 받고 환경 변수(`SERPAPI_API_KEY`)로 설정하세요

## 래퍼

### 유틸리티

SerpAPI 유틸리티가 있어 이 API를 래핑합니다. 이 유틸리티를 가져오려면:

```python
<!--IMPORTS:[{"imported": "SerpAPIWrapper", "source": "langchain_community.utilities", "docs": "https://api.python.langchain.com/en/latest/utilities/langchain_community.utilities.serpapi.SerpAPIWrapper.html", "title": "SerpAPI"}]-->
from langchain_community.utilities import SerpAPIWrapper
```

이 래퍼에 대한 더 자세한 연습은 [이 노트북](/docs/integrations/tools/serpapi)을 참조하세요.

### 도구

이 래퍼를 도구(Agent와 함께 사용하기 위해)로 쉽게 로드할 수 있습니다.
다음과 같이 할 수 있습니다:

```python
<!--IMPORTS:[{"imported": "load_tools", "source": "langchain.agents", "docs": "https://api.python.langchain.com/en/latest/agent_toolkits/langchain_community.agent_toolkits.load_tools.load_tools.html", "title": "SerpAPI"}]-->
from langchain.agents import load_tools
tools = load_tools(["serpapi"])
```

이에 대한 자세한 내용은 [이 페이지](/docs/modules/tools)를 참조하세요
