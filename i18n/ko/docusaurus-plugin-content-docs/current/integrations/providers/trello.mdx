# Trello

>[Trello](https://www.atlassian.com/software/trello)는 개인과 팀이 작업과 프로젝트를 구성하고 추적할 수 있게 해주는 웹 기반 프로젝트 관리 및 협업 도구입니다. "보드"라고 하는 시각적 인터페이스를 제공하여 사용자가 작업과 활동을 나타내는 목록과 카드를 만들 수 있습니다.
>TrelloLoader를 사용하면 `Trello` 보드에서 카드를 로드할 수 있습니다.

## 설치 및 설정

```bash
pip install py-trello beautifulsoup4
```
