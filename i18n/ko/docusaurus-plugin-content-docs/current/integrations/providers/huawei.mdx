# 화웨이

>[화웨이 테크놀로지스 Co., Ltd.](https://www.huawei.com/)는 중국의 다국적 디지털 통신 기술 기업입니다.
>
>[화웨이 클라우드](https://www.huaweicloud.com/intl/en-us/product/)는 전 세계적인 클라우드 컴퓨팅 서비스를 제공합니다.

## 설치 및 설정

`화웨이 클라우드`에 액세스하려면 액세스 토큰이 필요합니다.

또한 Python 라이브러리를 설치해야 합니다:

```bash
pip install -U esdk-obs-python
```
