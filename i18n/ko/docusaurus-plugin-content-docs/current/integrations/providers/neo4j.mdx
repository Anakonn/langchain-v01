# Neo4j

>What is `Neo4j`?

>- Neo4j는 그래프 데이터베이스 기술을 전문으로 하는 `오픈 소스 데이터베이스 관리 시스템`입니다.
>- Neo4j를 사용하면 노드와 엣지를 사용하여 데이터를 표현하고 저장할 수 있어, 연결된 데이터와 관계를 처리하는 데 적합합니다.
>- Neo4j는 `Cypher 쿼리 언어`를 제공하여 그래프 데이터와 상호 작용하고 쿼리하기 쉽습니다.
>- Neo4j를 사용하면 생산 수준 시스템에 적합한 `고성능 그래프 탐색 및 쿼리`를 달성할 수 있습니다.

>Neo4j 시작하기: [공식 웹사이트](https://neo4j.com/)를 방문하세요.

## 설치 및 설정

## 텍스트에서 지식 그래프 구축하기

텍스트 데이터에는 다양한 관계와 통찰력이 포함되어 있어, 분석, 추천 엔진 또는 지식 관리 애플리케이션에 유용할 수 있습니다.
Diffbot의 NLP API를 사용하면 비정형 텍스트 데이터에서 엔티티, 관계 및 의미를 추출할 수 있습니다.
Diffbot의 NLP API와 Neo4j 그래프 데이터베이스를 결합하면 텍스트에서 추출된 정보를 기반으로 강력하고 동적인 그래프 구조를 만들 수 있습니다.
이러한 그래프 구조는 완전히 쿼리 가능하며 다양한 애플리케이션에 통합될 수 있습니다.
