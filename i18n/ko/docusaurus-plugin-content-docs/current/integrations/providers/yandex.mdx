# 야넥스

야넥스 클라우드와 관련된 모든 기능

>[야넥스 클라우드](https://cloud.yandex.com/en/)는 공개 클라우드 플랫폼입니다.

## 설치 및 설정

야넥스 클라우드 SDK는 PyPI에서 pip를 통해 설치할 수 있습니다:

```bash
pip install yandexcloud
```
