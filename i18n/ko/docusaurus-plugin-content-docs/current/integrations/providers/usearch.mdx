# 검색

>[검색](https://unum-cloud.github.io/usearch/)은 더 작고 빠른 단일 파일 벡터 검색 엔진입니다.

>`검색`의 기본 기능은 `FAISS`와 동일하며, 근사 최근접 이웃 검색을 조사해 본 적이 있다면 인터페이스가 익숙할 것입니다.
> `검색`과 `FAISS`는 모두 `HNSW` 알고리즘을 사용하지만, 설계 원칙에서 크게 다릅니다. `검색`은 사용자 정의 메트릭과 더 적은 종속성에 중점을 두면서도 성능을 유지하는 간결하고 광범위하게 호환되는 솔루션입니다.

## 설치 및 설정

`usearch` 파이썬 패키지를 설치해야 합니다.

```bash
pip install usearch
```
