# ElevenLabs

>[ElevenLabs](https://elevenlabs.io/about)는 모든 언어와 음성으로 콘텐츠를 보편적으로 접근 가능하게 만드는 것을 목표로 하는 음성 AI 연구 및 배포 회사입니다.
>
>`ElevenLabs`는 29개 언어로 수백 개의 새로운 및 기존 음성을 생성할 수 있는 가장 현실적이고 다재다능하며 상황에 맞는 AI 오디오를 만듭니다.

## 설치 및 설정

먼저 ElevenLabs 계정을 설정해야 합니다. [여기](https://docs.elevenlabs.io/welcome/introduction)의 지침을 따를 수 있습니다.

Python 패키지를 설치하세요:

```bash
pip install elevenlabs
```
