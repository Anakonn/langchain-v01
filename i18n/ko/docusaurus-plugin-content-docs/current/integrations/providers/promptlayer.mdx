# PromptLayer

>[PromptLayer](https://docs.promptlayer.com/introduction)는 프롬프트 엔지니어링을 위한 플랫폼입니다.
> 또한 요청, 프롬프트 버전, 사용량을 시각화하여 LLM 관찰 기능을 제공합니다.
>
>`PromptLayer`에는 LangChain과 직접 통합되는 LLM이 있지만(예: [`PromptLayerOpenAI`](https://docs.promptlayer.com/languages/langchain)),
> 콜백을 사용하는 것이 LangChain과 `PromptLayer`를 통합하는 권장 방법입니다.

## 설치 및 설정

`PromptLayer`를 사용하려면 다음을 수행해야 합니다:
- `PromptLayer` 계정 생성
- API 토큰 생성 및 환경 변수(`PROMPTLAYER_API_KEY`)로 설정

Python 패키지 설치:

```bash
pip install promptlayer
```
