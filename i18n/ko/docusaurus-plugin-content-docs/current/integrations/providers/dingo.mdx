# DingoDB

이 페이지에서는 LangChain 내에서 DingoDB 생태계를 사용하는 방법을 다룹니다.
두 부분으로 나뉘어 있습니다: 설치 및 설정, 그리고 특정 DingoDB 래퍼에 대한 참조.

## 설치 및 설정

- `pip install dingodb`로 Python SDK를 설치하세요.

## VectorStore

DingoDB 인덱스를 래핑하는 래퍼가 있어, 시맨틱 검색 또는 예제 선택을 위해 벡터 스토어로 사용할 수 있습니다.

이 벡터 스토어를 가져오려면:

```python
<!--IMPORTS:[{"imported": "Dingo", "source": "langchain_community.vectorstores", "docs": "https://api.python.langchain.com/en/latest/vectorstores/langchain_community.vectorstores.dingo.Dingo.html", "title": "DingoDB"}]-->
from langchain_community.vectorstores import Dingo
```

DingoDB 래퍼에 대한 더 자세한 연습은 [이 노트북](/docs/integrations/vectorstores/dingo)을 참조하세요.
