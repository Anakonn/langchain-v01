# 짜증나게 하다

> [Annoy](https://github.com/spotify/annoy) (`Approximate Nearest Neighbors Oh Yeah`)는 주어진 쿼리 포인트에 가까운 공간 내 포인트를 검색하는 C++ 라이브러리로, Python 바인딩이 제공됩니다. 또한 많은 프로세스가 동일한 데이터를 공유할 수 있도록 메모리에 매핑되는 대용량 읽기 전용 파일 기반 데이터 구조를 생성합니다.

## 설치 및 설정

```bash
pip install annoy
```
