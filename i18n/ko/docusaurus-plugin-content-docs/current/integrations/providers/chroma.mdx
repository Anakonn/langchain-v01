# 크로마

>[크로마](https://docs.trychroma.com/getting-started)는 임베딩을 사용하여 AI 애플리케이션을 구축하기 위한 데이터베이스입니다.

## 설치 및 설정

```bash
pip install langchain-chroma
```

## VectorStore

크로마 벡터 데이터베이스를 래핑하는 래퍼가 있어 의미 검색 또는 예제 선택을 위해 벡터 스토어로 사용할 수 있습니다.
