# EverNote

>[EverNote](https://evernote.com/)은 사진, 오디오 및 저장된 웹 콘텐츠를 포함할 수 있는 메모를 보관하고 작성하기 위한 것입니다. 메모는 가상 "노트북"에 저장되며 태그, 주석, 편집, 검색 및 내보내기가 가능합니다.

## 설치 및 설정

먼저 `lxml` 및 `html2text` Python 패키지를 설치해야 합니다.

```bash
pip install lxml
pip install html2text
```
