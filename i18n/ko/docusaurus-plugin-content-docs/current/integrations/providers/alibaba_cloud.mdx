# 알리바바 클라우드

>[알리바바 그룹 홀딩스 (위키백과)](https://en.wikipedia.org/wiki/Alibaba_Group), 또는 `알리바바`
> (중국어: 阿里巴巴)는 전자상거래, 소매, 인터넷 및 기술 분야에 특화된 중국의 다국적 기술 기업입니다.
>
> [알리바바 클라우드 (위키백과)](https://en.wikipedia.org/wiki/Alibaba_Cloud), 또한 `Aliyun`
> (중국어: 阿里云; 병음: Ālǐyún; 문자 그대로 '알리 클라우드')로 알려져 있는, 클라우드 컴퓨팅 회사이자 `알리바바 그룹`의 자회사입니다. `알리바바 클라우드`는 온라인 기업과 알리바바의 전자상거래 생태계에 클라우드 컴퓨팅 서비스를 제공합니다.
