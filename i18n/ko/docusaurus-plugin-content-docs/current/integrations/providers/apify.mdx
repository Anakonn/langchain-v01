# Apify

>[Apify](https://apify.com)는 웹 스크래핑과 데이터 추출을 위한 클라우드 플랫폼으로, 다양한 스크래핑, 크롤링, 추출 사용 사례를 위한 *Actors*라는 1,000개 이상의 준비된 앱 [생태계](https://apify.com/store)를 제공합니다.

[![Apify Actors](/img/ApifyActors.png)](https://apify.com/store)

이 통합을 통해 `Apify` 플랫폼에서 Actors를 실행하고 LangChain에 결과를 로드하여 웹사이트, 블로그 또는 지식베이스의 문서와 데이터로 벡터 인덱스를 생성할 수 있습니다.

## 설치 및 설정

- `pip install apify-client`로 Python용 Apify API 클라이언트를 설치하세요.
- [Apify API 토큰](https://console.apify.com/account/integrations)을 얻고 환경 변수(`APIFY_API_TOKEN`)로 설정하거나 `ApifyWrapper` 생성자에 `apify_api_token`으로 전달하세요.
