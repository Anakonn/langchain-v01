# 액티브루프 딥 레이크

>[액티브루프 딥 레이크](https://docs.activeloop.ai/)는 딥 러닝 애플리케이션을 위한 데이터 레이크로, 벡터 스토어로 사용할 수 있습니다.

## 왜 딥 레이크인가?

- 단순한 (멀티모달) 벡터 스토어 이상입니다. 자신만의 LLM 모델을 미세 조정하는 데 사용할 수 있습니다.
- 임베딩만 저장하는 것이 아니라 자동 버전 관리와 함께 원본 데이터도 저장합니다.
- 완전히 서버리스입니다. 다른 서비스가 필요 없으며 주요 클라우드 공급자(`AWS S3`, `GCS` 등)와 함께 사용할 수 있습니다.

`액티브루프 딥 레이크`는 `SelfQuery Retrieval`을 지원합니다:
[액티브루프 딥 레이크 셀프 쿼리 리트리버](/docs/integrations/retrievers/self_query/activeloop_deeplake_self_query)
