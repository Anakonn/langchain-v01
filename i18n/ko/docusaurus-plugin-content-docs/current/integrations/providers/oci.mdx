# Oracle Cloud Infrastructure (OCI)

`LangChain` 통합 관련 [Oracle Cloud Infrastructure](https://www.oracle.com/artificial-intelligence/).

## LLMs

### OCI Generative AI

> Oracle Cloud Infrastructure (OCI) [Generative AI](https://docs.oracle.com/en-us/iaas/Content/generative-ai/home.htm)는 다양한 사용 사례를 다루는 최신 맞춤형 대규모 언어 모델(LLM)을 제공하는 완전 관리형 서비스입니다. OCI Generative AI 서비스를 사용하면 사전 학습된 모델을 바로 사용하거나 전용 AI 클러스터에서 자체 데이터를 기반으로 사용자 정의 모델을 만들고 호스팅할 수 있습니다.

사용하려면 최신 `oci` Python SDK가 설치되어 있어야 합니다.

```bash
pip install -U oci
```

[사용 예제](/docs/integrations/llms/oci_generative_ai)를 참조하세요.


### OCI Data Science Model Deployment Endpoint

> [OCI Data Science](https://docs.oracle.com/en-us/iaas/data-science/using/home.htm)는 데이터 과학 팀을 위한 완전 관리형 서버리스 플랫폼입니다. OCI Data Science 플랫폼을 사용하면 기계 학습 모델을 구축, 학습 및 관리할 수 있으며, 이를 [OCI Data Science Model Deployment Service](https://docs.oracle.com/en-us/iaas/data-science/using/model-dep-about.htm)를 사용하여 OCI Model Deployment Endpoint로 배포할 수 있습니다.

VLLM 또는 TGI 프레임워크로 LLM을 배포한 경우 `OCIModelDeploymentVLLM` 또는 `OCIModelDeploymentTGI` 클래스를 사용하여 상호 작용할 수 있습니다.

사용하려면 최신 `oracle-ads` Python SDK가 설치되어 있어야 합니다.

```bash
pip install -U oracle-ads
```

[사용 예제](/docs/integrations/llms/oci_model_deployment_endpoint)를 참조하세요.

```python
<!--IMPORTS:[{"imported": "OCIModelDeploymentVLLM", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.oci_data_science_model_deployment_endpoint.OCIModelDeploymentVLLM.html", "title": "Oracle Cloud Infrastructure (OCI)"}, {"imported": "OCIModelDeploymentTGI", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.oci_data_science_model_deployment_endpoint.OCIModelDeploymentTGI.html", "title": "Oracle Cloud Infrastructure (OCI)"}]-->

```
