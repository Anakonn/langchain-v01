# 아르길라

>[아르길라](https://argilla.io/)는 LLM을 위한 오픈 소스 데이터 큐레이션 플랫폼입니다.
>`Argilla`를 사용하면 누구나 더 빠른 데이터 큐레이션을 통해 강력한 언어 모델을 구축할 수 있습니다.
>`Argilla`는 데이터 레이블링에서 모델 모니터링까지 MLOps 사이클의 각 단계를 지원합니다.

## 설치 및 설정

[API 키](https://platform.openai.com/account/api-keys)를 받으세요.

Python 패키지를 설치하세요:

```bash
pip install argilla
```
