# AI21 Labs

>[AI21 Labs](https://www.ai21.com/about)는 자연어 처리(NLP)를 전문으로 하는 회사로, 자연어를 이해하고 생성할 수 있는 AI 시스템을 개발합니다.

이 페이지에서는 `LangChain`에서 `AI21` 생태계를 사용하는 방법을 다룹니다.

## 설치 및 설정

- AI21 API 키를 얻고 환경 변수(`AI21_API_KEY`)로 설정합니다.
- Python 패키지를 설치합니다:

```bash
pip install langchain-ai21
```
