# 브라우저리스

>[브라우저리스](https://www.browserless.io/docs/start)는 클라우드에서 헤드리스 Chrome 인스턴스를 실행할 수 있는 서비스입니다. 자신의 인프라를 관리할 필요 없이 브라우저 기반 자동화를 대규모로 실행할 수 있는 좋은 방법입니다.

## 설치 및 설정

[여기](https://www.browserless.io/pricing/)에서 API 키를 얻어야 합니다.
