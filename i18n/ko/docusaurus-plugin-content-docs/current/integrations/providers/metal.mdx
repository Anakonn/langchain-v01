# 금속

이 페이지에서는 LangChain 내에서 [Metal](https://getmetal.io)을 사용하는 방법을 다룹니다.

## Metal이란?

Metal은 프로덕션을 위해 구축된 관리형 검색 및 메모리 플랫폼입니다. 쉽게 데이터를 `Metal`에 인덱싱하고 의미론적 검색 및 검색을 수행할 수 있습니다.

![Metal 대시보드의 샘플 데이터가 포함된 Browse Index 기능을 보여주는 스크린샷.](/img/MetalDash.png "Metal 대시보드 인터페이스")

## 빠른 시작

[Metal 계정 생성](https://app.getmetal.io/signup)으로 시작하세요.

그런 다음 `MetalRetriever` 클래스를 사용하여 의미론적 검색, 프롬프트 컨텍스트 등을 위해 데이터를 검색할 수 있습니다. 이 클래스는 `Metal` 인스턴스와 Metal API에 전달할 매개변수 사전을 사용합니다.

```python
<!--IMPORTS:[{"imported": "MetalRetriever", "source": "langchain.retrievers", "docs": "https://api.python.langchain.com/en/latest/retrievers/langchain_community.retrievers.metal.MetalRetriever.html", "title": "Metal"}]-->
from langchain.retrievers import MetalRetriever
from metal_sdk.metal import Metal


metal = Metal("API_KEY", "CLIENT_ID", "INDEX_ID");
retriever = MetalRetriever(metal, params={"limit": 2})

docs = retriever.invoke("search term")
```
