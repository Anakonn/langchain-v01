# Groq

Groq에 오신 것을 환영합니다! 🚀 Groq에서는 세계 최초의 Language Processing Unit™, 즉 LPU를 개발했습니다. Groq LPU는 결정론적이고 단일 코어 스트리밍 아키텍처를 가지고 있어, 어떤 워크로드에 대해서도 예측 가능하고 반복 가능한 성능으로 GenAI 추론 속도의 표준을 설정합니다.

아키텍처 외에도 우리의 소프트웨어는 개발자 여러분이 혁신적이고 강력한 AI 애플리케이션을 만들 수 있는 도구를 제공하도록 설계되었습니다. Groq를 엔진으로 사용하면 다음과 같은 이점을 누릴 수 있습니다:

* 실시간 AI 및 HPC 추론을 위한 타협 없는 낮은 지연 시간과 성능 🔥
* 어떤 워크로드에 대해서도 정확한 성능과 계산 시간 알 수 있음 🔮
* 선두를 유지할 수 있는 최첨단 기술 활용 💪

Groq에 대해 더 알고 싶으시다면 [웹사이트](https://groq.com)를 방문하시고 [Discord 커뮤니티](https://discord.gg/JvNsBDKeCG)에 참여하여 개발자들과 교류해 보세요!

## 설치 및 설정

통합 패키지를 설치하세요:

```bash
pip install langchain-groq
```

[API 키](https://wow.groq.com)를 요청하고 환경 변수로 설정하세요:

```bash
export GROQ_API_KEY=gsk_...
```
