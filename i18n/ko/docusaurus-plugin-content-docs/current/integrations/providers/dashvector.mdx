# DashVector

> [DashVector](https://help.aliyun.com/document_detail/2510225.html)는 고차원 밀집 및 희소 벡터, 실시간 삽입 및 필터링된 검색을 지원하는 완전 관리형 벡터DB 서비스입니다. 자동으로 확장되며 다양한 애플리케이션 요구 사항에 적응할 수 있습니다.

이 문서는 LangChain 생태계에서 DashVector를 활용하는 방법을 보여줍니다. 특히 DashVector 설치 및 LangChain의 VectorStore 플러그인으로 사용하는 방법을 설명합니다.
이 문서는 설치 및 설정, 그리고 특정 DashVector 래퍼에 대한 참조로 구성됩니다.

## 설치 및 설정

Python SDK를 설치합니다:

```bash
pip install dashvector
```

## VectorStore

DashVector Collection은 LangChain 내에서 기본 VectorStore로 래핑되어 의미 검색 또는 예제 선택과 같은 다양한 시나리오에서 사용할 수 있습니다.

VectorStore를 다음과 같이 가져올 수 있습니다:

```python
<!--IMPORTS:[{"imported": "DashVector", "source": "langchain_community.vectorstores", "docs": "https://api.python.langchain.com/en/latest/vectorstores/langchain_community.vectorstores.dashvector.DashVector.html", "title": "DashVector"}]-->
from langchain_community.vectorstores import DashVector
```

DashVector 래퍼에 대한 자세한 내용은 [이 노트북](/docs/integrations/vectorstores/dashvector)을 참조하십시오.
