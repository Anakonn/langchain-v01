# Cassandra

> [Apache Cassandra®](https://cassandra.apache.org/)는 NoSQL, 행 지향적, 고확장성 및 고가용성 데이터베이스입니다.
> 5.0 버전부터 데이터베이스에 [벡터 검색 기능](https://cassandra.apache.org/doc/trunk/cassandra/vector-search/overview.html)이 포함되어 있습니다.

이 페이지에 설명된 통합은 `Cassandra`뿐만 아니라 `Cassandra Query Language` 프로토콜을 사용하는 다른 CQL 호환 데이터베이스에서도 사용할 수 있습니다.

## 설치 및 설정

다음 Python 패키지를 설치하세요:

```bash
pip install "cassio>=0.1.6"
```
