# Ontotext GraphDB

>[Ontotext GraphDB](https://graphdb.ontotext.com/)는 RDF 및 SPARQL을 준수하는 그래프 데이터베이스 및 지식 발견 도구입니다.

## 종속성

[rdflib](https://github.com/RDFLib/rdflib) 패키지를 설치하세요.

```bash
pip install rdflib==7.0.0
```
