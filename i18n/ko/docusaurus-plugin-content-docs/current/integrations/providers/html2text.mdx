# HTML을 텍스트로 변환하기

>[html2text](https://github.com/Alir3z4/html2text/)는 `HTML` 페이지를 깨끗하고 읽기 쉬운 일반 `ASCII 텍스트`로 변환하는 Python 패키지입니다.

ASCII 텍스트는 `Markdown`(텍스트를 HTML 형식으로 변환하는) 형식으로도 유효합니다.

## 설치 및 설정

```bash
pip install html2text
```

## 문서 변환기

[사용 예시](/docs/integrations/document_transformers/html2text)를 참고하세요.

```python
from langchain_community.document_loaders import Html2TextTransformer
```
