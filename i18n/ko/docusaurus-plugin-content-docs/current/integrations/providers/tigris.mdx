# 타이그리스

> [타이그리스](https://tigrisdata.com)는 고성능 벡터 검색 애플리케이션 구축을 단순화하도록 설계된 오픈 소스 Serverless NoSQL 데이터베이스 및 검색 플랫폼입니다.
> `Tigris`는 여러 도구를 관리, 운영 및 동기화하는 인프라 복잡성을 제거하여 훌륭한 애플리케이션 구축에 집중할 수 있게 해줍니다.

## 설치 및 설정

```bash
pip install tigrisdb openapi-schema-pydantic
```
