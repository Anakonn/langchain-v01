# Zilliz

>[Zilliz Cloud](https://zilliz.com/doc/quick_start)는 `LF AI Milvus®`에 대한 완전히 관리되는 클라우드 서비스입니다.

## 설치 및 설정

Python SDK를 설치하세요:

```bash
pip install pymilvus
```
