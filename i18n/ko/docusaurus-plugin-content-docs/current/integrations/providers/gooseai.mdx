# GooseAI

이 페이지에서는 LangChain 내에서 GooseAI 생태계를 사용하는 방법을 다룹니다.
두 부분으로 나뉘어 있습니다: 설치 및 설정, 그리고 특정 GooseAI 래퍼에 대한 참조.

## 설치 및 설정

- `pip install openai`로 Python SDK를 설치하세요.
- 이 링크 [여기](https://goose.ai/)에서 GooseAI API 키를 받으세요.
- 환경 변수(`GOOSEAI_API_KEY`)를 설정하세요.

```python
import os
os.environ["GOOSEAI_API_KEY"] = "YOUR_API_KEY"
```
