# 랜턴

이 페이지에서는 LangChain 내에서 [랜턴](https://github.com/lanterndata/lantern)을 사용하는 방법을 다룹니다.
설정과 특정 랜턴 래퍼에 대한 참조로 구성되어 있습니다.

## 설정

1. 첫 번째 단계는 `lantern` 확장이 설치된 데이터베이스를 만드는 것입니다.

    [랜턴 설치 가이드](https://github.com/lanterndata/lantern#-quick-install)의 단계를 따라 데이터베이스와 확장을 설치하세요. 도커 이미지가 시작하기 가장 쉬운 방법입니다.

## 래퍼

### VectorStore

Postgres 벡터 데이터베이스에 대한 래퍼가 있어, 의미 검색 또는 예제 선택을 위해 벡터스토어로 사용할 수 있습니다.

이 벡터스토어를 가져오려면:

```python
<!--IMPORTS:[{"imported": "Lantern", "source": "langchain_community.vectorstores", "docs": "https://api.python.langchain.com/en/latest/vectorstores/langchain_community.vectorstores.lantern.Lantern.html", "title": "Lantern"}]-->
from langchain_community.vectorstores import Lantern
```

### 사용법

랜턴 래퍼에 대한 더 자세한 연습은 [이 노트북](/docs/integrations/vectorstores/lantern)을 참고하세요.
