# Robocorp

>[Robocorp](https://robocorp.com/)은 어디서나 어떤 규모로든 원활하게 실행되는 Python 작업자를 구축하고 운영하는 데 도움을 줍니다.

## 설치 및 설정

`langchain-robocorp` Python 패키지를 설치해야 합니다:

```bash
pip install langchain-robocorp
```

에이전트 애플리케이션에서 통신하려면 실행 중인 Action Server 인스턴스가 필요합니다. Action Server를 설정하고 작업을 만드는 방법은 [Robocorp Quickstart](https://github.com/robocorp/robocorp#quickstart)를 참조하세요.

Action Server의 `new` 명령을 사용하여 새 프로젝트를 부트스트랩할 수 있습니다.

```bash
action-server new
cd ./your-project-name
action-server start
```
