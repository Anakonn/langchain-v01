# ArangoDB

>[ArangoDB](https://github.com/arangodb/arangodb)는 연결된 데이터에서 더 빠르게 가치를 창출할 수 있는 확장 가능한 그래프 데이터베이스 시스템입니다. 기본 그래프, 통합 검색 엔진 및 단일 쿼리 언어를 통한 JSON 지원을 제공합니다. ArangoDB는 온-프레미스, 클라우드 등 어디서나 실행할 수 있습니다.

## 설치 및 설정

[ArangoDB Python Driver](https://github.com/ArangoDB-Community/python-arango) 패키지를 설치하세요.

```bash
pip install python-arango
```
