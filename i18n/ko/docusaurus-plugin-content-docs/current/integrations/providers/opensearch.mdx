# OpenSearch

이 페이지에서는 LangChain 내에서 OpenSearch 생태계를 사용하는 방법을 다룹니다.
두 부분으로 나뉘어 있습니다: 설치 및 설정, 그리고 특정 OpenSearch 래퍼에 대한 참조.

## 설치 및 설정

- `pip install opensearch-py`로 Python 패키지를 설치하세요.

## 래퍼

### VectorStore

OpenSearch 벡터 데이터베이스에 대한 래퍼가 있어, 이를 semantic search를 위한 vectorstore로 사용할 수 있습니다. 이는 lucene, nmslib 및 faiss 엔진을 통한 근사 벡터 검색 또는 painless 스크립팅과 스크립트 점수 기능을 통한 brute-force 벡터 검색을 지원합니다.

이 vectorstore를 가져오려면:

```python
<!--IMPORTS:[{"imported": "OpenSearchVectorSearch", "source": "langchain_community.vectorstores", "docs": "https://api.python.langchain.com/en/latest/vectorstores/langchain_community.vectorstores.opensearch_vector_search.OpenSearchVectorSearch.html", "title": "OpenSearch"}]-->
from langchain_community.vectorstores import OpenSearchVectorSearch
```

OpenSearch 래퍼에 대한 더 자세한 연습은 [이 노트북](/docs/integrations/vectorstores/opensearch)을 참조하세요.
