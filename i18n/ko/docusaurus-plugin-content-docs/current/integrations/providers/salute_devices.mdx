# 디바이스 경례

Salute Devices는 GigaChat LLM 모델을 제공합니다.

GigaChat에 액세스하는 방법에 대한 자세한 내용은 [여기](https://developers.sber.ru/docs/ru/gigachat/api/integration)를 참조하십시오.

## 설치 및 설정

GigaChat 패키지는 PyPI에서 pip를 통해 설치할 수 있습니다:

```bash
pip install gigachat
```
