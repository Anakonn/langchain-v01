# TigerGraph

>[TigerGraph](https://www.tigergraph.com/tigergraph-db/)은 기본적으로 분산되어 있고 고성능 그래프 데이터베이스입니다.
> 정점과 엣지로 구성된 그래프 형식으로 데이터를 저장하면 풍부한 관계가 생성되어
> LLM 응답을 구체화하는 데 이상적입니다.

## 설치 및 설정

[TigerGraph 데이터베이스에 연결하는 방법](https://docs.tigergraph.com/pytigergraph/current/getting-started/connection)에 대한 지침을 따르세요.

Python SDK를 설치하세요:

```bash
pip install pyTigerGraph
```
