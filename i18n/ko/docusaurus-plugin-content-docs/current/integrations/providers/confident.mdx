# 자신감 있는 AI

>[자신감 있는 AI](https://confident-ai.com)는 `DeepEval`의 제작자입니다.
>
>[DeepEval](https://github.com/confident-ai/deepeval)은 LLM에 대한 단위 테스트 패키지입니다.
> `DeepEval`을 사용하면 단위 테스트와 통합 테스트를 통해 더 빠른 반복으로 강력한 언어 모델을 구축할 수 있습니다. `DeepEval`은 합성 데이터 생성에서 테스트에 이르는 각 단계를 지원합니다.

## 설치 및 설정

[DeepEval API 자격 증명](https://app.confident-ai.com)을 얻어야 합니다.

`DeepEval` Python 패키지를 설치해야 합니다:

```bash
pip install deepeval
```
