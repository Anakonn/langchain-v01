# Remembrall

>[Remembrall](https://remembrall.dev/)은 언어 모델에 장기 메모리, 검색 보조 생성 및 완전한 관찰성을 제공하는 플랫폼입니다.

## 설치 및 설정

시작하려면 [Remembrall 플랫폼에서 Github로 로그인](https://remembrall.dev/login)하고 [설정 페이지에서 API 키를 복사](https://remembrall.dev/dashboard/settings)하세요.

## 메모리

[메모리 사용 예시](/docs/integrations/memory/remembrall)를 참고하세요.
