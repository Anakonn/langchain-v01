# 소나무 열매

>[소나무 열매](https://docs.pinecone.io/docs/overview)는 광범위한 기능을 가진 벡터 데이터베이스입니다.

## 설치 및 설정

Python SDK를 설치하세요:

```bash
pip install langchain-pinecone
```

## 벡터 저장소

Pinecone 인덱스를 감싸는 래퍼가 있어, 의미 검색이나 예제 선택을 위해 이를 벡터 저장소로 사용할 수 있습니다.

```python
<!--IMPORTS:[{"imported": "PineconeVectorStore", "source": "langchain_pinecone", "docs": "https://api.python.langchain.com/en/latest/vectorstores/langchain_pinecone.vectorstores.PineconeVectorStore.html", "title": "Pinecone"}]-->
from langchain_pinecone import PineconeVectorStore
```

Pinecone 벡터 저장소에 대한 더 자세한 연습은 [이 노트북](/docs/integrations/vectorstores/pinecone)을 참고하세요.
