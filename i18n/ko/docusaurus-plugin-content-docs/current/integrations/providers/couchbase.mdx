# Couchbase

>[Couchbase](http://couchbase.com/)는 클라우드, 모바일, AI, 엣지 컴퓨팅 애플리케이션에 대한 탁월한 다재다능성, 성능, 확장성 및 재무적 가치를 제공하는 수상 경력의 분산 NoSQL 클라우드 데이터베이스입니다.

## 설치 및 설정

`couchbase` 패키지를 설치해야 합니다.

```bash
pip install couchbase
```
