# VoyageAI

VoyageAI와 관련된 모든 기능

>[VoyageAI](https://www.voyageai.com/) Voyage AI는 더 나은 검색 품질을 위해 귀하의 도메인과 회사에 맞춤화된 임베딩 모델을 구축합니다.

## 설치 및 설정

통합 패키지를 설치하세요.

```bash
pip install langchain-voyageai
```

VoyageAI API 키를 받아 환경 변수(`VOYAGE_API_KEY`)로 설정하세요.

## 텍스트 임베딩 모델
