# 큐브

>[큐브](https://cube.dev/)는 데이터 앱을 구축하기 위한 시맨틱 레이어입니다. 데이터 엔지니어와 애플리케이션 개발자가 현대 데이터 저장소에서 데이터에 액세스하고, 일관된 정의로 구성하며, 모든 애플리케이션에 제공할 수 있도록 도와줍니다.

## 설치 및 설정

API 키와 큐브 인스턴스의 URL을 얻어야 합니다. [이 지침](https://cube.dev/docs/product/apis-integrations/rest-api#configuration-base-path)을 참조하세요.
