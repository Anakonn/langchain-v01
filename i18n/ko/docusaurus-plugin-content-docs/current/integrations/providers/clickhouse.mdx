# ClickHouse

> [ClickHouse](https://clickhouse.com/)는 완전한 SQL 지원과 분석 쿼리 작성을 돕는 다양한 기능을 갖춘 실시간 애플리케이션 및 분석을 위한 빠르고 리소스 효율적인 오픈 소스 데이터베이스입니다. `L2Distance`와 같은 데이터 구조와 거리 검색 기능 및 [근사 최근접 이웃 검색 인덱스](https://clickhouse.com/docs/en/engines/table-engines/mergetree-family/annindexes)를 제공하여 ClickHouse를 SQL을 사용하여 벡터를 저장하고 검색할 수 있는 고성능 및 확장 가능한 벡터 데이터베이스로 사용할 수 있습니다.

## 설치 및 설정

`clickhouse-connect` Python 패키지를 설치해야 합니다.

```bash
pip install clickhouse-connect
```
