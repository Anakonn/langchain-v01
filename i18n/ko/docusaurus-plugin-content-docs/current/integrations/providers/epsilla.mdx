# Epsilla

이 페이지에서는 LangChain 내에서 [Epsilla](https://github.com/epsilla-cloud/vectordb)를 사용하는 방법을 다룹니다.
설치 및 설정, 그리고 특정 Epsilla 래퍼에 대한 참조로 구성되어 있습니다.

## 설치 및 설정

- `pip/pip3 install pyepsilla`로 Python SDK를 설치하세요.

## 래퍼

### VectorStore

Epsilla 벡터 데이터베이스에 대한 래퍼가 존재하여, 시맨틱 검색 또는 예제 선택을 위해 벡터 스토어로 사용할 수 있습니다.

이 벡터 스토어를 가져오려면:

```python
<!--IMPORTS:[{"imported": "Epsilla", "source": "langchain_community.vectorstores", "docs": "https://api.python.langchain.com/en/latest/vectorstores/langchain_community.vectorstores.epsilla.Epsilla.html", "title": "Epsilla"}]-->
from langchain_community.vectorstores import Epsilla
```

Epsilla 래퍼에 대한 자세한 내용은 [이 노트북](/docs/integrations/vectorstores/epsilla)을 참조하세요.
