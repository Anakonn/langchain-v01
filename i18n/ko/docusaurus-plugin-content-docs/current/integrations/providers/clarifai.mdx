# Clarifai

>[Clarifai](https://clarifai.com)는 2013년에 설립된 최초의 딥러닝 플랫폼 중 하나입니다. Clarifai는 이미지, 비디오, 텍스트 및 오디오 데이터에 대한 데이터 탐색, 데이터 레이블링, 모델 학습, 평가 및 추론을 위한 전체 AI 수명 주기를 제공하는 AI 플랫폼을 제공합니다. LangChain 생태계에서 우리가 알고 있는 바로는, Clarifai는 LLM, 임베딩 및 벡터 스토어를 하나의 프로덕션 규모 플랫폼에서 지원하는 유일한 공급업체이므로 LangChain 구현을 운영화하는 데 탁월한 선택이 될 수 있습니다.

## 설치 및 설정

- Python SDK 설치:

```bash
pip install clarifai
```

[가입](https://clarifai.com/signup)하여 Clarifai 계정을 만든 다음 [보안 설정](https://clarifai.com/settings/security)에서 개인 액세스 토큰을 가져와 환경 변수(`CLARIFAI_PAT`)로 설정하십시오.

## 모델

Clarifai는 다양한 사용 사례에 대해 1,000개 이상의 AI 모델을 제공합니다. [여기](https://clarifai.com/explore)에서 사용 사례에 가장 적합한 모델을 찾을 수 있습니다. 이러한 모델에는 OpenAI, Anthropic, Cohere, AI21 등의 다른 공급업체가 만든 모델과 Falcon, InstructorXL 등의 최신 오픈 소스 모델이 포함됩니다. 따라서 귀하의 제품에 최고의 AI를 구축할 수 있습니다. 이러한 모델은 creator의 user_id와 application이라고 하는 project의 app_id로 구성됩니다. 사용 사례에 가장 적합한 모델을 찾으면 이러한 ID를 메모해 두어야 합니다.

또한 이미지, 비디오, 텍스트 및 오디오 이해를 위한 많은 모델이 있으므로 다양한 AI 모델을 전문가로 활용하여 흥미로운 AI 에이전트를 구축할 수 있습니다.

### LLMs

Clarifai 플랫폼의 LLM 선택을 찾으려면 [여기](https://clarifai.com/explore/models?filterData=%5B%7B%22field%22%3A%22model_type_id%22%2C%22value%22%3A%5B%22text-to-text%22%5D%7D%5D&page=1&perPage=24)에서 text to text 모델 유형을 선택하십시오.

```python
<!--IMPORTS:[{"imported": "Clarifai", "source": "langchain_community.llms", "docs": "https://api.python.langchain.com/en/latest/llms/langchain_community.llms.clarifai.Clarifai.html", "title": "Clarifai"}]-->
from langchain_community.llms import Clarifai
llm = Clarifai(pat=CLARIFAI_PAT, user_id=USER_ID, app_id=APP_ID, model_id=MODEL_ID)
```

자세한 내용은 Clarifai LLM 래퍼에 대한 문서에서 [자세한 안내](/docs/integrations/llms/clarifai)를 제공합니다.

### 텍스트 임베딩 모델

Clarifai 플랫폼의 텍스트 임베딩 모델 선택을 찾으려면 [여기](https://clarifai.com/explore/models?page=1&perPage=24&filterData=%5B%7B%22field%22%3A%22model_type_id%22%2C%22value%22%3A%5B%22text-embedder%22%5D%7D%5D)에서 text to embedding 모델 유형을 선택하십시오.

LangChain에는 Clarifai Embedding 모델이 있으며, 다음과 같이 액세스할 수 있습니다:

```python
<!--IMPORTS:[{"imported": "ClarifaiEmbeddings", "source": "langchain_community.embeddings", "docs": "https://api.python.langchain.com/en/latest/embeddings/langchain_community.embeddings.clarifai.ClarifaiEmbeddings.html", "title": "Clarifai"}]-->
from langchain_community.embeddings import ClarifaiEmbeddings
embeddings = ClarifaiEmbeddings(pat=CLARIFAI_PAT, user_id=USER_ID, app_id=APP_ID, model_id=MODEL_ID)
```

자세한 내용은 Clarifai Embeddings 래퍼에 대한 문서에서 [자세한 안내](/docs/integrations/text_embedding/clarifai)를 제공합니다.

## 벡터 스토어

Clarifai의 벡터 DB는 2016년에 출시되었으며 실시간 검색 쿼리를 지원하도록 최적화되었습니다. Clarifai 플랫폼의 워크플로우에서 데이터는 자동으로 임베딩 모델과 선택적으로 다른 모델에 의해 인덱싱되어 DB에 저장됩니다. 벡터뿐만 아니라 메타데이터 일치, 다른 AI 예측 개념 및 지리 좌표 검색을 통해 DB를 쿼리할 수 있습니다. 간단히 애플리케이션을 만들고 데이터 유형에 적합한 기본 워크플로를 선택한 다음 [여기에 문서화된 대로](https://docs.clarifai.com/api-guide/data/create-get-update-delete) API를 통해 또는 clarifai.com의 UI를 통해 업로드하십시오.

또한 LangChain에서 직접 데이터를 추가할 수 있으며 자동 인덱싱이 수행됩니다. 다른 벡터 스토어와 달리 LangChain에서 임베딩 모델을 제공하고 텍스트에서 임베딩을 가져와 인덱스에 기록할 필요가 없다는 점을 알 수 있습니다. 이것은 단순히 더 편리할 뿐만 아니라 배경에서 모든 인덱싱을 수행하는 Clarifai의 분산 클라우드를 사용하는 것이 훨씬 더 확장 가능합니다.

```python
<!--IMPORTS:[{"imported": "Clarifai", "source": "langchain_community.vectorstores", "docs": "https://api.python.langchain.com/en/latest/vectorstores/langchain_community.vectorstores.clarifai.Clarifai.html", "title": "Clarifai"}]-->
from langchain_community.vectorstores import Clarifai
clarifai_vector_db = Clarifai.from_texts(user_id=USER_ID, app_id=APP_ID, texts=texts, pat=CLARIFAI_PAT, number_of_docs=NUMBER_OF_DOCS, metadatas = metadatas)
```

자세한 내용은 Clarifai 벡터 스토어에 대한 문서에서 [자세한 안내](/docs/integrations/vectorstores/clarifai)를 제공합니다.
