# Airbyte

>[Airbyte](https://github.com/airbytehq/airbyte)는 API, 데이터베이스 및 파일에서 데이터 웨어하우스 및 레이크로의 ELT 파이프라인을 위한 데이터 통합 플랫폼입니다. 데이터 웨어하우스와 데이터베이스로의 가장 큰 ELT 커넥터 카탈로그를 가지고 있습니다.

## 설치 및 설정

```bash
pip install -U langchain-airbyte
```

:::note

현재 `langchain-airbyte` 라이브러리는 Pydantic v2를 지원하지 않습니다.
이 패키지를 사용하려면 Pydantic v1로 다운그레이드해야 합니다.

이 패키지는 현재 Python 3.10+ 버전을 요구합니다.

:::

이 통합 패키지에는 설정해야 할 전역 환경 변수가 없지만, 일부 통합(예: `source-github`)에는 자격 증명이 필요할 수 있습니다.
