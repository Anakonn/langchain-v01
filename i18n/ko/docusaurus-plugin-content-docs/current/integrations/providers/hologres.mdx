# 홀로그레스

>[홀로그레스](https://www.alibabacloud.com/help/en/hologres/latest/introduction)는 알리바바 클라우드에서 개발한 통합 실시간 데이터 웨어하우징 서비스입니다. 홀로그레스를 사용하여 대량의 데이터를 실시간으로 작성, 업데이트, 처리 및 분석할 수 있습니다.
>`홀로그레스`는 표준 `SQL` 구문을 지원하며 `PostgreSQL`과 호환되고 대부분의 PostgreSQL 기능을 지원합니다. 홀로그레스는 페타바이트 규모의 데이터에 대한 온라인 분석 처리(OLAP) 및 임시 분석을 지원하며 고 동시성 및 저 지연 온라인 데이터 서비스를 제공합니다.

>`홀로그레스`는 [Proxima](https://www.alibabacloud.com/help/en/hologres/latest/vector-processing)를 채택하여 **벡터 데이터베이스** 기능을 제공합니다.
>`Proxima`는 `Alibaba DAMO Academy`에서 개발한 고성능 소프트웨어 라이브러리입니다. 벡터의 최근접 이웃을 검색할 수 있습니다. Proxima는 Faiss와 같은 유사한 오픈 소스 소프트웨어보다 더 높은 안정성과 성능을 제공합니다. Proxima를 사용하면 높은 처리량과 낮은 지연 시간으로 텍스트 또는 이미지 임베딩의 유사성을 검색할 수 있습니다. 홀로그레스는 Proxima와 깊이 통합되어 고성능 벡터 검색 서비스를 제공합니다.
