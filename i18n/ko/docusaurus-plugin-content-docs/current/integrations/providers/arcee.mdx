# Arcee

>[Arcee](https://www.arcee.ai/about/about-us)는 우리가 SLM(Small, Specialized, Secure, Scalable Language Models)이라고 명명한 것의 개발과 발전을 가능하게 합니다.
> SLM 적응 시스템과 seamless, secure 통합을 제공함으로써 `Arcee`는 기업이 도메인 적응 언어 모델의 완전한 잠재력을 활용하여 운영의 혁신적인 변화를 이끌 수 있게 합니다.

## 설치 및 설정
