# Infinispan VS

> [Infinispan](https://infinispan.org) Infinispan은 Java 객체에서 일반 텍스트에 이르기까지 모든 유형의 데이터를 보유할 수 있는 키/값 데이터 저장소를 제공하는 오픈 소스 인메모리 데이터 그리드입니다. 버전 15부터 Infinispan은 캐시에 대한 벡터 검색을 지원합니다.

## 설치 및 설정

[시작하기](https://infinispan.org/get-started/)를 참조하여 Infinispan 서버를 실행하십시오. 현재 지원되지 않는 인증을 비활성화할 수 있습니다.
