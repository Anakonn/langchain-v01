# 비정형

> [Unstructured.IO](https://www.unstructured.io/)의 `unstructured` 패키지는 PDF와 Word 문서와 같은 원시 소스 문서에서 깨끗한 텍스트를 추출합니다.
이 페이지에서는 LangChain 내에서 [`unstructured`](https://github.com/Unstructured-IO/unstructured) 생태계를 사용하는 방법을 다룹니다.

## 설치 및 설정

로컬에서 실행되는 로더를 사용하는 경우 `unstructured`와 해당 종속성을 로컬에서 실행하려면 다음 단계를 따르세요.

- `pip install unstructured`로 Python SDK를 설치합니다.
    - 문서별 종속성은 extras로 설치할 수 있습니다. 예: `pip install "unstructured[docx]"`.
    - 모든 문서 유형의 종속성을 설치하려면 `pip install "unstructured[all-docs]"`를 사용하세요.
- 시스템에 아직 없는 경우 다음 시스템 종속성을 설치하세요. 구문 분석하는 문서 유형에 따라 모든 종속성이 필요하지는 않습니다.
    - `libmagic-dev` (파일 유형 감지)
    - `poppler-utils` (이미지 및 PDF)
    - `tesseract-ocr`(이미지 및 PDF)
    - `libreoffice` (MS Office 문서)
    - `pandoc` (EPUB)

설정 없이 빠르게 시작하려면 `pip install unstructured`를 실행하고 `UnstructuredAPIFileLoader` 또는 `UnstructuredAPIFileIOLoader`를 사용할 수 있습니다. 이렇게 하면 호스팅된 Unstructured API를 사용하여 문서를 처리합니다.

`Unstructured API`를 사용하려면 API 키가 필요합니다.
[여기](https://unstructured.io/api-key-hosted)에서 API 키를 요청하고 오늘 바로 사용을 시작할 수 있습니다!
[여기](https://github.com/Unstructured-IO/unstructured-api)의 README를 확인하여 API 호출을 시작하는 방법을 알아보세요.
[커뮤니티 슬랙](https://join.slack.com/t/unstructuredw-kbe4326/shared_invite/zt-1x7cgo0pg-PTptXWylzPQF9xZolzCnwQ)에서 피드백을 보내주세요.
품질과 성능 향상을 위한 개선 사항이 계속 추가될 예정입니다!
[여기](https://github.com/Unstructured-IO/unstructured-api#dizzy-instructions-for-using-the-docker-image)의 지침을 확인하여 Unstructured API를 자체 호스팅하거나 로컬에서 실행할 수 있습니다.
