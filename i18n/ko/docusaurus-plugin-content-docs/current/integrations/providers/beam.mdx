# 빔 (Beam)

>[빔(Beam)](https://www.beam.cloud/)은 GPU가 탑재된 원격 서버에서 코드를 실행할 수 있는 클라우드 컴퓨팅 플랫폼입니다.

## 설치 및 설정

- [계정 생성하기](https://www.beam.cloud/)
- `curl https://raw.githubusercontent.com/slai-labs/get-beam/main/get-beam.sh -sSfL | sh` 명령어로 Beam CLI 설치하기
- `beam configure` 명령어로 API 키 등록하기
- `BEAM_CLIENT_ID` 및 `BEAM_CLIENT_SECRET` 환경 변수 설정하기
- Beam SDK 설치하기:

```bash
pip install beam-sdk
```
