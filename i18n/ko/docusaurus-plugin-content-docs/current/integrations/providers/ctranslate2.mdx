# CTranslate2

>[CTranslate2](https://opennmt.net/CTranslate2/quickstart.html)는 Transformer 모델의 효율적인 추론을 위한 C++ 및 Python 라이브러리입니다.
>
>이 프로젝트는 가중치 양자화, 레이어 융합, 배치 재정렬 등의 많은 성능 최적화 기술을 적용하는 사용자 정의 런타임을 구현하여 CPU 및 GPU에서 Transformer 모델의 속도를 높이고 메모리 사용량을 줄입니다.
>
>[프로젝트 저장소](https://opennmt.net/CTranslate2/guides/transformers.html)에는 지원되는 기능과 모델의 전체 목록이 포함되어 있습니다. 시작하려면 공식 [빠른 시작 가이드](https://opennmt.net/CTranslate2/quickstart.html)를 확인하세요.

## 설치 및 설정

Python 패키지를 설치하세요:

```bash
pip install ctranslate2
```
