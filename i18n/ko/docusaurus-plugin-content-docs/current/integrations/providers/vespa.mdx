# Vespa

>[Vespa](https://vespa.ai/)는 완전한 기능을 갖춘 검색 엔진이자 벡터 데이터베이스입니다.
> 동일한 쿼리에서 벡터 검색(ANN), 어휘 검색, 구조화된 데이터 검색을 지원합니다.

## 설치 및 설정

```bash
pip install pyvespa
```
