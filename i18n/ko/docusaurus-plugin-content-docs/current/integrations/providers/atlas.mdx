# 아틀라스

>[Nomic Atlas](https://docs.nomic.ai/index.html)는 소규모 및 인터넷 규모의 비정형 데이터세트와 상호 작용하기 위한 플랫폼입니다.

## 설치 및 설정

- `pip install nomic`을 사용하여 Python 패키지를 설치하세요.
- `Nomic`은 langchains poetry extras에도 포함되어 있습니다. `poetry install -E all`을 실행하세요.
