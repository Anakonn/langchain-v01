# 태그

콜백에 태그를 추가할 수 있습니다. `call()`/`run()`/`apply()` 메서드에 `tags` 인수를 전달하면 됩니다. 이는 로그를 필터링하는 데 유용합니다. 예를 들어 특정 `LLMChain`에 대한 모든 요청을 기록하려면 태그를 추가하고 해당 태그로 로그를 필터링할 수 있습니다. 생성자와 요청 콜백 모두에 태그를 전달할 수 있습니다. 자세한 내용은 위의 예를 참조하세요. 이러한 태그는 `on_llm_start`, `on_chat_model_start`, `on_chain_start`, `on_tool_start`와 같은 "start" 콜백 메서드의 `tags` 인수로 전달됩니다.
