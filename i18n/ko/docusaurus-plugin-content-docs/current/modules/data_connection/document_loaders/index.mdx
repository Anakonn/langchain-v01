# 문서 로더

:::info
3rd-party 도구와의 내장 문서 로더 통합에 대한 문서는 [Integrations](/docs/integrations/document_loaders/)를 참고하세요.
:::

문서 로더를 사용하여 소스에서 `Document`로 데이터를 로드합니다. `Document`는 텍스트와 관련 메타데이터의 조각입니다. 예를 들어, 단순한 `.txt` 파일을 로드하거나, 웹 페이지의 텍스트 내용을 로드하거나, YouTube 동영상의 트랜스크립트를 로드하는 문서 로더가 있습니다.

문서 로더는 구성된 소스에서 문서로 데이터를 로드하는 "load" 메서드를 제공합니다. 또한 메모리에 데이터를 지연 로드하는 "lazy load"를 구현할 수 있습니다.

## 시작하기

가장 간단한 로더는 파일을 텍스트로 읽어 들여 모두 하나의 문서에 배치합니다.

```python
<!--IMPORTS:[{"imported": "TextLoader", "source": "langchain_community.document_loaders", "docs": "https://api.python.langchain.com/en/latest/document_loaders/langchain_community.document_loaders.text.TextLoader.html", "title": "Document loaders"}]-->
from langchain_community.document_loaders import TextLoader

loader = TextLoader("./index.md")
loader.load()
```
