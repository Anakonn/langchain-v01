# 프롬프트

언어 모델에 대한 프롬프트는 사용자가 제공하는 일련의 지침 또는 입력으로, 모델의 응답을 안내하여 문맥을 이해하고 관련성 있고 일관된 언어 기반 출력을 생성할 수 있도록 합니다. 이는 질문에 답변하거나, 문장을 완성하거나, 대화에 참여하는 등의 작업에 활용됩니다.
