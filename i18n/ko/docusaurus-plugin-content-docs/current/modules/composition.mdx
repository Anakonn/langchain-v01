# 구성

이 섹션에는 다른 임의의 시스템(예: 외부 API 및 서비스) 및/또는 LangChain 기본 구성 요소를 결합하는 상위 수준 구성 요소가 포함되어 있습니다.

이 섹션을 시작하기 위한 좋은 입문서는 [LangChain 표현 언어](/docs/expression_language/get_started)에 대한 섹션을 읽고 파이핑 및 제공되는 다양한 기본 구성 요소를 통해 시퀀스를 구성하는 것에 익숙해지는 것입니다.

이 섹션에서 다루는 구성 요소는 다음과 같습니다:

## [도구](/docs/modules/tools/)

도구는 LLM 및 기타 구성 요소가 다른 시스템과 상호 작용할 수 있는 인터페이스를 제공합니다. 예로는 Wikipedia, 계산기, Python REPL 등이 있습니다.

## [에이전트](/docs/modules/agents)

에이전트는 언어 모델을 사용하여 도구로 정의된 작업을 결정합니다. 에이전트에는 `executor`가 필요한데, 이는 에이전트의 런타임입니다.
실행기는 에이전트를 호출하고, 에이전트가 선택한 도구를 실행하며, 작업 출력을 에이전트에게 다시 전달하고 반복하는 역할을 합니다.
에이전트는 이전 결과의 출력을 구문 분석하고 다음 단계를 선택할 책임이 있습니다.

## [체인](/docs/modules/chains)

다른 기본 구성 요소 및 구성 요소를 사용하여 구축하는 블록 스타일 구성입니다.
