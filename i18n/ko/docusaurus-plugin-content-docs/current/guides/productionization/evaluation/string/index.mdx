# 문자열 평가기

문자열 평가는 LangChain 내에서 언어 모델의 성능을 평가하기 위해 생성된 출력(예측값)을 참조 문자열 또는 입력과 비교하는 구성 요소입니다. 이러한 비교는 언어 모델의 평가에서 생성된 텍스트의 정확도나 품질을 측정하는 중요한 단계입니다.

실제로 문자열 평가는 예측된 문자열을 질문이나 프롬프트와 같은 주어진 입력과 비교하는 데 사용됩니다. 종종 참조 라벨이나 컨텍스트 문자열이 제공되어 올바르거나 이상적인 응답이 어떻게 생겼는지를 정의합니다. 이러한 평가기는 평가 프로세스를 특정 애플리케이션의 요구 사항에 맞추기 위해 사용자 정의할 수 있습니다.

커스텀 문자열 평가기를 만들려면 `StringEvaluator` 클래스를 상속하고 `_evaluate_strings` 메서드를 구현하십시오. 비동기 지원이 필요한 경우 `_aevaluate_strings` 메서드도 구현하십시오.

다음은 문자열 평가기와 관련된 주요 속성과 메서드 요약입니다:

- `evaluation_name`: 평가의 이름을 지정합니다.
- `requires_input`: 평가기가 입력 문자열을 필요로 하는지 여부를 나타내는 부울 속성입니다. True인 경우, 입력이 제공되지 않으면 평가기가 오류를 발생시킵니다. False인 경우, 입력이 제공되더라도 평가에 고려되지 않음을 나타내는 경고가 기록됩니다.
- `requires_reference`: 평가기가 참조 라벨을 필요로 하는지 여부를 나타내는 부울 속성입니다. True인 경우, 참조가 제공되지 않으면 평가기가 오류를 발생시킵니다. False인 경우, 참조가 제공되더라도 평가에 고려되지 않음을 나타내는 경고가 기록됩니다.

문자열 평가기는 다음과 같은 메서드를 구현합니다:

- `aevaluate_strings`: 체인 또는 언어 모델의 출력을 비동기적으로 평가하며, 선택적 입력 및 라벨을 지원합니다.
- `evaluate_strings`: 체인 또는 언어 모델의 출력을 동기적으로 평가하며, 선택적 입력 및 라벨을 지원합니다.
