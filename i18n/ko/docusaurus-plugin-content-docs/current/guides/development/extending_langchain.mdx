# LangChain 확장

LangChain의 기본 추상화를 확장하는 것은 오픈 소스 저장소에 기여하거나 맞춤형 내부 통합을 구축하려는 경우 모두 권장됩니다.

다음 모듈에 대한 사용자 지정 클래스를 빌드하는 가이드들을 확인해 보세요:

- 대화형 언어 모델과 인터페이스하기 위한 [챗 모델](/docs/modules/model_io/chat/custom_chat_model).
- 텍스트 언어 모델과 인터페이스하기 위한 [LLM](/docs/modules/model_io/llms/custom_llm).
- 언어 모델 출력을 처리하기 위한 [출력 파서](/docs/modules/model_io/output_parsers/custom).
