# 📕 패키지 버전 관리

현재 LangChain은 비공식 릴리스 프로세스를 가지고 있습니다. 릴리스는 유지보수자에 의해 높은 빈도로 커팅되고 [PyPI](https://pypi.org/)에 게시됩니다.
각 패키지는 약간 다르게 버전이 매겨집니다.

## `langchain-core`

`langchain-core`는 현재 버전 `0.1.x`입니다.

`langchain-core`는 LangChain 생태계의 기본 추상화와 런타임을 포함하고 있기 때문에, 모든 주요 변경 사항은 사전에 공지되고 버전이 증가할 것입니다. 예외는 `beta` 데코레이터가 표시된 기능들입니다 (API 참조에서 확인할 수 있으며, 이러한 기능을 사용할 때 경고가 표시됩니다). 베타 기능의 이유는 변화의 속도가 빠른 분야이므로 신속하게 움직이는 것이 여전히 우선 순위이기 때문입니다.

다음과 같은 경우 마이너 버전이 증가합니다:

- `beta`로 표시된 공개 인터페이스의 주요 변경 사항

다음과 같은 경우 패치 버전이 증가합니다:

- 버그 수정
- 새로운 기능
- 비공개 인터페이스의 변경 사항
- `beta` 기능의 변경 사항

## `langchain`

`langchain`은 현재 버전 `0.1.x`입니다.

다음과 같은 경우 마이너 버전이 증가합니다:

- `beta`로 표시되지 않은 공개 인터페이스의 주요 변경 사항

다음과 같은 경우 패치 버전이 증가합니다:

- 버그 수정
- 새로운 기능
- 비공개 인터페이스의 변경 사항
- `beta` 기능의 변경 사항

우리는 레거시 체인과 에이전트에 대한 몇 가지 주요 변경 사항이 포함된 `langchain` v0.2 릴리스를 작업 중입니다.
또한, `langchain-community`를 종속성에서 제거하고, `langchain-community`로 이동된 통합 기능을 다시 내보내지 않을 것입니다.

## `langchain-community`

`langchain-community`는 현재 버전 `0.0.x`입니다.

모든 변경 사항은 패치 버전 증가와 함께 이루어집니다.

## `langchain-experimental`

`langchain-experimental`은 현재 버전 `0.0.x`입니다.

모든 변경 사항은 패치 버전 증가와 함께 이루어집니다.

## 파트너 패키지

파트너 패키지는 독립적으로 버전이 매겨집니다.
