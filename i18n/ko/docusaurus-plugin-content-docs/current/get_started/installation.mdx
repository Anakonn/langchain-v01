# 설치

## 공식 릴리스

LangChain을 설치하려면 다음 명령을 실행하세요:

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
import CodeBlock from '@theme/CodeBlock';

<Tabs>
  <TabItem value='pip' label='Pip' default>
    <CodeBlock language='bash'>pip install langchain</CodeBlock>
  </TabItem>
  <TabItem value='conda' label='Conda'>
    <CodeBlock language='bash'>conda install langchain -c conda-forge</CodeBlock>
  </TabItem>
</Tabs>

이렇게 하면 LangChain의 최소 요구사항이 설치됩니다.
LangChain의 많은 가치는 다양한 모델 제공자, 데이터 저장소 등과 통합할 때 나타납니다.
기본적으로 이러한 통합에 필요한 종속성은 설치되지 않습니다. 특정 통합을 위해서는 종속성을 별도로 설치해야 합니다.

## 소스에서 설치

소스에서 설치하려면, 리포지토리를 클론하고 디렉토리가 `PATH/TO/REPO/langchain/libs/langchain`인지 확인한 후 다음 명령을 실행하세요:

```bash
pip install -e .
```

## LangChain 코어

`langchain-core` 패키지는 LangChain 생태계의 나머지가 사용하는 기본 추상화와 LangChain Expression Language를 포함합니다. `langchain`에 의해 자동으로 설치되지만, 별도로 사용할 수도 있습니다. 설치하려면 다음 명령을 실행하세요:

```bash
pip install langchain-core
```

## LangChain 커뮤니티

`langchain-community` 패키지는 서드 파티 통합을 포함합니다. `langchain`에 의해 자동으로 설치되지만, 별도로 사용할 수도 있습니다. 설치하려면 다음 명령을 실행하세요:

```bash
pip install langchain-community
```

## LangChain 실험용

`langchain-experimental` 패키지는 연구 및 실험적 용도를 위한 LangChain 코드를 포함합니다. 설치하려면 다음 명령을 실행하세요:

```bash
pip install langchain-experimental
```

## LangGraph

`langgraph`는 LLM을 사용하여 상태가 있는 다중 행위자 애플리케이션을 구축하기 위한 라이브러리로, LangChain 위에 구축되었으며 함께 사용하기 위해 설계되었습니다. 설치하려면 다음 명령을 실행하세요:

```bash
pip install langgraph
```

## LangServe

LangServe는 개발자가 LangChain 런너블 및 체인을 REST API로 배포할 수 있게 돕습니다.
LangServe는 LangChain CLI에 의해 자동으로 설치됩니다.
LangChain CLI를 사용하지 않는 경우, 다음 명령을 실행하여 설치하세요:

```bash
pip install "langserve[all]"
```

클라이언트 및 서버 종속성을 모두 설치하려면 이 명령을 사용하세요. 클라이언트 코드만 설치하려면 `pip install "langserve[client]"`, 서버 코드만 설치하려면 `pip install "langserve[server]"`를 실행하세요.

## LangChain CLI

LangChain CLI는 LangChain 템플릿 및 기타 LangServe 프로젝트 작업에 유용합니다. 설치하려면 다음 명령을 실행하세요:

```bash
pip install langchain-cli
```

## LangSmith SDK

LangSmith SDK는 LangChain에 의해 자동으로 설치됩니다.
LangChain을 사용하지 않는 경우, 다음 명령을 실행하여 설치하세요:

```bash
pip install langsmith
```
