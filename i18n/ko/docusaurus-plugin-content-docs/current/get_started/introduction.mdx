# 소개

**LangChain**은 대형 언어 모델(LLM)을 기반으로 하는 애플리케이션을 개발하는 프레임워크입니다.

LangChain은 LLM 애플리케이션 수명 주기의 모든 단계를 간소화합니다:
- **개발**: LangChain의 오픈 소스 [빌딩 블록](/docs/expression_language/)과 [컴포넌트](/docs/modules/)를 사용하여 애플리케이션을 구축하세요. [타사 통합](/docs/integrations/platforms/) 및 [템플릿](/docs/templates)을 사용하여 바로 시작할 수 있습니다.
- **제품화**: [LangSmith](/docs/langsmith/)를 사용하여 체인을 검사, 모니터링 및 평가하여 지속적으로 최적화하고 신뢰할 수 있게 배포할 수 있습니다.
- **배포**: [LangServe](/docs/langserve)를 사용하여 모든 체인을 API로 변환하세요.

구체적으로, 이 프레임워크는 다음과 같은 오픈 소스 라이브러리로 구성됩니다:

- **`langchain-core`**: 기본 추상화 및 LangChain 표현 언어.
- **`langchain-community`**: 타사 통합.
  - 파트너 패키지(예: **`langchain-openai`**, **`langchain-anthropic`** 등): 일부 통합은 **`langchain-core`**에만 의존하는 독립형 패키지로 분리되었습니다.
- **`langchain`**: 애플리케이션의 인지 아키텍처를 구성하는 체인, 에이전트 및 검색 전략.
- **[langgraph](/docs/langgraph)**: 그래프에서 단계를 가장자리와 노드로 모델링하여 LLM과 함께 견고하고 상태적인 다중 행위자 애플리케이션을 구축합니다.
- **[langserve](/docs/langserve)**: LangChain 체인을 REST API로 배포합니다.

더 넓은 생태계는 다음을 포함합니다:

- **[LangSmith](/docs/langsmith)**: LLM 애플리케이션을 디버그, 테스트, 평가 및 모니터링할 수 있는 개발자 플랫폼으로, LangChain과 원활하게 통합됩니다.

## 시작하기

첫 번째 LangChain 애플리케이션을 구축하여 프레임워크를 익히려면 [Quickstart](/docs/get_started/quickstart) 가이드를 따르는 것이 좋습니다.

LangChain을 설치하고 환경을 설정하고 빌드를 시작하는 방법에 대한 지침은 [여기](/docs/get_started/installation)를 참조하세요.

:::note

이 문서는 Python LangChain 라이브러리에 중점을 둡니다. JavaScript LangChain 라이브러리에 대한 문서는 [여기](https://js.langchain.com)에서 확인하세요.

:::

## 표현 언어

LangChain 표현 언어(LCEL)는 LangChain의 많은 구성 요소의 기초이며, 체인을 구성하는 선언적 방법입니다. LCEL은 가장 간단한 "프롬프트 + LLM" 체인부터 가장 복잡한 체인까지, 코드 변경 없이 프로토타입을 생산에 적용할 수 있도록 처음부터 설계되었습니다.

- **[시작하기](/docs/expression_language/)**: LCEL 및 그 이점
- **[실행 가능한 인터페이스](/docs/expression_language/interface)**: LCEL 객체에 대한 표준 인터페이스
- **[원시 요소](/docs/expression_language/primitives)**: LCEL에 포함된 원시 요소에 대한 추가 정보
- 그 외에도 더 많이!

## 생태계

### [🦜🛠️ LangSmith](/docs/langsmith)

언어 모델 애플리케이션 및 지능형 에이전트를 추적하고 평가하여 프로토타입에서 생산까지 이동할 수 있도록 도와줍니다.

### [🦜🕸️ LangGraph](/docs/langgraph)

LLM을 사용하여 상태적이고 다중 행위자 애플리케이션을 구축하며, LangChain 원시 요소 위에 구축되어 이를 사용하도록 설계되었습니다.

### [🦜🏓 LangServe](/docs/langserve)

LangChain 실행 가능 항목 및 체인을 REST API로 배포합니다.
