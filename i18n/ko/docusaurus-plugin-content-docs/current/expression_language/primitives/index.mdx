# 원시 기능

LCEL과 함께 사용할 수 있는 다양한 [컴포넌트](/docs/modules) 외에도 LangChain에는 데이터를 전달하고 형식을 지정하며, 인수를 바인딩하고, 사용자 정의 로직을 호출하는 등의 작업을 돕는 다양한 원시 기능이 포함되어 있습니다.

이 섹션에서는 이러한 컴포넌트들이 유용한 곳과 방법에 대해 더 깊이 탐구합니다.

import DocCardList from "@theme/DocCardList";
import { useCurrentSidebarCategory } from '@docusaurus/theme-common';

<DocCardList items={useCurrentSidebarCategory().items.filter((item) => item.href !== "/docs/expression_language/primitives/")} />
